6. 1. [tex]jika \: 2 log(5) = a \: dan \: 3 log(2) = b \: maka \: 6 log(50) adalah[/tex]tolong yaaa​


Nilai dari [tex] \rm {{}^{6} log_{50}} [/tex] adalah [tex] \rm {\dfrac{b \: + \: 2ab}{b \: + \: 1} }[/tex] .

PEMBAHASAN

Logaritma adalah suatu operasi matematika kbalikan dari perpangkatan.

Apabila:

[tex] \rm { {}^{a} log_{b} \: = \: c \: \: dan \: \: {a}^{c} \: = \: b}[/tex]

Dengan syarat:

a > 0a ≠ 1b > 0

Berikut ini adalah sifat-sifat logaritma:

[tex] \boxed{ \rm{1) \: \: {}^{a} log_{a} \: = \: 1}}[/tex]

[tex] \boxed{ \rm{2) \: \: {}^{a} log_{b} \: + \: {}^{a} log_{c} \: = \: {}^{a} log_{(b \: . \: c)}}}[/tex]

[tex] \boxed{ \rm{3) \: \: {}^{a} log_{b} \: - \: {}^{a} log_{c} \: = \: {}^{a} log_{ (\frac{b}{c}) }}}[/tex]

[tex] \boxed{ \rm{4) \: \: {}^{ {a}^{p} } log_{ {b}^{c} } \: = \: \frac{q}{p} \: .\: {}^{a} log_{b}}}[/tex]

[tex] \boxed{\rm {5) \: \: {}^{a} log_{b} \: = \: \dfrac{1}{ {}^{b} log_{a}}} }[/tex]

[tex] \boxed{ \rm{6) \: \: {}^{a} log_{b} \: = \: \dfrac{ {}^{x} log_{b} }{ {}^{x} log_{a}}} }[/tex]

[tex] \boxed{ \rm{7) \: \: {}^{a} log_{b} \:. \: {}^{a} log_{c} \: . \: {}^{a} log_{d} \: = \: {}^{a} log_{d} }}[/tex]

[tex] \boxed{ \rm{8) \: \: {a}^{ {}^{a} log_{x}} \: = \:x }}[/tex]

DIKETAHUI:

[tex] \rm{ {}^{2} log_{5} \: = \: a}[/tex]

[tex] \rm{ {}^{3} log_{2} \: = \: b}[/tex]

Ingat! Bahwasannya itu adalah perkalian logaritma seperti sifat ketujuh. Pembuktian:

[tex] \rm { {}^{3} log_{2} \: .\: {}^{2} log_{5} \: = \: {}^{3} log_{5} \: \to \: ab}[/tex]

DITANYAKAN:

[tex] \rm{Nilai \: \: {}^{6} log_{50} \: ?}[/tex]

PENYELESAIAN:

Untuk mempermudah, cari faktorisasi dari 6 dan 50.

6 = 2 . 3

50 = 2 . 5

[tex] \rm { {}^{6} log_{50} \: = \: {}^{2 \: . \: 3} log_{2 \: . \: {5}^{2} }}[/tex]

[tex] \rm{ {}^{6} log_{50} \: = \: \dfrac{ {}^{x} log_{ {2 \: . \: 5}^{2} }}{ {}^{x} log_{2 \: . \: 3}}}[/tex]

X ganti dengan angka 3

[tex] \rm { {}^{6} log_{50} \: = \: \dfrac{ {}^{3} log_{2 \: . \: {5}^{2} } }{ {}^{3} log_{2 \: . \: 3} } }[/tex]

Gunakan sifat kedua, yaitu dari perkalian menjadi penjumlahan.

[tex] \rm { {}^{6} log_{50} \: = \: \dfrac{ {}^{3} log_{2} \: + \: {}^{3} log_{ {5}^{2} } }{ {}^{3} log_{2} \: + \: {}^{3} log_{3} } }[/tex]

[tex] \rm { {}^{6} log_{50} \: = \: \dfrac{b \: + \: \frac{2}{1} \: . \: {}^{3} log_{5} }{b \: + \: 1} }[/tex]

[tex] \boxed{ \rm{ {}^{6} log_{50} \: = \: \dfrac{b \: + \:2ab }{b \: + \: 1} }}[/tex]

KESIMPULAN:

Nilai dari [tex] \rm {{}^{6} log_{50} }[/tex] adalah [tex] \rm {\dfrac{b \: + \: 2ab}{b \: + \: 1} }[/tex] .
