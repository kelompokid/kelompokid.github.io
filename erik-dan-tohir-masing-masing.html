4. erik dan tohir masing masing memiliki sehelai kertas karton.Karton erik berbentuk persegi dengan panjang sisinya (x-2)cmdan karton tohir berbentuk persegi panjang dengan ukuran panjang (x+8)cm sedangkan lebarnya (x-2)cm.Bila luas karton mereka sama,maka hitunglah jumlah luas karton mereka


Kelas        : 7 
Mapel        : Matematika 
Kategori    : Bab 4 - Persamaan dan Pertidaksamaan Linear Satu Variabel
Kata kunci : luas karton, persegi panjang, persegi

Kode : 7.2.4 [Kelas 7 Matematika Bab 4 - Persamaan dan Pertidaksamaan Linear Satu Variabel]

Soal secara lengkap :

Erik dan Tohir masing masing memiliki sehelai kertas karton. Karton Erik berbentuk persegi dengan panjang sisinya (x + 2) cm dan karton Tohir berbentuk persegi panjang dengan ukuran panjang (x + 8) cm sedangkan lebarnya (x - 2) cm. Bila luas karton mereka sama, maka hitunglah jumlah luas karton mereka ...
a. 100 cm²
b. 121 cm²
c. 144 cm²
d. 169 cm²

Penjelasan : 

Diketahui : 
Erik → s = (x + 2) cm
Tohir → p = (x + 8) cm dan l = (x - 2) cm
L persegi (Erik) = L persegi panjang (Tohir)

ditanya : 
Luas karton mereka ?

Jawab : 

Luas persegi = Luas persegi panjang
s × s              = p × l
(x + 2) (x + 2) = (x + 8) (x - 2)
x² + 4x + 4    = x² + 6x - 16
x² - x² + 4x - 6x = -16 - 4
             -2x   = -20
                 x  = -20 / -2
                  x = 10

Luas karton Erik = s × s
                           = (x + 2) (x + 2) cm²
                           = (10 + 2) (10 + 2) cm²
                           = 12 × 12 cm²
                           = 144 cm²

Luas karton Tohir = p × l
                             = (x + 8) (x - 2) cm²
                             = (10 + 8) (10 - 2) cm²
                             = 18 × 8 cm²
                             = 144 cm² 

Jadi luas karton masing-masing adalah 144 cm²  (C)

Jumlah luas karton mereka = 144 cm² + 144 cm²
                                            = 288 cm²


Semoga bermanfaat

18. Erik dan Tohir masing-masing memiliki sehelai kertas karton karton Erik berbentuk persegi dengan panjang sisinya x + 2 cm dan karton Tohir berbentuk persegi panjang dengan ukuran panjang x + 8 cm sedangkan lebar X - 2 cm bila luas karton mereka sama maka hitunglah jumlah luas karton mereka


Jawab:

﻿

Karna luas karton mereka sama jadi Cari dulu luas karton salah satu Dr Mrk. Trus Klo Udh kali kan 2. Karena yg ditanya jumlah nya.

Insya Allah jawabannya 288 cm2

Diketahui :
- Kertas karton Erik berbentuk persegi.
s = (x + 2) cm
- Kertas karton Tohir berbentuk persegi panjang.
p = (x + 8) cm
l = (x - 2) cm
L. karton persegi = L. karton persegi panjang

Ditanya :
Luas karton =?

Pembahasan :
Bentuk umum dari luas persegi adalah : s², sedangkan bentuk umum dari luas persegi panjang adalah : panjang × lebar .

Di dalam soal, masing - masing ukuran sisi telah diketahui dan luas karton yang berbeda bentuk tersebut adalah sama. Maka, penyelesaiannya adalah sebagai berikut :

Luas persegi = Luas persegi panjang
s² = p × l
(x + 2)² = (x + 8)(x - 2)
x² + 4x + 4 = x² + 6x - 16
-2x = -20
x = 10

Setelah mengetahui nilai x, kita bisa menentukan :
•sisi karton persegi = x + 2 cm
= 10 + 2
= 12 cm

•panjang karton persegi panjang = x + 8 cm
= 10 + 8
= 18 cm

•lebar karton persegi panjang = x - 2 cm
= 10 - 2
= 8 cm

Terakhir, kita akan mengetahui luas karton tersebut. Karena kedua karton memiliki luas yang sama, kita akan pilih salah satu yang akan dihitung luasnya.

Luas karton persegi = s²
= 12²
= 144 cm²

Karena membahas mengenai "jumlah luas karton", maka luas kedua karton harus dijumlahkan.
Maka, luas 1 + luas 2 = 144 cm² + 144 cm²
= 288 cm²

29. Erik dan tohir masing-masing memiliki sehelai kertas karton.Karton Erik berbentuk persegi panjang sisinya (x+2) cm dan karton tohir berbentuk persegi panjang dengan ukuran panjang (x+8) cm sedangkan lebarnya (x-2) cm.Bila luas karton mereka sama,maka hitunglah jumlah luas karton mereka


L1 = (x + 2)^2
L2 = (x + 8) . (x - 2)

           L1            =          L2 
      (x + 2)^2       =  (x + 8) . (x - 2)
   x^2 + 4x + 4     = x^2 + 6x - 16
        4x  - 6x       =    - 16 - 4 
           -2x           =   -20
              x           =   10

L1 + L2 = (x + 2)^2 + (x + 8) . (x - 2)
             = (10 + 2)^2 + (10 + 8) . (10 - 2)
             = 144 + 144 
             = 288cm^2
