9. Ika diketahui sebuah persamaan kuadrat x2-6x+9=0 maka jenis akar akarnya adalah​


Jawaban:

Akar Kembar (Kedua Akarnya sama)

Penjelasan dengan langkah-langkah:

Jenis akar - akar persamaan kuadrat bisa dicirikan dari nilai Diskriminan (D) :

D > 0 Kedua Akar Real

D = 0 Kedua Akar Kembar

D < 0 Kedua Akar Imajiner

x² - 6x + 9 = 0

D = b² - 4.a.c

= (- 6)² - 4.(1).(9)

= 36 - 36

= 0

Karena D = 0 , maka Akar2 nya Kembar

Jadi, jenis akar2 persamaan kuadrat x² - 6x + 9 = 0 adalah akar kembar

12. ika x = 1303 x 1301 - 13022 : 3 dan y = 1762 - 178 x 174 maka ........a. x= y b. x > y c. x< y d. x dan y = 0 e. hubungan x dan y tidak dapat ditentukan


Pernyataan yang lebih tepat adalah x > y. (Jawaban B) . Soal tersbebut meruapkan soal tentang operasi hitung bilangan bulat.

Penjelasan dengan langkah-langkah

Soal tersebut merupakan soal matematika yang membahas tentang  bilangan bulat. Bilangan bulat merupakan bilangan yang bersifat bulat. Bilangan bulat terdiri dari bilangan bulat positif dan bilangan bulat negatif.

Operasi hitung bilangan bulat terdiri dari penjumlahan, pengurangan, perkalian, dan pembagian.

Pada suatu garis bilangan, bilangan bulat positif letaknya di kanan titik 0 dan bilangan negatif letaknya di kiri titik 0. 

Penyelesaian soal

Diketahui:

x = 1303 x 1301 - 13022 : 3y = 1762 - 178 x 174

Ditanyakan:

Tentukan nilai perbandingan x dan y!

Jawab:

Mencari nilai x

x = 1303 x 1301 - 13022 : 3x = 1.695.203 - 4.340,67x = 1.690.862,33

Mencari nilai y

y = 1762 - 178 x 174y = 1762 - 30.972y = -29.210

Berdasarkan nilai yang sudah dicari, maka nilai x > y.

Jadi, nilai x > y. (Jawaban B).
