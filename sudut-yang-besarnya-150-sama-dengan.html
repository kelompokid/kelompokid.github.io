Pada soal ini kita akan belajar untuk melakukan konversi dari derajat ke radian

Radian  dan Derajat adalah satuan yang digunakan untuk mengukur besar sudut

Sebuah lingkaran terdiri dari 2π radian atau setara dengan 360°, kedua nilai ini melambangkan "satu keliling lingkaran"

1π radian melambangkan keliling lingkaran sejauh 180°

Rumus :

π Radians = 180°

1π Radians = [tex]\frac{180}{\pi} [/tex]

1 putaran = 360°

Pembahasan :

Diketahui :

150° = ... rad

Ditanya :

Jadikan dalam radian ?

Dijawab :

150° = 150° x [tex]\frac{\pi}{180} = \frac{5}{6}\pi[/tex] radian

15. jika besar sudut ADC adalah 150° maka besar sudut BCD adalah.....​


Penjelasan dengan langkah-langkah:

<DAB = <BCD

<ABC = ADC

= 105°

<DAB + <BCD + <ABC + <ADC = 360°

<DAB + <BCD + 105° + 105° = 360°

<DAB + <BCD + 210° = 360°

<DAB + <BCD = 360° - 210°

<DAB + <BCD = 150°

Karena <DAB dan <BCD besarnya sama, maka

nilai masing-masing = 150° - 2

= 75°

jadi, besar sudut BCD adalah 75°
