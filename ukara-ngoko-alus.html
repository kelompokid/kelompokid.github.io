5. 5 ukara ngoko, lajeng diowahi dados ukara ngoko alus, krama, lan krama inggil/alus.tolong dong​


Gawe ukara miturut Unggah ungguhing basa :

Nomer 1

Kowe arep lunga menyang ngendi?

→ Ngoko Alus : Panjenengan arep tindak menyang endi?

→ Krama lugu : sampeyan badhe kesah ten pundi?

→ Krama Alus : Panjenengan badhe tindak dhateng pundi?

Nomer 2

Aku arep dolan nggone mbah sesuk.

→ Ngoko Alus : aku arep sowan nggone Simbah sesuk.

→ Krama lugu : Kula badhe sowan Ten griyane simbah mbenjing.

→ Krama Alus : Kula badhe sowan dhateng dalemipun eyang mbenjing.

Nomer 3

Bulik wis mulih saka pasar numpak becak.

→ Ngoko Alus : Bulik wis kondur saka pasar nitih becak.

→ Krama lugu : Bulik mpun wangsul saking peken nitih becak.

→ Krama Alus : bulik sampun kondur saking peken nitih becak.

Nomer 4

Ora usah kesusu, dientekne dhisik mangane.

→ Ngoko Alus : ora usah kesesa, ditelaske dhisik dhaharane.

→ Krama lugu : mboten usah kesesa, ditelasake riyin dhaharane.

→ Krama Alus : mboten sisah kesesa, dipun telasaken rumiyin dhaharanipun.

Nomer 5

Bapak ora iso kerja, Merga lara untu.

→ Ngoko Alus : bapak ora saged kerja, amerga gerah Waja.

→ Krama lugu : bapak mboten saged nyambut amarga gerah Waja.

→ Krama Alus : bapak mboten saged nyambut damel amargi gerah wajaipun.

Pembahasan

Unggah ungguh basa kaperang dadi 4 :

1️⃣ BASA NGOKO LUGU

yaiku basane Ngoko nanging ora dicampur karo basa Krama Alus.

>> tuladha : Melu aku, gelem ora?

2️⃣ BASA NGOKO ALUS

yaiku basane Ngoko nanging wis kecampur antara basa Ngoko lan Krama Alus.

>> tuladha : nderek aku, kersa ora?

3️⃣ BASA KRAMA LUGU

yaiku basane Krama, nanging ora dicampur karo basa Ngoko lan Krama Alus.

>> tuladha : tumut Kula, purun mboten?

4️⃣ BASA KRAMA ALUS

yaiku basane Krama Alus, ora dicampur karo basa Ngoko

>> tuladha : nderek Kula, kersa mboten?
