30. saya sangat perlu bantuan apa ada yang bisa?​


Adi = x

Raka = x + 3

Jumlah kelereng Adi + Raka = 19

x + (x + 3) = 19

x + x + 3 = 19

2x + 3 = 19

2x + 3 - 3 = 19 - 3

2x = 16

2x/2 = 16/2

x = 8

Jadi, kelereng Raka adalah 11 buah
