2. Diketahui segitiga DEF dengan panjang sisi-sisinya d, e, dan f. Benar atau salahkah pernyataan berikut untuk segitiga DEF? a) d^2 = e^2 +f^2 , maka segitiga DEF siku-siku di F. b) d^2 = e^2+f^2, maka segitiga DEF siku-siku di E. c) f^2 = d^2+ e^2, maka segitiga DEF siku-siku di F.​


Diketahui segitiga DEF dengan panjang sisi-sisinya d, e, dan f. Pernyataan yang benar untuk segitiga DEF adalah f² = d² + e², maka segitiga DEF siku-siku di F. (C)

Pembahasan

Segitiga adalah suatu bangun datar yang memiliki 3 sisi dan 3 sudut. Apabila salah satu sudutnya siku-siku, maka segitiga ini disebut segitiga siku-siku. Pada segitiga siku-siku, berlaku rumus:  

Luas = sisi alas x sisi tinggi x (1/2)  Keliling = sisi alas + sisi tinggi + sisi miring  

Teorema Pythagoras adalah teorema yang diterapkan pada segitiga siku-siku. Teorema ini menyatakan hubungan panjang sisi antar segitiga siku-siku. Pada teorema ini berlaku:  

[tex]\boxed{c^{2} = a^{2} + b^{2}}[/tex]

Keterangan:  

a = sisi alas  b = sisi tinggi  c = sisi miring

Dari penjelasan tersebut, mari selesaikan permasalahan di atas!

Diketahui:

Segitiga DEF dengan sisi d, e, dan f.

Ditanyakan:

Pernyataan yang benar dari:

a) d² = e² + f², maka segitiga DEF siku-siku di F.

b) d² = e² + f², maka segitiga DEF siku-siku di E.

c) f² = d² + e², maka segitiga DEF siku-siku di F.

Jawab:

Apabila segitiga DEF memiliki sisi d, e, dan f, maka:

Sisi d berada di hadapan sudut D.Sisi e berada di hadapan sudut E.Sisi f berada di hadapan sudut F.

Pada teorema Pythagoras berlaku kuadrat sisi miring sama dengan jumlah kuadrat sisi yang lainnya. Karena sisi miring berada di depan sudut siku-siku, maka berlaku:

f² = d² + e², maka segitiga DEF siku-siku di F.d² = f² + e², maka segitiga DEF siku-siku di D.e² = d² + f², maka segitiga DEF siku-siku di E.

Jadi, pernyataan yang benar adalah f² = d² + e², maka segitiga DEF siku-siku di F. (C)

9. Diketahui segitiga siku siku DEF siku siku di E dengan cos D= 2/3 nilai tan F adalah​


Jawaban:

[tex]\sf{\tan F = \boxed{\sf{ \frac{2}{5} \sqrt{5} }} }[/tex]

Penjelasan dengan langkah-langkah:

Diketahui

Segitiga DEF siku-siku di E, dengan cos D = 2/3

Ditanya nilai dari tan F

______________________________

[tex]\sf{\cos D = \frac{DE}{DF} }[/tex]

[tex]\sf{\cos D = \frac{2}{3} }[/tex]

Sudah diketahui bahwa DE = 2, dan DF = 3. Di mana DE merupakan sisi samping dan DF adalah sisi miring.

Selanjutnya cari EF dengan Pythagoras:

[tex]\sf{\:\:\:\:\: DF^2 = DE^2 + EF^2 }[/tex]

[tex]\sf{\:\:\:\:\: 3^2 = 2^2 + EF^2 }[/tex]

[tex]\sf{\:\:\:\:\: 9 = 4 + EF^2 }[/tex]

[tex]\sf{\:\:\:\:\: EF^2 = 9 - 4 }[/tex]

[tex]\sf{\:\:\:\:\: EF^2 = 5 }[/tex]

[tex]\sf{\:\:\:\:\: EF = \sqrt{5} }[/tex]

.

Untuk mencari tan F, maka:

[tex]\sf{\:\:\:\:\:\tan F = \frac{DE}{EF} }[/tex]

[tex]\sf{\:\:\:\:\:\tan F = \frac{2}{\sqrt{5}} }[/tex]

[tex]\sf{\:\:\:\:\:\tan F = \frac{2}{\sqrt{5}} \times \frac{\sqrt{5}}{\sqrt{5}} = \boxed{\sf{ \frac{2}{5} \sqrt{5} }} }[/tex]

11. segitiga ABC siku siku di A dan panjang AB= 12 cm. segitigaDEF siku siku di E dan panjang DF =13 cm. segitiga DEF kongruen dengan segitiga ABC. luas segitiga DEF adalah


Segitiga ABC siku siku di A dan panjang AB = 12 cm. Segitiga DEF siku siku di E dan panjang DF = 13 cm. Segitiga DEF kongruen dengan segitiga ABC. Luas segitiga DEF adalah 30 cm². Kongruen adalah keadaan dimana dua buah bangun datar memiliki ukuran dan bentuk yang sama, sehingga sisi-sisi yang bersesuaiannya sama panjang dan sudut-sudut yang bersesuaiannya sama besar. Lambang dari kongruen adalah ≅.


Pembahasan

Diketahui  

Segitiga ABC siku-siku di A dengan AB = 12 cm Segitiga DEF siku-siku di E dengan DF = 13 cm Segitiga ABC ≅ segitiga DEF

Ditanyakan

Luas segitiga DEF


Jawab

Perhatikan gambar segitiga ABC dan segitiga DEF pada lampiran

Karena Segitiga ABC ≅ segitiga DEF, maka

AB = EF = 12 cm BC = DF = 13 cm

Dengan menggunakan teorema pythagoras diperoleh panjang ED atau AC yaitu

ED = √(DF² – EF²)

ED = √(13² – 12²)

ED = √(169 – 144)

ED = √(25)

ED = 5 cm


Jadi luas segitiga DEF adalah

L = ½ × alas × tinggi

L = ½ × ED × EF

L = ½ × 5 cm × 12 cm

L = ½ × 60 cm²

L = 30 cm²
