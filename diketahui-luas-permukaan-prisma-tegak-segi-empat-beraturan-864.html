1. diketahui luas permukaan prisma tegak segi empat beraturan 864 cm² dan tinggi prisma 12cm.panjang didi alas prisma adalah?


Jenis prisma = prisma persegi  (alas dan penutupnya berbentuk persegi)

LP prisma tegak segi empat beraturan           = 864 cm²
(2 X L alas)      + Keliling alas X t prisma)       = 864
(2 X L persegi) + Keliling persegi X t prisma)  = 864
(2 X s^2)          + (4 X s X 12)                          = 864
2s^2 + 48s - 864         = 0    (semua suku dibagi 2):
  s^2 + 24s - 432         = 0  (kemudian faktorisasikan) diperoleh:
(s - 12) (s + 36)           = 0    ,
maka:
s - 12 = 0                                             s + 36 = 0
s        = 12 ( memenuhi)                      s         = -36   (tidak memenuhi)

Jadi sisi alas prisma adalah 12 cm

30. diketahui luas permukaan prisma tegak segi empat beraturan 864 cm² dan tinggi prisma 12 cm panjang sisi alas prisma adalah

Lp = 2(s × s) + ((4 × s) × t)
864 = 2(s × s) + ((4 × s) × 12)

864 = 2s² + 48s
0 = 2s² + 48s - 864
0 = s² + 24s - 432
0 = (s + 36) (s - 12)

0 = s + 36
s₁ = -36

0 = s - 12
s₂ = 12

Jadi panjang sisi alas prisma tersebut adalah 12 cm.
