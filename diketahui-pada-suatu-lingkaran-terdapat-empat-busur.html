1. Jawaban + Cara ada di Gambar

2.Diketahui :

Segitiga ABC

titik-titik sudut berada pada lingkaran O

O = titik pusat

AB melalui O

Ditanya :

Besar sudut BCA

Dijawab :

Kita perhatikan gambar ilustrasi yang terlampir

∠AOB = sudut pusat

∠BCA = sudut keliling

Karena AB melalui titik O dan merupakan garis lurus, maka ∠AOB = 180°

Kemudian kita masukkan kedalam rumus :

Sudut pusat = 2 x sudut keliling

Sudut keliling = x sudut pusat

∠BCA = x ∠AOB

∠BCA = x 180°

∠BCA = 90°

∴ Jadi besar sudut BCA adalah 90°

Esai :

1.Diketahui :

Sudut pusat = 130°

Ditanya :

Sudut keliling = ... ?

Jawab :

Menentukan besar sudut keliling

Sudut keliling = 1/2 × sudut pusat

= 1/2 × 130°

= 65°

Jadi besar sudut keliling tersebut adalah 65°

3.Diketahui :

∠ MAN = 120°

Ditanya :

∠ MON = ... ?

Jawab :

∠ MON merupakan sudut minor pada sudut pusat lingkaran.

Kita cari dulu sudut mayor MON.

Karena ∠ MAN dan ∠ MON mayor menghadap busur yang sama

∠ MON mayor = 2 × ∠ MAN

= 2 × 120°

= 240°

∠ MON minor = 360° - ∠ MON mayor

= 360° - 240°

= 120°

Jadi besar ∠ MON adalah 120°

4.Diketahui m∠ PQR = 125°, m∠ QRS = 78°

Jawab :

a. m∠ SPQ

∠ QRS + ∠ SPQ = 180°

78 + ∠ SPQ = 180°

∠ SPQ = 180 - 78°

∠ SPQ = 102°

b. m∠ RSP.

∠ PQR + ∠ RSP = 180°

125 + ∠ RSP = 180°

∠ RSP = 180 - 125°

∠ RSP = 55°

7.misal sudut pusat a: b : c = 2 : 3 : 4

jumlah perbandingan = 2 + 3 + 4 = 9

a + b + c = 360°

a = 2/9 x 360 = 80°

b = 3/9 x 360 = 120°

c = 4/9 x 360 = 160°

11. Diketahui pada suatu lingkaran terdapat empat busur,yaitu AB,CD,EF,dan GH.Panjang AB>panjangCD> panjangEF>panjangGH>.jika pada masing masing busur tersebut dibuat sudut pusat yg bersesuaian,maka sudut pusat terkecil menghadap busur?


kelas : VIII SMP
mapel : matematika
kategori : lingkaran
kata kunci : sudut pusat, panjang busur

kode : 8.2.6 [matematika SMP kelas 8 Bab 6 lingkaran]

Pembahasan:

hubungan panjang busur lingkaran dengan sudut pusat lingkaran, semakin besar sudutnya maka semakin panjang juga panjang busurnya 
maka jika panjang busur AB > panjang busur CD > panjang busur EF > panjang busur GH, 
maka besar sudut AOB > besar sudut COD > besar sudut EF > besar sudut GH

sehingga sudut pusat terkecil adalah sudut pusat yang menghadap busur GH
