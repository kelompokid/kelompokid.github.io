y' = (3x² (x³ - x - 2) - (x³ + 2) . (3x² - 1))/(x³ - x - 2)²
y' = (3x⁵ - 3x³ - 6x² - 3x⁵ + x³ - 6x² + 2)/(x³ - x - 2)²

Tentukan turunan pertama dari F(x) =-50x (pangkat) 3/2 Tentukan turunan pertama dari F(x) =8x(pangkat) 4+3x(pangkat) 3-4x(pangkat) 2+5 Tentukan turunan pertama dari F(x) =3 (8x (pangkat) 3+3x(pangkat) 2) Tentukan turunan pertama dari F(x) =(8x(pangkat) 2+3x) (2x+3) Tentukan turunan pertama dari F(x) =(4x(pangkat)2+3x+5) (2x+3)


*penyelesaian :

(1) f(x) = -50x^3/2
jadi, turunan pertamanya
f'(x) = -50(3/2)x^3/2-1
f'(x) = -75x^1/2✔

(2) f(x) = 8x^4 + 3x^3 - 4x^2 + 5
jadi, turunan pertamanya
f'(x) = 8(4)x^4-1 + 3(3)x^3-1 - 4(2)x^2-1 + 0
f'(x) = 32x^3 + 9x^2 - 8x✔

(3) f(x) = 3(8x^3 + 3x^2)
= 24x^3 + 9x^2
jadi, turunan pertamanya
f'(x) = 24(3)x^3-1 + 9(2)x^2-1
f'(x) = 72x^2 + 18x✔

(4) f(x) = (8x^2 + 3x)(2x + 3)
jadi,turunan pertamanya
u = 8x^2 + 3x
u' = 16x + 3
v = 2x + 3
v' = 2

f'(x) = u'v + uv'
f'(x) = (16x + 3(2x + 3))+(8x^2 + 3x(2))
f'(x) = 32x^2 + 48x + 6x + 9 + 16x^2 + 6x
f'(x) = 48x^2 + 60x + 9✔

(5) f(x) = (4x^2 +3x + 5)(2x + 3)
u = 4x^2 +3x + 5
u' = 8x + 3
v = 2x + 3
v' = 2

f'(x) = u'v + uv'
f'(x) = (8x + 3(2x + 3))+(4x^2 +3x + 5(2))
f'(x) = 16x^2 + 24x + 6x + 9 + 8x^2 + 6x + 10
f'(x) = 24x^2 + 36x + 19✔
