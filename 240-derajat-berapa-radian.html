2. Nilai dari 240 derajat=.....radian


// persamaan pada 1 putaran lingkaran //

1 putaran = 360°

1 putaran = 2π rad

// sehingga //

2π rad = 360°

// kalikan kedua ruas dengan 2/3 //

2/3(2π rad) = 2/3(360°)

      2(120°) = 4/3π rad

         240° = 4π/3 rad

                  = 4,18 rad

4. Konfersi sudut 240 derajat menjadi radian


1° = ( π / 180° ) . rad


240°

= 240 . 1°

= 240° . ( π / 180° ) rad

= ( ( 240° : 60° ) / ( 180° : 60° ) ) π rad

= ( 4 / 3 ) π rad

= ( 4π / 3 ) rad

= 4,19 rad

8. 45 derajat + 105 derajat kurang 300 derajat + 240 derajat = titik-titik phi radian​


Penjelasan dengan langkah-langkah:

45° + 105° - 300° + 240° = 90°

90° = 90° x π/180° rad

90° = 1/2 . π rad
