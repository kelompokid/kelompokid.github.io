3. Pada suhu tertentu benzena murni mempunyai tekanan uap 75 mmhg. Berapakah tekanan uap benzena pada larutan 6,4% naftalena (mr=128 g/moi) dalam benzena (mr=78 g/moi)


Dik : P°=75 mmHg , Mr benzena = 78 g/mol ,Mr Naftalena = 128 g/mol
Dit : P benzena pada larutan 6,4%naftalena?
Jawab
Larutan 6,4% naftalena mempunya pengertian bahwa terdapat 6,4 gram naftalena(larutan) dan 93,6 gram pelarut

P lar = Xp.P°
Xp= np/np+nt
np= gr/mr = 93,6/77= 1,2 mol
nt = gr/mr = 6,4 / 128 = 0,05 mol
Xp = 1,2/1,2+0,05 = 1,2/1,25 = 24/25
P lar = (24/25) x 75 = 24 x 3 = 72 mmHg

4. tekanan uap murni benzena (C6H6) pada 30 derajat celcius adalah 121,8 mmHg. berapa gram zat A yang non volatil (Mr = 356) yang telah terlarut dalam 250gr benzena, jika tekanan uap larutannya 120,2 mmHg?


mol benzena = gr ÷ Mr
                     = 250 ÷ 78
                     = 3,2

P = Xp × P°
120,2 = Xp × 121,8
Xp = 0,9868

Xp = np ÷ (np + nt)
0,9868 = 3,2 ÷ (3,2 + nt)
3,158 + 0,9868nt = 3,2
           nt              = 0,0426

massa A = mol × Mr
              = 0,0426 × 356
               = 15,12 gr


5. Larutan urea mempunyai tekanan uap 24,5 mmHg . Pada suhu yang sama air murni mempunyai tekanan uap jenuh 25 mmHg. Berapakah kadar urea tersebut..


kita pilih rumus tekanan uap jenuh larutan,
p = p₀ . Xp
24,5 = 25Xp
Xp = 0,98

misalkan massa urea g, dan massa air p, Mr urea = 60, dan Mr air = 18
Xp = [mol air] / [mol air + mol urea]
0,98 = [p/18] / [(p/18) + (g/60)]
0,98 = [p/3] / [(p/3) + (g/10)]
0,98 = [10p] / [10p + 3g]
9,8p + 2,94g = 10p
0,2p = 2,94g
p = 14,7g

selanjutnya kadar urea = g / [g + p] x 100%
kadar = g / [g + 14,7g] x 100%
kadar = [1/15,7] x 100%
 ∴ kadar = 6,37%

Pembahasan

Sifat Koligatif Larutan Nonelektrolit

Sifat koligatif larutan adalah salah satu sifat larutan yang tidak bergantung pada jenis zat terlarut tetapi hanya tergantung pada jumlah zat terlarut. Yang merupakan sifat koligatif larutan adalah

Penurunan tekanan uap jenus (∆P)Kenaikan titik didih (∆Tb)Penurunan titik beku (∆Tf)Tekanan osmotik (π)

Penurunan Tekanan Uap Jenuh (∆P)

Tekanan uap jenuh adalah tekanan uap yang ditimbulkan oleh uap jenuh suatu zat pada saat tercapai kondisi kesetimbangan. Sedangkan tekanan yang ditimbulkan oleh uap jenuh air disebut tekanan uap jenuh air. Penurunan tekanan uap jenuh larutan (∆P) adalah selisih dari tekanan uap jenuh pelarut murni (Po) dengan tekanan uap jenuh larutan (P).

∆P =  P° – P


Tahun 1830-1901, menurut Hukum Raoult, menyatakan tekanan uap jenuh larutan sama dengan tekanan uap jenuh pelarut murni dikalikan dengan fraksi mol pelarut. Sedangkan penurunan tekanan uap jenuh larutan sama dengan tekanan uap jenuh pelarut murni dikalikan dengan fraksi mol terlarut.


 P =  P° . Xpelarut

∆P = P° . Xterlarut


Keterangan:

∆P   = penurunan tekanan uap jenuh

P° = tekanan uap jenuh pelarut air murni

Xterlarut   = fraksi mol zat terlarut

Xpelarut   = fraksi mol zat pelarut


Fraksi Mol (x)

Fraksi mol menyatakan perbandingan mol zat terlarut atau zat pelarut dengan jumlah mol campuran (terlarut dan pelarut).

Rumus :

Xt = nt / nt+np

atau        

Xp = np / nt+np

maka,

Xt  +  Xp  =   1


Dengan :

Xt = fraksi mol zat terlarut                              

Xp = fraksi mol zat pelarut

nt = mol zat terlarut                                        

np = mol zat pelarut



Diket :

massa C₁₀H₈ = 6,4 gram

massa C₆H₆ = 500 gram

P° = 77,9 mmHg


Ditanya :

P


Jawab :

Menentukan Mr C₁₀H₈

Mr C₁₀H₈ = 10. Ar C + 8. Ar H

Mr C₁₀H₈ = 10. 12 + 8. 1

Mr C₁₀H₈ = 120 + 8

Mr C₁₀H₈ = 128

Menentukan Mr C₆H₆

Mr C₆H₆ = 6. Ar C + 6. Ar H

Mr C₆H₆ = 6. 12 + 6. 1

Mr C₆H₆ = 72 + 6

Mr C₆H₆ = 78

Menentukan mol naftalena dan benzena

n C₁₀H₈ = [tex]\frac{massa}{Mr}[/tex]

n C₁₀H₈ = [tex]\frac{6,4}{128}[/tex]

n C₁₀H₈ = 0,05 mol


n C₆H₆ = [tex]\frac{massa}{Mr}[/tex]

n C₆H₆ = [tex]\frac{500}{78}[/tex]

n C₆H₆ = 6,41 mol

Menentukan fraksi mol pelarut

Xp = [tex]\frac{np}{nt~+~np}[/tex]

Xp = [tex]\frac{6,41}{0,05~+~6,41}[/tex]

Xp =  [tex]\frac{6,41}{6,46}[/tex]

Xp = 0,99

Menentukan tekanan uap jenuh larutan

P = P° . Xpelarut

P = 77,9 x 0,99

P = 77,121 mmHg

Sifat koligatif merupakan sifat larutan yang bergantung pada jumlah zat terlarut dan tidak tergantung pada jenis zat. Zat terlarut hanya dikelompokkan antara zat terlarut elektrolit dan zat terlarut non elektrolit. Sifat koligatif larutan terdiri atas penurunan tekanan uap, penurunan titik beku, kenaikan titik didih, dan tekanan osmotik.

Adanya zat terlarut dalam suatu pelarut cair mengakibatkan penurunan tekanan uap jenuh. Hal ini sesuai dengan Hukum Raoult yang menyatakan "besarnya tekanan uap larutan sebanding dengan fraksi mol pelarut dalam larutan". Semakin banyak zat terlarut dalam larutan, semakin sedikit bagian pelarutnya sehingga tekanan uap juga semakin rendah. Tekanan uap jwnuh larutan dirumuskan sebagai berikut :

P = Xp · P°

P = tekanan uap jenuh larutan (cmHg)

Xp = fraksi mol zat pelarut, dirumuskan dengan np / (nt + np)

P° = tekanan uap jenuh air murni

Pembahasan

Diketahui :

P = 24,5 mmHg = 2,45 cmHg

P° = 25 mmHg = 2,5 cmHg

dengan menggunakan persamaan hukum Raoult

P = Xp · P°

2,45 cmHg = Xp · 2,5 cmHg

Xp = 2,45 cmHg / 2,5 cmHg

Xp = 0,98

Xp + Xt = 1

0,98 + Xt = 1

Xt  = 1 - 0,98 = 0,02 (atau 2/100)

Xt = nt ÷ (nt + np)

0,02 = nt ÷ (nt + np)

Sifat Koligatif Larutan Nonelektrolit

Sifat koligatif larutan adalah salah satu sifat larutan yang tidak bergantung pada jenis zat terlarut tetapi hanya tergantung pada jumlah zat terlarut. Yang merupakan sifat koligatif larutan adalah

Penurunan Tekanan Uap Jenuh (∆P)

Tekanan uap jenuh adalah tekanan uap yang ditimbulkan oleh uap jenuh suatu zat pada saat tercapai kondisi kesetimbangan. Sedangkan tekanan yang ditimbulkan oleh uap jenuh air disebut tekanan uap jenuh air. Penurunan tekanan uap jenuh larutan (∆P) adalah selisih dari tekanan uap jenuh pelarut murni (Po) dengan tekanan uap jenuh larutan (P).

∆P =  P° – P


Tahun 1830-1901, menurut Hukum Raoult, menyatakan tekanan uap jenuh larutan sama dengan tekanan uap jenuh pelarut murni dikalikan dengan fraksi mol pelarut. Sedangkan penurunan tekanan uap jenuh larutan sama dengan tekanan uap jenuh pelarut murni dikalikan dengan fraksi mol terlarut.


 P =  P° . Xpelarut

∆P = P° . Xterlarut


Keterangan:

∆P   = penurunan tekanan uap jenuh

P° = tekanan uap jenuh pelarut air murni

Xterlarut   = fraksi mol zat terlarut

Xpelarut   = fraksi mol zat pelarut


Fraksi Mol (x)

Fraksi mol menyatakan perbandingan mol zat terlarut atau zat pelarut dengan jumlah mol campuran (terlarut dan pelarut).

Rumus :

Xt = nt / nt+np

atau        

Xp = np / nt+np

maka,

Xt  +  Xp  =   1


Dengan :

Xt = fraksi mol zat terlarut                              

Xp = fraksi mol zat pelarut

nt = mol zat terlarut                                        

np = mol zat pelarut



Diket :

massa C₁₀H₈ = 6,4 gram

massa C₆H₆ = 500 gram

P° = 77,9 mmHg


Ditanya :

P


Jawab :

Menentukan Mr C₁₀H₈

Mr C₁₀H₈ = 10. Ar C + 8. Ar H

Mr C₁₀H₈ = 10. 12 + 8. 1

Mr C₁₀H₈ = 120 + 8

Mr C₁₀H₈ = 128

Menentukan Mr C₆H₆

Mr C₆H₆ = 6. Ar C + 6. Ar H

Mr C₆H₆ = 6. 12 + 6. 1

Mr C₆H₆ = 72 + 6

Mr C₆H₆ = 78

Menentukan mol naftalena dan benzena

n C₁₀H₈ = [tex]\frac{massa}{Mr}[/tex]

n C₁₀H₈ = [tex]\frac{6,4}{128}[/tex]

n C₁₀H₈ = 0,05 mol


n C₆H₆ = [tex]\frac{massa}{Mr}[/tex]

n C₆H₆ = [tex]\frac{500}{78}[/tex]

n C₆H₆ = 6,41 mol

Menentukan fraksi mol pelarut

Xp = [tex]\frac{np}{nt~+~np}[/tex]

Xp = [tex]\frac{6,41}{0,05~+~6,41}[/tex]

Xp =  [tex]\frac{6,41}{6,46}[/tex]

Xp = 0,99

Menentukan tekanan uap jenuh larutan

P = P° . Xpelarut

P = 77,9 x 0,99

P = 77,121 mmHg
