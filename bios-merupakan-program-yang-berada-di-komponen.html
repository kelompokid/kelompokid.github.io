4. bagaimana cara menjalankan program BIOS?


ada F1, F2, dan delete dan tergantung merk pc nyaPertama kali nyalakan monitor kemudian unit sistem. Amatilah operasi melalui layar dan dengar apakah ada tanda suara beep dari speaker sistem. Sistem akan otomatis jalan melalui proses Power On Self Test (POST) termasukvideo bios checking, test RAM, dan biasanya melaporkan beberapa komponen yang terpasang. Jika fatal error terjadi saat POST, kemungkinan tidak akan terlihat apa-apa di Layar dan sistem akan berbunyi beep dalam beberapa kali, hal ini mengindikasikan adanya kesalahan tertentu. Jika tidak ada fatal error, maka akan terlihat display POST di layar. Anda dapat menekan tombol atau kombinasi tombol untuk melakukan interupsi dari urutan boot normal yang tergantung pada tipe BIOS motherboard, seperti Phonix, AMI, AWARD atau yang lain sehingga akan keluar layar program Setup yang digunakan untuk memasukkan beberapa informasi sistem yang penting. Secara normal, sistem akan di indikasikan melalui layar display yang diaktifkan melalui program setup BIOS saat POST, akan tetapi jika tidak, teliti manual motherboard untuk mencari kombinasi tombol untuk memasukkan Setup BIOS. Beberapa tombol yang biasanya digunakan untuk memasukkan BIOS Setup adalah F1, F2, F10, Esc, Ins dan DEL. Setelah program Setup berjalan, gunakan menu pada program Setup untuk memasukkan tanggal sekarang dan waktunya, setting Hard Drive, jenis drive, card video, setting keyboard, dan lain-lain. Beberapa BIOS motherboard dapat melakukan deteksi otomatis pada hard drive, sehingga tidak perlu secara manual memasukkan beberapa parameter. Memasukkan informasi hard drive adalah sangat penting saat anda membangun sistem anda yang baru. Beberapa fitur BIOS modern mempunyai autodetect atau type untuk mengatur drive. Setelah meneliti semua setting di Setup BIOS, ikutilah perintah di Layar atau manual motherboard untuk menyimpan setting dan keluar dari menu Setup.CATATAN
Beberapa sistem mempunyai program setup yang disertakan pada software ROM BIOS. Program setup built-in dapat diaktifkan dengan menekan kombinasi tombol saat POST berjalan. Beberapa sistem memperlihatkan prompt di Layar saat POST mengindikasikan tekanan tombol untuk Setup BIOS.

10. komponen yg dapat di set melalui BIOS


Jawaban:

 

Komponen yg dapat diset melalui BIOS adalah:

 

1.    Sistem operasi yang diunduh

2.    Penyimpanan yang digunakan

3.    Pengaturan waktu sistem

4.    Pasword pengaman BIOS 

5.    Frekuensi dan voltase komputer

6.    Komponen yang dapat digunakan komputer

 

Pembahasan:

 

BIOS (Basic Input/Output System) adalah perangkat lunak yang melakukan penyiapan atau inisialisasi perangkat keras saat suatu komputer dinyalakan atau pada proses booting.

 

BIOS terdapat pada setiap komputer dan pertama kali digunakan pada IBM PC pad tahun 1970an.

 

BIOS dapat digunakan untuk melakukan konfigurasi pada komputer antara lain untuk:

 

1.    Sistem operasi yang diunduh

 

Suatu komputer dapat memiliki beberapa sistem operasi yang ada di penyimpanannya, misalnya Microsoft Windows di drive C dan Linux di drive D. Dengan BIOS, kita bisa mengatur, sistem operasi mana yang diunduh untuk dijalankan pada saat booting.

 

Selain itu, pengunduhan sistem operasi ini juga dilakukan saat akan menginstal sistem operasi baru. Sistem operasi yang akan diinstal biasanya disimpan di penyimpanan portabel seperti CD, DVD atau flash disk. Dengan BIOS, sistem operasi ini dapat diunduh dan kemudian bisa diinstal.

 

2.    Penyimpanan yang digunakan

 

Pada komputer dapat dipasang beberapa hard disk, atau terdapat satu hard disk yang dipartisi. Dengan BIOS, kita bisa mengatur mana hard disk yang digunakan dan terlihat oleh sistem operasi. 

 

3.    Pengaturan waktu sistem

 

Dengan BIOS, kita bisa mengatur zona waktu yang digunakan oleh komputer. Misalnya, awalnya komputer diatur menggunakan waktu Amerika Serikat, dengan BIOS kita bisa atur agar menggunakan Waktu Indonesia Barat.

 

4.    Pasword pengaman BIOS 

 

Untuk menghindari akses atau perubahan pengaturan oleh orang yang tidak berwenang, dan untuk mengamankan komputer, kita bisa memberikan password pad BIOS.

 

5.    Frekuensi dan voltase komputer

 

Pengaturan frekuensi dan volkase digunakan pada saat ingin meningkatkan performa komputer hingga di atas perfoma standar, atau disebut dengan overclocking.

 

6.    Komponen yang dapat digunakan komputer

 

Pada BIOS, kita bisa mengatur koneksi dari komputer pada perangkat peripheral, seperti kartu jaringan, printer atau perangkat input sperti floppy disk, CD ROM, mouse dan keyboard.

17. komponen apa saja yang di atur dalam BIOS?


BIOS (Basic Input Output System)    
            BIOS adalah singkatan dari Basic Input Output System. merupakan suatu software (ditulis dalam bahasa assembly) yang mengatur fungsi dasar dari perangkat keras (hardware) komputer. BIOS tertanam dalam sebuah chip memory (ROM ataupun Flash Memory berbahan Comlpimentari Metal Oxide Semiconductor (CMOS) yang terdapat pada motherboard. Sebuah baterai yang biasa disebut sebagai baterai CMOS berfungsi untuk menjaga agar tanggal dan settingan lainnya yang telah kita set pada BIOS tidak hilang atau kembali ke konfigurasi awal meskipun komputer dimatikan.
Fungsi utama BIOS           Fungsi utama BIOS adalah untuk memberikan instruksi yang dikenal dengan istilah POST (Power On Selft Test) yaitu perintah untuk menginisialisasi dan identifikasi perangkat sistem seperti CPU, RAM, VGA Card, Keyboard dan Mouse, Hardisk drive, Optical (CD/DVD) drive dan hardware lainnya pada saat komputer mulai booting.
Cara Kerja BIOS            Cara kerja BIOS adalah dimulai dengan proses inisialisasi, dimana dalam proses ini kita bisa melihat jumlah memory yang terinstall, jenis hardisk dan kapasitasnya dan sebagainya. BIOS kemudian akan mencari, menginisialisasi dan menampilkan informasi dari Graphics Card. Kemudian akan mengecek device ROM lain seperti hardisk dan kemudian melakukan pengetesan RAM yaitu memory count up test. Setelah  semua test komponen berhasil dilakukan, BIOS kemudian akan mencari lokasi booting device dan Sistem Operasi.
MENU-MENU PENGATURAN PADA BIOS
Cara Akses BIOSUntuk mengakses BIOS dapat kita lakukan dengan menekan tombol tertentu (biasanya tombol Delete atau F2) pada Keyboard pada saat pertama kali komputer dinyalakan. Akan terdapat tulisan misalnya "Pres F2 to enter setup", maka langsung saja tekan tombol F2 berulang-ulang.
Cara seting atau konfigurasi BIOS ini berbeda-beda tergantung dari vendor pembuatnya, disini saya akan menampilkan menu-menu pada BIOS yang umum kita temui yaitu Phoenix Award BIOS. Menu utama pada BIOS ini adalah :1.    Standard CMOS Features, untuk seting tanggal dan melihat hardisk yang terdeteksi, dll.2.    Advanced BIOS Features, pengaturan boot device priority (pilihan device untuk pertama booting) dapat diset disini.3.    Advanced Chipset Features4.    Integrated Peripherals5.    Power Management Setup, pembagian tegangan untuk masing-masing periferal dimana ini sering digunakan untuk overclocking6.    PnP/PCI Configuration, mengkonfigurasi clock/kecepatan dari setiap perangakat yang terpasang pada port PCI/PnP,misal vga pci ,lancard pci, wirelles port pci, HDMI,dll7.    PC Health Status, kita bisa cek temperatur dan tegangan dari Power Suplly disini.8.    Load Fail-Safe Defaults (Load Factory Setting), pilih menu ini untuk mengembalikan seluruh setingan ke mode asalnya (default).9.    Load Optimized Defaults, mengembalikan settingan optimal yang direkomendasikan oleh bawaan pabrik.10.    Set Supervisor Password, memberi kata sandi agar tidak sembarangan user mampu mengubah-ubah settingan BIOS11.    Set User Password, 12.    Save & Exit Setup, menyimpan settingan BIOS lalu keluar.13.    Exit Without Saving , keluar dari layar bios tanpa menyimpan settingan.
