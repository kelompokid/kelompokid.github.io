Besar sudut <POQ adalah 38⁰

Penjelasan dengan langkah-langkah:

besar sudut pelurus yaitu 180⁰

<POQ + <QOR + <ROS = 180⁰

2x⁰ + 3x⁰ + 85⁰ = 180⁰

5x⁰ = 180⁰ - 85⁰

5x⁰ = 95⁰

x = 95⁰/5⁰

x = 19⁰

=========

subsitusikan x = 19⁰ ke :

<POQ = 2x = 2(19)⁰ = 38⁰
