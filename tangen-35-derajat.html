21. nilai dari tangen 210 derajat adalah


Nilai dari tangen 210 derajat adalah ...

Perbandingan sudut dalam trigonometri.

Dalam trigonometri, besar sudut α dibagi menjadi 4 kelompok, yaitu :

Kuadran I : 0° < α < 90° (0 < α < π/2)

sin (90° - α) = cos α

cos (90° - α) = sin α

tan (90° - α) = cotan α

cotan (90° - α) = tan α

secan (90° - α) = sec α

sec (90° - α) = cosec α

Kuadran II : 90° < α < 180° (π/2 < α < π)

sin (180° - α) = sin α

cos (180° - α) = -cos α

tan (180° - α) = -tan α

cotan (180° - α) = -cotan α

secan (180° - α) = secan α

sec (180° - α) = -sec α

Kuadran III : 180° < α < 270° (π < α < 3π/2)

sin (270° - α) = -cos α

cos (270° - α) = -sin α

tan (270° - α) = cotan α

cotan (270° - α) = tan α

secan (270° - α) = -sec α

sec (270° - α) = -cosec α

Kuadran IV : 270° < α < 360° (3π/2 < α < 2π)

sin (360° - α) = sin (-α) = -sin α

cos (360° - α) = cos (-α) = cos α

tan (360° - α) = tan (-α) = -tan α

cotan (360° - α) = cotan (-α) = -cotan α

secan (360° - α) = secan (-α) = -secan α

sec (360° - α)= sec (-α) = sec α

Pembahasan

Tan 210° berada di kuadran III

= Tan (270 - 60)°

= Cotan 60°

= [tex]\frac{1}{3} \sqrt{3}[/tex]

Jadi Tan 210° = [tex]\frac{1}{3} \sqrt{3}[/tex].

22. berapakah -tangen 15 derajat


-tangen 15° = - tan (45-30)°

[tex] = - \frac{tan45 - tan30}{1 + tan45.tan30} \\ = - \frac{1 - \frac{1}{3} \sqrt{3} }{1 + 1. \frac{1}{3} \sqrt{3} } \\ = - \frac{ \frac{3 - \sqrt{3} }{3} }{ \frac{3 + \sqrt{3} }{3} } \\ = - \frac{3 - \sqrt{3} }{3} \times \frac{3}{3 + \sqrt{3} } \\ = - \frac{3 - \sqrt{3} (3 - \sqrt{3)} }{9 - 3} \\ = - \frac{9 - 6 \sqrt{3 } + 3}{6} \\ = - (2 - \sqrt{3} ) \\ = \sqrt{3 } - 2[/tex]

Jadi nilai - tan 15° adalah akar 3 - 2 atau - 2 + akar 3

25. nilai dari tangen 75 derajat


Tangen 75° adalah (2+√3).

Pembahasan

Trigonometri adalah salah satu ilmu yang digunakan untuk mempelajari hubungan garis dan sudut dalam segitiga.

Rumus jumlah dan selisih sudut:

cos (a+b) = cos a. cos b - sin a. sin bcos (a-b) = cos a. cos b + sin a. sin bsin (a+b) = sin a. cos b + sin b. cos asin (a-b) = sin a. cos b - sin b. cos atan (a+b) = (tan a + tan b)/(1 - tan a. tan b)tan (a-b) = (tan a - tan b)/(1 + tan a. tan b)

Penyelesaian

tan 75°

= tan (45°+30°)

=(tan 45° + tan 30°)/(1 - tan 45° .tan 30°)

[tex]=\frac{1+\frac{1}{3}\sqrt{3}}{1-(1)(\frac{1}{3}\sqrt{3})}\\=\frac{1+\frac{1}{3}\sqrt{3}}{1-\frac{1}{3}\sqrt{3}}\\=\frac{1+\frac{1}{3}\sqrt{3}}{1-\frac{1}{3}\sqrt{3}}.\frac{1+\frac{1}{3}\sqrt{3}}{1+\frac{1}{3}\sqrt{3}}\\=\frac{1+\frac{2}{3}\sqrt{3}+\frac{1}{3}}{1-\frac{1}{3}}\\=\frac{\frac{4}{3}+\frac{2}{3}\sqrt{3}}{\frac{2}{3}}\\=\frac{\frac{2}{3}(2+\sqrt{3})}{\frac{2}{3}}\\=2+\sqrt{3}[/tex]

28. nilai dari tangen 75 derajat


Pembahasan

Diminta untuk menentukan nilai dari tan 75⁰.

Persoalan ini merupakan penerapan dari rumus tangen jumlah dua sudut. Seperti kita ketahui bahwa sudut 75⁰ bukanlah sudut istimewa, namun sudut ini dapat terbentuk dari jumlah dua sudut istimewa yakni 30⁰ dan 45⁰.

Rumus tangen jumlah dua sudut

[tex] \boxed{tan (a + b) = \frac{tan \ a + tan \ b}{1 - tan \ a.tan \ b}} [/tex]

Sekadar mengingat kembali untuk rumus tangen selisih dua sudut, yaitu

[tex] \boxed{tan (a - b) = \frac{tan \ a - tan \ b}{1 + tan \ a.tan \ b}} [/tex]

Pengerjaan

tan 75⁰ = tan (30 + 45)⁰

[tex] = \frac{tan \ 45^0 \ + \ tan \ 30^0}{1 \ - (tan \ 45^0 . tan \ 30^0)} [/tex]

Ingat, tan 45⁰ = 1 dan [tex] tan \ 30^0 = \frac{1}{3} \sqrt{3} [/tex]

[tex] = \frac{1 \ + \frac{1}{3}\sqrt{3}}{1 \ - (1)(\frac{1}{3}\sqrt{3})} [/tex]

[tex] = \frac{1 \ + \frac{1}{3}\sqrt{3}}{1 \ - \frac{1}{3}\sqrt{3}} [/tex]

Kalikan pembilang dan penyebut dengan tiga untuk menghilangkan bentuk pecahan.

[tex] = \frac{3 \ + \sqrt{3}}{3 \ - \sqrt{3}} [/tex]

Kalikan akar sekawan untuk rasionalisasi.

[tex] = \frac{3 \ + \sqrt{3}}{3 \ - \sqrt{3}} \times \frac{3 \ + \sqrt{3}}{3 \ + \sqrt{3}} [/tex]

[tex] = \frac{9 \ + \ 3\sqrt{3} \ + \ 3\sqrt{3} \ + 3}{9 \ - \ 3} [/tex]

[tex] = \frac{12 \ + \ 6\sqrt{3}}{6} [/tex]

Penyebut 6 adalah pembagi untuk para pembilang.

Dari langkah-langkah pengerjaan, diperoleh hasil [tex] \boxed{tan \ 75^0 = 2 \ + \sqrt{3}} [/tex]
