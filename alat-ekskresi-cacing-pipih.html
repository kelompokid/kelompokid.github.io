27. alat ekskresi pada cacing pipih dan pita adalah....tolong penjelasannya


Kelas :              XI
Kategori :        Sistem ekskresi
Sub kategori : Alat ekskresi pada invertebrata
Kata kunci :     Cacing pipih

Pembahasan :

Alat eskresi pada cacing pipih disebut flame cell (sel api).

Dimana cairan tubuh yg melewati sel api akan disaring dan zat-zat sisa yg dikandungnya akan diserap oleh sel api. Gerakan bulu getar pada sel api akan mendorong zat air ke arah saluran gabungan. 

Melalui saluran gabungan itulah, akhirnya zat-zat sisa dibuang ke luar melalui lubang ekskresi.
berupa sel api yang terdapat di kanan dan kiri tubuhnya,setiap sel api terdapat rambut getar, lalu getaran tersebut akan mendorong sisa metabolisme ke luar tubuh melalui lubang nefridiopori (cacing pita dan pipih memiliki alat ekskresi yang sama)

30. Jelaskan sistem ekskresi pada planaria (cacing pipih)!


Cacing pipih mempunyai organ nefridium yang disebut sebagai protonefridium. Protonefridium tersusun dari tabung dengan ujung membesar mengandung silia. Di dalam protonefridium terdapat sel api yang dilengkapi dengan silia.
Tiap sel api mempunyai beberapa flagela yang gerakannya seperti gerakan api lilin. Air dan beberapa zat sisa ditarik ke dalam sel api. Gerakan flagela juga berfungsi mengatur arus dan menggerakan air ke sel api pada sepanjang saluran ekskresi. Pada tempat tertentu, saluran bercabang menjadi pembuluh ekskresi yang terbuka sebagai lubang di permukaan tubuh (nefridiofora). Air dikeluarkan lewat lubang nefridiofora ini
