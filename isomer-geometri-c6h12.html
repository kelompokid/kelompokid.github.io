2. sbutkan isomer geometri dari C6H12..??


Isomer geometri dari C₆H₁₂ (heksena), adalah:

cis 3-heksena,  

        H           H

           \       /

            C = C

           /         \

CH₂-CH₃       CH₂-CH₃

trans 3-heksena,  

        H           CH₂-CH₃

           \       /

            C = C

           /         \

CH₂-CH₃         H

Pembahasan:

Isomer geometri adalah senyawa-senyawa dengan rumus struktur dan rumus kimia yang sama, namun memiliki struktur ruang yang berberda.  

Struktur ruang memiliki variasi yang dapat disebabkan oleh adanya perbedaan posisi ikatan cis dan trans pada atom dan gugus yang terikat pada atom karbon dengan ikatan ganda.

Isomer geometri menyebabkan adanya senyawa dengan isomer geometri berstruktur cis dan trans.

Pada struktur cis, bagian panjang pendek (umumnya atom H) berada pada satu sisi. Sedangkan pada struktur trans, panjang pendek berada pada sisi berlawanan.

Pada C₆H₁₂ (heksena), rumus strukturnya adalah:

CH₃-CH₂-CH=CH-CH₂-CH₃

pada atom karbon ke-3 dan ke-4, perbedaan gugus terikat panjang dan pendek akan menyebabkan isomer geometri dengan senyawa trans dan cis, yaitu:

cis 3-heksena, dengan atom H kedua atom pada sisi yang sama

        H           H

           \       /

            C = C

           /         \

CH₂-CH₃       CH₂-CH₃

trans 3-heksena, dengan atom H kedua atom pada sisi yang berlawanan

        H           CH₂-CH₃

           \       /

            C = C

           /         \

CH₂-CH₃         H
