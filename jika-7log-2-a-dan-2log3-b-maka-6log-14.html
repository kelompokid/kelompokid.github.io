sifat" logaritma

1) ᵃlog (b.c) = ᵃlog b + ᵃlog c

2) ᵃlog b x ᵇlog c = ᵃlog c

3) ᵃlog (ᵇ/ₓ) = ᵃlog b - ᵃlog x

4) ᵃlog b = 1/ᵇlog a

5) ᵃlog (b/c) = - ᵃlog (c/b)

6) ᵃlog bⁿ = n. ᵃlog b

7) ᵃ²log b = ¹/₂. ᵃlog b

8) ᵃlog aⁿ = n

9) ᵃlog a = 1
