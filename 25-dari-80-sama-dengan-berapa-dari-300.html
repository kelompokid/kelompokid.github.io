14. Jika sebuah benda bermasssa 300 gr dipanaskan dari suhu 25 sampai 80 derajat celcius membutuhkan kalor membutuhkan 6435 joule. maka jenis benda tersebut adalah ....​


Kalor jenis benda tersebut adalah 390 J/kg°C.

Pembahasan

Rumus yang digunakan

[tex]\boxed{\boldsymbol{Q = m\cdot c\cdot \Delta T}}[/tex]

Keterangan :

Q = Kalor (Joule atau kal)

m = Massa benda (kg atau g)

c = Kalor jenis (kal/g°C atau J/kg°C)

∆T = Perubahan suhu (°C)

--

Diketahui

Massa benda : 300 grPerubahan suhu (∆T) = 80 - 25 = 55°CKalor (Q) : 6.435 J

Ditanya

Kalo jenis benda tersebut ?

Jawaban

Ubah massa benda yang bersatuan gr ke kg.

gr → kg = [tex]\frac{1}{1.00}[/tex]

[tex]\begin{aligned}\\&\rm \frac{300}{1.000}\\&\rm \frac{3\cancel{00}}{1.0\cancel{00}}\\&\rm \frac{3}{10}\\&\rm = 0,3~kg\end{aligned}[/tex]

Lalu, masukkan ke rumus.

[tex]\begin{aligned}\\&\rm Q = m\cdot c\cdot \Delta T\\&\rm 6.435 = 0,3\cdot c\cdot 55\\&\rm 6.435 = 16,5\cdot c\\&\rm c = \frac{6.435}{16,5}\\&\rm c = 6.435\div \frac{165}{10}\\&\rm c = 6.435\times \frac{10}{165}\\&\rm c = \frac{64.350}{165}\\&\rm c = 390~J/kg^{\circ}C\end{aligned}[/tex]

Maka, kalor jenis benda tersebut adalah 390 J/kg°C.

Pertanyaan :

Diketahui :

m (massa) = 300 gr = 0,3 kg

ΔT (perubahan suhu) = 80 – 25 = 55ºC

c (kalor jenis benda) = 4200J/Kg K

Ditanya : Q (jumlah kalor yang digunakan)?

Jawab :

[tex]\text{Q\:=\:m.c.} \Delta T \\ \text{Q}= (0,3)(4200)(55)\\ \text{Q}= 69300\:\text{Joule}[/tex]
