B. 3x² - 14x + 11 = 0

Penjelasan dengan langkah-langkah:

diket :

persamaan kuadrat (PK) : 3x² - 2x - 5 = 0

persamaan kuadrat baru (PKB) : x1 + 2 dan x2 + 2

Pertama, tentukan nilai abc :

a = 3

b = -2

c = -5

jika x1 + 2 dan x2 + 2 maka :

a(x - 2)² + b(x - 2) + c = 0

3(x - 2)² + (-2)(x - 2) + (-5) = 0

3.[x(x - 2) - 2(x - 2)] + (-2x + 4) - 5 = 0

3.[x² - 2x - 2x + 4] - 2x + 4 - 5 = 0

3.[x² - 4x + 4] - 2x - 1 = 0

3x² - 12x + 12 - 2x - 1 = 0

3x² + (-12x - 2x) + (12 - 1) = 0

3x² - 14x + 11 = 0

Jadi, pilihan yang tepat untuk persamaan kuadrat baru tersebut adalah B. 3x² - 14x + 11 = 0

Jawaban

B. 3x² - 6x + 15 = 0

Pembahasan

Bentuk umum dari persamaan kuadrat adalah Ax² + Bx + C = 0. Jadi, ruas sebelah kanan dijadikan sama dengan nol dengan kedua ruas dikurangi 2x² + 4x - 12.

Maka:

5x² - 2x + 3 = 2x² + 4x - 12

5x² - 2x + 3 - (2x² + 4x - 12) = 0

5x² - 2x + 3 - 2x² - 4x + 12 = 0

(5x² - 2x²) + (-2x - 4x) + (3 + 12) = 0

3x² - 6x + 15 = 0

Jadi, bentuk umum persamaan kuadrat

5x² - 2x + 3 = 2x² + 4x - 12 adalah

3x² - 6x + 15 = 0.
