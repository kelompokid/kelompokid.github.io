7. Hitunglah kadar nitrogen dalam senyawa senyawa berikut : (NH4)2 SO4


Langkah pertama; Hitung Mr Senyawa tersebut.
Data Ar: N = 14, H = 1, S = 32, O = 16

Dalam senyawa itu, N menyumbang Mr sebanyak 2 x 14 = 28 gram/mol.

%N (persen massa / kadar) = 28 / 132 x 100% = 21,21%

% N dalam senyawa (NH₄)₂SO₄ adalah 21%

% N dalam senyawa NH₄NO₃ adalah 35%

Pembahasan

[tex]\boxed{\boxed{\bold{Stoikiometri}}}[/tex]  

Dalam bahasa Yunani, kata stoicheion berarti unsur sedangkan stoikiometri (stoichiometry) berarti mengukur unsur meliputi massa atom dan massa rumus,kadar, konsentrasi, rumus empiris, rumus molekul, persamaan kimia serta semua yang berhubungan dengan reaksi kimia. Dalam stoikiometri, satuan yang digunakan untuk menyatakan jumlah partikel dalam zat dinamakan mol.  

[tex]\boxed{\boxed{\bold{Diket}}}[/tex]  

Senyawa

a. (NH₄)₂SO₄

b. NH₄NO₃

Ar H = 1

Ar N = 14

Ar O = 16

Ar S = 32

[tex]\boxed{\boxed{\bold{Ditanya}}}[/tex]  

Kadar nitrogen dalam senyawa

[tex]\boxed{\boxed{\bold{Jawab}}}[/tex]  

Kadar unsur dalam senyawa

Kadar unsur-unsur dalam suatu senyawa dapat ditentukan dari rumus molekul atau rumus empiris senyawa, menurut rumus :

[tex]\boxed{\boxed{\bold{\%A~dalam~A_p_B_q~=~\frac{p~x~Ar~A}{Mr~A_pB_q}~x~100\%}}}[/tex]

[tex]\boxed{\boxed{\bold{\%B~dalam~A_p_B_q~=~\frac{q~x~Ar~B}{Mr~A_pB_q}~x~100\%}}}[/tex]

==============================

Menentukan Mr (NH₄)₂SO₄

Mr (NH₄)₂SO₄ = 2. Ar N + 8, Ar H + 1. Ar S + 4. Ar O

Mr (NH₄)₂SO₄ = 2. 14 + 8. 1 + 1. 32 + 4. 16

Mr (NH₄)₂SO₄ = 28 + 8 + 32 + 64

Mr (NH₄)₂SO₄ = 132

==============================

Menentukan Mr NH₄NO₃

Mr NH₄NO₃ = 2. Ar N + 4. Ar H + 3. Ar O

Mr NH₄NO₃ = 2. 14 + 4. 1 + 3. 16

Mr NH₄NO₃ = 28 + 4 + 48

Mr NH₄NO₃ = 80

==============================

Menentukan kadar nitrogen dalam senyawa (NH₄)₂SO₄

% N = [tex]\frac{jumlah~atom~N~x~Ar~N}{Mr~(NH_4)_2SO_4}[/tex] x 100%

% N = [tex]\frac{2~x~14}{132}[/tex] x 100%

% N = [tex]\frac{28}{132}[/tex] x 100%

% N = ‭0,2121212121212121‬ x 100%

% N = 0,21 x 100%

% N = 21%

==============================

Menentukan kadar nitrogen dalam senyawa NH₄NO₃

% N = [tex]\frac{jumlah~atom~N~x~Ar~N}{Mr~NH_4NO_3}[/tex] x 100%

% N = [tex]\frac{2~x~14}{80}[/tex] x 100%

% N = [tex]\frac{28}{80}[/tex] x 100%

% N = ‭0,35‬ x 100%

% N = 35%

[tex]\boxed{\boxed{\bold{Kesimpulan}}}[/tex]  

% N dalam senyawa (NH₄)₂SO₄ = 21%
