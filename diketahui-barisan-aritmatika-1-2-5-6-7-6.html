Pembahasan

Barisan aritmatika adalah barisan dimana nilai selisih dua suku berurutan bersifat tetap. Selisih dua suku berurutan tersebut disebut dengan beda (b).

Rumus :

Un = a + (n - 1) b

Ut = ¹/₂ (a + Un)

Sn = ⁿ/₂ (a + Un)

Sn = ⁿ/₂ (2a + (n - 1)b)

Keterangan :

a = suku pertama

b = beda

Un = suku ke-n

Ut = suku tengah

Sn = Jumlah n suku pertama

1.

Diketahui :

U₄ = 6

U₁₀ = 42

Ditanya : beda

Jawab :

Un = a + (n - 1)b

U₄ = a + (4 - 1)b

6 = a + 3b

a = 6 - 3b

U₁₀ = a + (10 - 1)b

42 = 6 - 3b + 9b

6b = 42 - 6

6b = 36

b = 6

Jadi beda antar suku barisan tersebut adalah 6

2.

Diketahui : barisan 3,5,7,...,15.

Ditanya : suku tengah

Jawab :

Ut = ¹/₂ (a + Un)

Ut = ¹/₂ (3 + 15)

Ut = ¹/₂ (18)

Ut = 9

Jadi suku tengah barisan tersebut adalah 9.

3.

Diketahui :

U₅ = 8

U₉ = 20

Ditanya : a

Jawab :

Un = a + (n - 1) b

U₅ = a + (5 - 1)b

8 = a + 4 b

4b = 8 - a

U₉ = a + (9 - 1)b

20 = a + 8b

20 = a + 2 (4b)

a + 2 (8 - a) = 20

a + 16 - 2a = 20

- a = 20 - 16

- a = 4

a = - 4

Jadi suku pertama barisan tersebut adalah -4.

4.

beda = b = U₃ - U₂

Un = 9 + (n - 1) (-2)

Un = 9 + (-2n + 2)

Un = 11 - 2n

Jadi rumus suku ke-n barisan tersebut adalah 11 - 2n.
