2. Hitunglah massa zat berikut.​


a. Massa FeCl2

Menghitung Mr FeCl2

Mr = (1xAr Fe) + (2xAr Cl)

= (1x56) + (2x35,5)

= 56 + 71

= 127

Menghitung mol

mol = V x M

= 200 x 0,1

= 20 mmol

= 0,02 mol

Menghitung massa

massa = mol x Mr

= 0,02 x 127

= 2,54

Jadi, massa FeCl2 adalah 2,54 gram.

b. Massa Fe2(SO4)3

Menghitung Mr Fe2(SO4)3

Mr = (2xAr Fe) + (3xAr S) + (12xAr O)

= (2x56) + (3x32) + (12x16)

= 112 + 96 + 192

= 400

Menghitung mol

mol = partikel : 6,02.10^23

= 2,408.10^23 : 6,02.10^23

= 0,4 mol

Menghitung massa

massa = mol x Mr

= 0,4 x 400

= 160

Jadi, massa Fe2(SO4)3 adalah 160 gram.

c. Massa C5H12

Menghitung Mr C5H12

Mr = (5xAr C) + (12xAr H)

= (5x12) + (12x1)

= 60 + 12

= 72

Menghitung mol

mol = V : 22,4

= 5,6 : 22,4

= 0,25 mol

Menghitung massa

massa = mol x Mr

= 0,25 x 72

= 18

Jadi, massa C5H12 adalah 18 gram.

d. Massa N2O

Menghitung Mr N2O

Mr = (2xAr N) + (1xAr O)

= (2x14) + (1x16)

= 28 + 16

= 44

Menghitung partikel N2O

[tex]partikel \: n2o= \frac{v \: n2o}{v \: co2} \times partikel \: co2[/tex]

[tex]partikel \: n2o= \frac{2.5}{10} \times 3.01.{10}^{22} [/tex]

partikel N2O = 0,25 x 3,01.10^22

= 0,7525.10^22

= 0,07525.10^23

Menghitung mol

mol = partikel : 6,02.10^23

= 0,07525.10^23 : 6,02.10^23

= 0,0125 mol

Menghitung massa

massa = mol x Mr

= 0,0125 x 44

= 0,55

Jadi, massa N2O adalah 0,55 gram.

e. Massa Na2SO4.5H2O

Menghitung Mr

Mr = (2xAr Na) + (1xAr S) + (9xAr O) + (10xAr H)

= (2x24) + (1x32) + (9x16) + (10x1)

= 48 + 32 + 144 + 10

= 234

Menghitung mol

[tex]mol \: Na2SO4.5H2O = \frac{mr \: Na2SO4.5H2O}{10 \times ar \: h} \times mol \: h[/tex]

[tex]mol \: Na2SO4.5H2O = \frac{234}{10} \times 5[/tex]

mol Na2SO4.5H2O = 117 mol

Menghitung massa

massa = mol x Mr

= 117 x 234

= 27378

Jadi, massa Na2SO4.5H2O adalah 27.378 gram.

5. Hitunglah massa molekul relatif tiap zat berikut NH4Br!


Massa molekul relatif (Mr) dari NH₄Br (Ar N = 14; H = 1 dan Br = 80) adalah 98 g/mol.

Pembahasan :

Diketahui : Senyawa NH₄Br

                  Ar N = 14; H = 1 dan Br = 80

Ditanya : Mr NH₄Br ?  

Penyelesaian soal :

Untuk menghitung massa molekul relatif (Mr) dapat dihitung dengan menggunakan rumus massa molekul relatif (Mr) berikut ini :  

Mr AxBy = (x × Ar A) + (y × Ar B)  

Keterangan :

A = Unsur A  

x = Indeks unsur A  

B = Unsur B  

y = Indeks unsur B  

Secara perhitungan soal diatas maka menjadi :

Mr NH₄Br = {Ar N + Ar Br + (4 × Ar H)} g/mol

                = {14 + 80 + (4 × 1)} g/mol

                = {14 + 80 + 4} g/mol

                = 98 g/mol

⇒ Kesimpulan, maka massa molekul relatif (Mr) dari NH₄Br (Ar N = 14; H = 1 dan Br = 80) adalah 98 g/mol.

19. Hitunglah Massa Molekul Relatif (Mr) Zat Zat Berikut (a).NaNo2 (b).Fe2(So4) (c).MgSo67h2O


Massa molekul relatif (Mr) dari senyawa berikut

(a) NaNO₂ adalah 69 g/mol.

(b) Fe₂(SO₄)₃ adalah 400 g/mol.

(c) MgSO₄.7H₂O adalah 146 g/mol.

Penyelesaian Soal :

a. Diketahui : Ar Na = 23 g/mol

                      Ar N = 14 g/mol

                      Ar O = 16 g/mol

Ditanya : Mr NaNO₂ ?

Jawab :

Mr NaNO₂ = Ar Na + Ar N + (2 × Ar O)

                  = 23 g/mol + 14 g/mol + (2 × 16 g/mol)

                  = 23 g/mol + 14 g/mol + 32 g/mol

                  = 69 g/mol

b. Diketahui : Ar Fe = 56 g/mol

                      Ar S = 32 g/mol

                      Ar O = 16 g/mol

Ditanya : Mr Fe₂(SO₄)₃ ?

Jawab :

Mr Fe₂(SO₄)₃ = (2 × Ar Fe) + (3 × Ar S) + (12 × Ar O)

                     = (2 × 56 g/mol) + (3 × 32 g/mol) + (12 × 16 g/mol)

                     = 112 g/mol + 96 g/mol + 192 g/mol

                     = 400 g/mol

c. Diketahui : Ar Mg = 24 g/mol

                      Ar S = 32 g/mol

                      Ar O = 16 g/mol

                      Ar H = 1 g/mol

Ditanya : Mr MgSO₄.7H₂O ?

Jawab :

Mr MgSO₄.7H₂O = Ar Mg + Ar S + (11 × Ar O) + (14 × Ar H)

                            = 24 g/mol + 32 g/mol + (11 × 16 g/mol) + (14 × 1 g/mol)

                            = 24 g/mol + 32 g/mol + 176 g/mol + 14 g/mol

                            = 146 g/mol

20. Hitung jumlah dari massa zat berikut : a. 0,5 mol NaOH b. 0,8 mol CaO


Jawaban:

1. STOIKIOMETRI 1

2. Persamaan Reaksi 2  Reaksi kimia selalu berhubungan dengan persamaan reaksi.  Persamaan reaksi menunjukkan jenis dan keadaan fisik zat-zat pereaksi dan hasil reaksi yang dinyatakan dengan rumus struktur berikut tanda fasenya.

3. Hukum Lavoisier (Hukum Kekekalan Massa) 3  Penulisan persamaan reaksi harus menyatakan hubungan kuantitatif antara zat-zat pereaksi dan zat-zat hasil reaksi.  Hubungan kuantitatif dalam reaksi kimia pertama kali dikemukakan oleh Antonie Laurent Lavoisier (1743-1794), yaitu : “ Dalam setiap reaksi kimia jumlah massa zat-zat sebelum reaksi dan sesudah reaksi adalah sama.”

4. Hukum Kekekalan Massa 4 Hukum kekekalan massa merupakan pedoman untuk menyetarakan reaksi, sehingga dalam suatu persamaan reaksi diperoleh jumlah atom-atom zat yang bereaksi sama dengan jumlah atom-atom zat hasil reaksi. Persamaan reaksi harus disetarakan dengan cara menuliskan koefesien di depan rumus zat dan zat hasil reaksinya.

5. Perbandingan Koefisien Reaksi 5 Perbandingan koefisien reaksi dapat menyatakan : 1. Perbandingan jumlah partikel-partikel zat dalam suatu persamaan reaksi. 2. Perbandingan jumlah mol zat yang terlibat dalam reaksi, hal ini disebabkan jika jumlah partikel-partikel zat sama maka jumlah molnya juga sama. 3. Perbandingan volume zat yang terlibat dalam reaksi, jika zat-zat berwujud gas dan diukur pada temperatur yang sama, sesuai hipotesis Avogadro.

6. Contoh 1 6 Mg(s) + 2HCl(aq)  MgCl2(aq) + H2(g) Persamaan reaksi tersebut menyatakan : 1 atom Mg(s) bereaksi dengan 2 molekul HCl(aq) menghasilkan 1 molekul MgCl2(aq) dan 1 molekul H2(g) . Maka perbandingan molnya : mol Mg : mol HCl : mol MgCl2 : mol H2 1 : 2 : 1 : 1

7. Contoh 2 7 N2(g) + 3H2(g)  2NH3(g) Persamaan reaksi tersebut menyatakan : 1 mol N2(g) bereaksi dengan 3 molekul H2(g) menghasilkan 2 molekul NH3(g) . Karena reaksi tersebut seluruhnya berwujud gas, maka perbandingan volumenya : volume N2 : volume H2 : volume NH3 1 : 3 : 2

8. Jenis Reaksi 8 1. Reaksi Penetralan. H2SO4(aq) + 2KOH(aq)  K2SO4(aq) + 2H2O(l) Persamaan reaksi ionnya : 2H+ (aq) + SO4 -2 (aq) + 2K+ (aq) + 2OH- (aq)  2K(aq) + SO4 -2 (aq) + 2H2O(l)

9. Jenis Reaksi 9 2. Reaksi Pengendapan. 2NaCl(aq) + Pb(NO3)2(aq)  PbCl2(s) + 2NaNO3 - (aq) Persamaan reaksi ionnya : 2Na+ (aq) + 2Cl- (aq) + Pb+2 (aq) + 2NO3 - (aq)  PbCl2(s) + 2Na+ (aq) + 2NO3 - (aq)

10. Jenis Reaksi 10 3. Reaksi yang Menghasilkan Gas Hidrogen. Mg(s) + 2HCl(aq)  MgCl2(aq) + H2(g) Persamaan reaksi ionnya : Mg(s) + 2H+ (aq) + 2Cl- (aq)  Mg+2 (aq) + 2Cl- (aq) + H2(g)

29. Hitunglah massa molar masing-masing zat berikut? H2SO4


Jawaban:

H2SO4 = 98

Penjelasan:

Massa molekul relatif (Mr) adalah jumlah massa atom relatif dari seluruh atom penyusun molekul. Senyawa H2SO4 tersusun dari 2 atom H, 1 atom S dan 4 atom O. Sehingga rumus Mr H2SO4 adalah (2 x Ar H) + (1 x Ar S) + (4 x Ar O).

Nilai Mr H2SO4 = (2 x 1) + (1 x 32) + (4 x 16) = 98.
