27. prediksi soal UN matematika 2016/2017 persamaan garis yg melalui titik (-2,-3) dan tegak lurus terhadap garis 4x - 5y + 2 = 0 adalah ... a) -5x + 4y + 22 = 0 b) 5x + 4y + 22 = 0 c) 5x - 4y + 22 = 0 d) 5x + 4y - 22 = 0


4x -5y +2 = 0
m = -4/-5 = 4/5
tegak lurus,, m = -5/4
melalui (-2,-3)

y -(-3) = -5/4(x-(-2))
4(y+3) = -5(x+2)
4y + 12 = -5x -10
5x + 4y +12 +10 = 0
5x +4y +22 = 0

jawaban : b

Mapel : Matematika
Kelas : VIII SMP
Bab : Persamaan Garis Lurus

Pembahasan :
4x - 5y + 2 = 0

Gradien 1 = [tex]- \frac{x}{y} [/tex]
Gradien 1 = [tex] \frac{-4}{-5} [/tex]
Gradien 1 = [tex] \frac{4}{5} [/tex]

Karena TEGAK LURUS, maka :
M1  . M2 = -1
M2 = [tex] \frac{-1}{M1} [/tex]
M2 = [tex] \frac{-1}{ \frac{4}{5} } [/tex]
M2 = [tex] \frac{-5}{4} [/tex]

Persamaan Garis Lurusnya :
y - y1 = M2(x - x1)
y - (-3) = -5/4 (x + 2)
y + 3 = -5/4 (x + 2)
4y + 12 = -5(x + 2)
4y + 12 = -5x - 10
[tex]5x + 4y + 22 = 0[/tex]
