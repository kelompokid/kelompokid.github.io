11. everyone in this house.....going to see the concert tonight = a.were b.are c.was d.is​


Jawab: D. is

Everyone in this house.....going to see the concert tonight

a.were = ToBe utk Past (Plural)

b.are = ToBe utk Present (singular)

c.was = ToBe utk Past (singular)

d.is​ = ToBe utk Future (singular)

Pembahasan:

Alasannya kenapa memilih D. is

✅ Everyone bentuk singular

✅ Ada kata Going To dan Tonight maka gunakan Future (going to) yg memiliki pola/rumus:

(+) S + ToBe (is/am/are) + going to + V1 + O

= Everyone in this house [is] going to see the concert tonight

(Setiap orang di rumah ini akan melihat/menonton konser malam ini)

Keterangan:

* Subject = Everyone in this house

* To Be = is (Everyone=singular)

* Going To

* Verb 1 = see

* Object = the concert

* Adverb of time = Tonight

==== Semoga Membantu ====

Mapel: Bahasa Inggris

Kelas: SMP

Materi: Future (going to)

Kode Mapel: 5
