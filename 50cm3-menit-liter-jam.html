14. 50cm3/jam= liter/jam​


Jawaban:

50cm3/jam= liter/jam

Penjelasan dengan langkah-langkah:

50 cm³/menit = 3 liter/jam.

Pembahasan

Satuan debit adalah perbandingan dari satuan volume dengan satuan waktu.

Debit adalah kecepatan aliran suatu zat cair persatuan waktu

Rumus-rumus yang digunakan dalam perbandingan debit :

Debit = volume : waktu

Volume = debit x waktu

Waktu = volume : debit

22. 28cc + 50cm3 =......ml


Jawab:

hasil dari 28 cc ditambah 50 cm³ dalam satuan akhir ml adalah 78 ml

Penjelasan dengan langkah-langkah:

28 cc + 50 cm³ =......ml

___

1 cc = 1 x 1 = 1 ml

28 cc = 28 x 1 = 28 ml

1 cm³ = 1 x 1 = 1 ml

50 cm³ = 50 x 1 = 50 ml

28 cc + 50 cm³

= 28 ml + 50 ml

= 78 ml
