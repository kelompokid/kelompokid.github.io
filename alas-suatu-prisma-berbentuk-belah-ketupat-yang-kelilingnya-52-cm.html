27. keliling alas sebuah prisma yang berbentuk belah ketupat adalah 52 cm dan panjang salah satu diagonalnya adalah 24 cm jika tinggi prisma adalah 20 cm maka volume prisma tersebut adalah​


JAWABAN

[tex]V = 2.400 \ cm^3[/tex]

PEMBAHASAN

Diketahui :

[tex]K = 52 \ cm\\d_1 = 24 \ cm\\t_{prisma} = 20 \ cm[/tex]

Ditanya :

V = ...?

PENYELESAIAN SOAL

Pertama, mencari panjang sisi miring belah ketupat pada prisma belah ketupat

[tex]K \ \ \ \ \ = 4s\\52 \ \ \ \ \ = 4s\\\frac{52}{4} \ \ \ \ \ = s\\13 \ cm = s[/tex]

Kedua, mencari panjang diagonal yang lain belah ketupat pada prisma belah ketupat

[tex](d_2)^2 = (\sqrt{s^2 - (\frac{1}{2} \ d_1)^2}) \ 2\\(d_2)^2 = (\sqrt{13^2 - (\frac{1}{2} \ 24)^2}) \ 2\\(d_2)^2 = (\sqrt{169 - 12^2}) \ 2\\(d_2)^2 = (\sqrt{169 - 144}) \ 2\\d_2 \ \ \ \ = \sqrt{25} \ x \ 2\\d_2 \ \ \ \ = 5 \ x \ 2\\d_2 \ \ \ \ = 10 \ cm[/tex]

Terakhir, menghitung volume prisma belah ketupat

[tex]V = L_a \ x \ t_{prisma}\\V = (\frac{1}{2} \ d_1 \ x \ d_2) \ 20\\V = (\frac{1}{2} \ 24 \ x \ 10) \ 20\\V = (12 \ x \ 10) \ 20\\V = 120 \ x \ 20\\V = 2.400 \ cm^3[/tex]
