5. Rel kereta api tidak dipasang rapat agar


Rel kereta api tidak dipasang dengan rapat, hal ini dikarenakan agar Mencegah terjadinya pemuaian ketika pada siang hari dan mencegah penyusutan pada malam hari.

_______________________

Pembahasan

Perubahan ukuran pada tiap benda dibagi menjadi 2 yaitu pemuaian dan penyusutan.

1.) Pemuaian

Pemuaian adalah suatu benda yang mengalami naiknya suhu ketika dipanaskan sehingga bertambahnya ketika dipanaskan.

2.) Penyusutan

Hal ini terjadi karena suatu benda mengalami penurunan suhu ketika benda tersebut menjadi dingin sehingga mengalami berkurangnya ukuran pada benda.

Jenis-jenis pemuaian

Pemuaian terbagi menjadi beberapa jenis yaitu:

A. Pemuaian panjang

Pemuaian panjang terjadi pada saat mengalami perubahan yang panjang ketika dipanaskan pada benda yang mengalami kenaikan suhu. Hal ini karena terjadi pada zat-zat yang padat. Misalnya paku, batang besi, dan lain lain.

Pemuaian terjadi pada rel kereta api

Pemuaian terjadi pada rel kereta api karena suhu tersebut mengalami kenaikan akibat dipanaskan oleh matahari sehingga rel tersebut menjadi bengkok maupun melengkung. Agar rel kereta api tersebut tidak bengkok rel tersebut maka rel kereta api dipasang secara renggang.

Berbanding lurus dengan panjang.Berbanding lurus adanya perubahan suhu.Pemuaian tergantung jenis zatnya.

B. Pemuaian pada luas

Hal ini terjadi karena suhu mengalami perubahan pada luas benda kenaikan suhu ketika dipanaskan. Misalnya kaca pada jendela. Hal ini bisa terjadi karena mengalami pemuaian, suhu kaca tersebut mengalami kenaikan dan terjadi lebarnya pada kaca tersebut.

C. Pemuaian pada volume

Hal ini terjadi karena mengalami kenaikan suhu pada volume ketika dipanaskan sehingga ukuran benda tersebut akan menjadi luas.Misalnya air yang dimasak di dalam panci, hal ini karena pemuaian tehadap bertambahnya volume ketika dipanaskan.

Terjadinya perubahan pada zat volume:

Berbanding lurus pada volume.Berbanding lurus pada perubahan suhu (tergantung jenisnya).

Kesimpulan

Mencegah terjadinya Pemuaian pada siang hari dan mencegah terjadinya penyusutan pada malam hari.

Catatan

Pemuaian pada zat padat hal ini terjadi karena adanya kenaikan suhu yang dipanaskan (karena adanya kalor).

24. rel kereta api dipasang tidak rapat karena


Rel kereta api dipasang tidak rapat karena sambungan rel kereta api akan memuai di siang hari akibat panas terik matahari yang kemudian dapat menyebabkan sambungan rel bengkok. Simak penjelasan berikut.

Pembahasan

Pemuaian adalah bertambahnya ukuran akibat peningkatan suhu suatu benda. Peningkatan suhu terjadi karena adanya energi panas di sekitar benda. Berikut beberapa contoh pemuaian dalam kehidupan sehari-hari.

>> Pemuaian kaca jendela pada siang hari

Kaca jendela dipasang renggang agar kaca tidak pecah ketika memuai di siang hari karena terik matahari.

>> Pemuaian sambungan rel kereta api pada siang hari

Sambungan rel kereta api dipasang renggang (tidak rapat) agar rel tidak melengkung (bengkok) ketika memuai di siang hari karena terik matahari.

>> Pemuaian kabel listrik di jalan pada siang hari

Kabel listrik akan memanjang di siang hari karena pemuaian sehingga menyebabkan kabel makin molor. Pada praktiknya, kabel dipasang kendor agar tidak putus ketika menyusut di malam hari saat suhunya menurun.

>> Pemuaian gas dalam ban sepeda pada siang hari

Gas dalam ban sepeda akan memuai di siang hari karena udara di sekitar lebih panas daripada malam hari. Artinya, volume gas akan bertambah pada siang hari. Oleh karena itu, ban biasanya diisi tidak penuh agar ban tidak mudah pecah (meledak) saat terjadi pemuaian.

>> Pemuaian termometer raksa saat ditempelkan pada tubuh kita

Saat termometer klinis ditempelkan di bagian tertentu pada tubuh kita, raksa dalam termometer akan memuai dan termometer akan menunjukkan angka atau nilai yang menunjukkan suhu tubuh kita.

Dari uraian di atas, rel kereta api dipasang tidak rapat (renggang) karena sambungan rel kereta api akan memuai di siang hari akibat panas terik matahari yang kemudian dapat menyebabkan sambungan rel bengkok. Hal ini tentunya dapat mengganggu bahkan membahayakan jalannya kereta api. Oleh karena itu, sambungan rel kereta api dipasang tidak rapat agar sambungan rel tersebut tidak membengkok di siang hari.
