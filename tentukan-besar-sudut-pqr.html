2. tentukan besar sudut PQR !​


Besar sudut PQR adalah 56°

Langkah langkah

∠PRQ = 180 - 110

∠PRQ = 70°

∠PRQ + ∠PQR + ∠QPR = 180°

(70) + (x + 2) + (x) = 180

70 + x + 2 + x = 180

70 + 2 + x + x = 180

72 + 2x = 180

2x = 180 - 72

2x = 108

x = 108/2

x = 54

∠PQR = (x + 2)°

∠PQR = (54 + 2)°

∠PQR = 56°

6. tentukan besar sudut m<pqr​


Jawaban:

pertama Cari sudut ∆QPR

karena tegak lurus maka jumlah sudut SPR + QPR=180°

110+x=180

x=70

sudut ∆QPR= 70°

Jumlah sudut segitiga adalah 180°

180= 44+70+x

180= 114 + x

x= 180-114

x= 66

jadi sudut ∆PQR adalah 66°

Jawab:

Diketahui:

SPR = 110°

PRQ = 44°

Ditanya: Sudut PQR ?

Pembahasan

Karena SPQ = 180°

Maka, QPR = 180° - 110° = 70°

Karena sudut itu berhimpit di dalam segitiga (180°), maka:

180° = 70° + 44° + PQR°

180 - 70 - 44 = PQR°

66° = PQR

#Shiba's Solver
