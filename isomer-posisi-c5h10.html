JAWABAN:

 

Isomer C5H10 (pentena)

a.       1-pentena

b.      2-pentena

c.       3-pentena

d.      2 metil-1-butena

e.       3 metil-1-butena

f.       2 metil-2-butena

g.       3 metil-2-butena

 

PEMBAHASAN LEBIH LANJUT :

 

C5H10 termasuk alkena. Alkena memiliki rumus umum CnH2n

Langkah-langkah membuat isomer pentena

a.       buatlah ikatan rantai C lurus sejumlah 5 atom C dengan atom C pertama berikatan rangkap dua. Kemudian lengkapi dengan menambahkan atom H sehingga atom C berikatan dengan 4 atom yang lain. Senyawa ini disebut 1-pentena

b.      buatlah ikatan rantai C lurus sejumlah 5 atom C dengan atom C kedua dari ujung berikatan rangkap dua. Kemudian lengkapi dengan menambahkan atom H sehingga atom C berikatan dengan 4 atom yang lain. Senyawa ini disebut 2-pentena

c.       buatlah ikatan rantai C lurus sejumlah 5 atom C dengan atom C ketiga dari ujung berikatan rangkap dua. Kemudian lengkapi dengan menambahkan atom H sehingga atom C berikatan dengan 4 atom yang lain. Senyawa ini disebut 3-pentena

d.      ambil satu atom C paling ujung pada 1-pentena yang bukan ikatan rangkap dua, kemudian letakkan sebagai cabang pada atom C kedua. Lengkapi dengan menambahkan atom H sehingga atom C berikatan dengan 4 atom yang lain . Senyawa ini disebut 2 metil-1-butena

e.       ambil satu atom C paling ujung pada 1-pentena yang bukan ikatan rangkap dua, kemudian letakkan sebagai cabang pada atom C ketiga. Lengkapi dengan menambahkan atom H sehingga atom C berikatan dengan 4 atom yang lain . Senyawa ini disebut 3 metil-1-butena

f.       ambil satu atom C paling ujung pada 2-pentena, kemudian letakkan sebagai cabang pada atom C kedua.  Lengkapi dengan menambahkan atom H sehingga atom C berikatan dengan 4 atom yang lain. Senyawa ini disebut 2 metil1-2-butena

g.       ambil satu atom C paling ujung pada 2-pentena, kemudian letakkan sebagai cabang pada atom C ketiga. Lengkapi dengan menambahkan atom H sehingga atom C berikatan dengan 4 atom yang lain . Senyawa ini disebut 3 metil1-2-butena
