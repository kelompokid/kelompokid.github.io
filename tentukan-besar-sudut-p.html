Diketahui :

sudut Q = 180° - 112° = 68°

Ditanya :

sudut P = ?

Cara :

Karena ΔPQR adalah segitiga sama kaki, maka sudut P dan sudut R sama besar

180° = sudut Q + sudut R + sudut P

180° = 68° + x + x

180° = 68° + 2x

180° - 68° = 2x

112° = 2x

112° / 2 = x

56° = x

Jadi, sudut P = 56°
