5. Tentukan nilai x dan besar ∠NOK!


Jawaban:

Besar sudut berpelurus 180⁰

Besar sudut berpenyiku 90⁰

Pada gambar merupakan sudut berpelurus yang besar sudutnya 180⁰.

Maka;

a). Menentukan nilai x

<NOK + <MON + <LOM = 180⁰

2x⁰ + 3x⁰ + 85⁰ = 180⁰

5x⁰ = 180⁰-85⁰

5x⁰ = 95⁰

x = 95⁰/5⁰

x = 19

b). Menentukan besar sudut <NOK

nilai x = 19 , substitusi ke;

<NOK = 2x⁰ = 2(19)⁰ = 38⁰

Jawab :

∠ NOK = 2x°

∠ NOM = 3x°

∠ MOL = 85°

∠ Lurus = 180°

Menentukan nilai x :

⇔ 2x + 3x + 85 = 180°

⇔ 85 + 2x + 3x = 180°

⇔ 85 + 5x = 180°

⇔ ( 85 + ( -85 )) + 5x = ( 180 + ( -85 ))

⇔ 0 + 5x = 95°

⇔ x = 95 : 5

⇔ x = 19°

Besar ∠ NOK :

∠ NOK = 2x

            = 2 ( 19 )

            = 2 × 19

            = 38°

=========================================================

Tidak Yakin ??Ini Pembuktiannya ...

Besar ∠ KOL = 180°

Besar ∠ MOL = 85°

Besar ∠ NOK = 38°

Besar ∠ NOM = 3x

                       = 3 × 19

                       = 57°

Jumlah total dari ∠ KOL, MOL, NOK = 85 + 38 + 57 = 180°

Jadi, Jawabannya betul :D
