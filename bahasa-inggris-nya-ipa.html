18. Marlon mempunyai nilai rata-rata 75 untuk pelajaran Bahasa Inggris,Matematika dan IPA. Nilai IPA-nya lebih kecil 15 dari nilai bahasaInggrisnya. Nilai Matematikanya lebih besar 12 dibanding nilai bahasaInggris. Berapa nilai matematikanya ?Tolong dibantu teman teman​


Diketahui:

Misal:

IPA = a

Bahasa Inggris = b

Matematika = c

Rata" = 75

a+15 = b

c = b+12

Ditanyakan:

c

Jawab:

*) a+15 = b

a = b-15

*) c= b+12

*)Rata" = (a+b+c)/3

75 = ((b-15)+b+(b+12))/3

225 = 3b-3

225+3 = 3b

228 = 3b

76= b

c = b+12

= 76+12

= 88

Maka, nilai matematikanya 88.

Ipa : Bahasa Inggris
210 : 115  : 5
  42 : 23
Perbandingan 6 buku ipa dan 6 buku bahasa inggris itu tetap, yaitu 42 : 23.                       1 ipa : 1 b.ing = 210 : 115
bentuk sedrhana                   = 42 : 23
                       6 ipa : 6. ing = 6 x 210 : 115 x 6
bentuk sederhana                   = 210 : 115
                   = 42 : 23
