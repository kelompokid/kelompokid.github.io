6. dari suatu deret aritmatika diketahui U3+U6+U9+U12=72. Jumlah 14 suku pertama deret ini adalah?


Berdasarkan pernyataan U3 + U6 + U9 + U12 = 72, jumlah 14 suku pertama dari deret tersebut adalah 252.

Penjelasan dengan langkah-langkah:

Diketahui :

U3 + U6 + U9 + U12 = 72

Ditanya :

Jumlah 14 suku pertama

Jawab :

Untuk menentukan jumlah 14 suku pertama maka kita harus mengetahui terlebih dahulu rumus suku ke n dari soal tersebut dan rumus jumlah deret. Dimana seperti yang kita ketahui, unruk rumus suku ke n dapat dituliskan menjadi berikut

Un = a + (n-1) b

Sedangkan, untuk rumus jumlah suku ke n sendiri kita bisa menggunakan rumus berikut

Sn = [tex]\frac{1}{2}[/tex] n ( 2a + (n-1) b )

Dimana a adalah suku pertama atau bisa dikenal dengan U1, b adalah beda ( U2 - U1), dan n menyatakan suku keberapa dalam barisan.

Pada soal diketahui bahwa U3 + U6 + U9 + U12 = 72 sehingga dengan menggunakan rumus suku ke n didapatkan persamaan

U3 + U6 + U9 + U12 = 72

(a + 2b) + ( a + 5b) + (a + 8b) + (a +11b) = 72

4a + 26b = 72

2a + 13b = 36

Persamaan tersebut yang akan kita gunakan untuk mencari jumlah 14 suku pertama dengan menggunakan rumus jumlah suku ke n

Sn = [tex]\frac{1}{2}[/tex] n ( 2a + (n-1) b )

S14 = 0,5 (14) (2a + (14-1) b)

S14 = 7 (2a + 13b)

Karena kita sudah mendaptkan hasil 2a + 13b = 36, maka kita bisa langsung memasukkannya kedalam persamaan atau mensubstitusi hasil dari 2a + 13b = 36 kedalam persamaan S14 = 7 (2a + 13b).

S14 = 7 (2a + 13b)

S14 = 7 ( 36 )

S14 = 252

Maka, jumlah 14 suku pertama dari deret tersebut adalah 252.

[tex]U_6+U_9+ U_{12}+U_{15} = 20\\S_{20}=\,???[/tex]


Jawaban :


Suku Aritmatika :

[tex]Un=a+b(n-1),\,a=U_1\\U_n=U_m+b(n-m)[/tex]


Kita langsung kerjakan saja. Pengerjaan seperti ini :

[tex]U_6+U_9+ U_{12}+U_{15} = 20\\a+5b+a+8b+a+11b+a+14b=20\\4a+38b=20\\------------------------\,:2\\2a+19b=10[/tex]


Deret aritmatika :

[tex]S_{n}=\frac{1}{2}(a+U_n)\\S_n=\frac{1}{2}[2a+b(n-1)]\\\\a=U_1[/tex]


Kita langsung cari [tex]S_{20}[/tex]. Caranya seperti ini :

[tex]S_n=\frac{1}{2}[2a+b(n-1)]\\S_{20}=\frac{1}{2}[2a+b(20-1)]\\S_{20}=\frac{1}{2}(2a+19b)[/tex]


Nah, dari sini... Jika [tex]2a+19b=10[/tex] dari yang kita dapatkan sebelumnya, kita masukkan ke dalam hitungan yang sedang kita kerjakan ini. Caranya seperti ini :

[tex]S_{20}=\frac{1}{2}(2a+19b)\\S_{20}=\frac{1}{2}(10)\\S_{20}=5[/tex]


Jadi, jawaban pertanyaan itu :

[tex]S_{20}=5[/tex]

U6 + U9 + U12 + U15 = 20
Karena deret aritmatika, maka :
a + 5b + a + 8b + a + 11b + a + 14b = 20

2a + 19b = 10

S20 = 10(2a + 19b)

30. Dari deret aritmatika diketahui U6 + U9 + U12 + U15 = 20. Jumlah suku 20 pertama ?


untuk barisan aritmatika, berlaku
[tex]U_p+U_q=U_r+U_s[/tex]
jika [tex]p+q=r+s[/tex]

jadi, [tex]U_6+U_{15}=U_9+U_{12}= \frac{20}{2}=10 [/tex]

perhatikan bahwa [tex]U_1+U_{20}=U_6+U_{15}=10[/tex]
maka
[tex]S_{20}= \frac{20}{2}(U_1+U_{20})[/tex]
[tex]S_{20}=10(10)=100[/tex]
