16. ✨☁️Quiz Matematika☁️✨v.balok=pxlxt=1,5x0,2x0,6?-Copas Ngasal?Respost~​


Volume balok tersebut adalah 0,18 cm³

Pembahasan

Rumus yang digunakan

V = p x l x t

keterangan :

V = volumep = panjangl = lebart = tinggi

»Diketahui :

p = 1,5 cm

l = 0,2 cm

t = 0,6 cm

»Ditanya :

Volume balok?

»Dijawab :

Volume Balok

= p x l x t

= 1,5 cm x 0,2 cm x 0,6 cm

= 0,3 cm² x 0,6 cm

= 0,18 cm³ ✔

============================

Detail Jawaban

Mapel : Matematika

Kelas : VIII SMP

Materi : Bab 8 - Bangun Ruang

Kode kategorisasi : 8.2.8

Kata kunci : balok, volume, panjang, lebar, tinggi

Jawaban :

Jika satuan panjang nya cm ( centimeter ) :

0,18 cm³

Jika satuan panjang nya m ( meter ) :

0,18 m³

Langkah - langkah penyelesaian :

Soal :

Mencari volume balok ?

Jawab :

Diketahui :

Panjang balok ( p ) = 1,5 cm

Lebar balok ( l ) = 0,2 cm

Tinggi balok ( t ) = 0,6 cm

Ditanya :

Volume balok ( v ) = ... ?

Jawab :

Volume ( v ) = Panjang ( p ) × lebar ( l ) × tinggi ( t )

Volume ( v ) = 1,5 cm × 0,2 cm × 0,6 cm

Volume ( v ) = 0,3 cm × 0,6 cm

Volume ( v ) = 0,18 cm³

Penyelesaian :

0,18 cm³

Nb : Jangan lupa tulis pangkat 3 nya ya ( ...³ ) walaupun jawabannya benar tapi tidak ditulis pangkat 3 nya khawatir disalahkan.

Jika satuannya m ( meter ) , ubahlah semua cm menjadi m. Dan volumenya di ujungnya terdapat m³
