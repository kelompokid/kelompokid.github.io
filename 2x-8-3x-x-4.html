28. 2x + |8 - 3x| = |x- 4|


Jawab:

Tentukan nilai x dari   2x + |8 - 3x| = |x- 4|? tidak ada penyelesaian x bilangan real atau hp{x=∅}

Pembahasan:

Nilai mutlak dinotasikan sebagai "|  |" yang didefenisikan sebagai jarak antara sebuah bilangan dengan nol pada sebuah garis bilangan

atau dapat juga dikatakan nilai mutlak adalah nilai yang tidak negatif

Definisi nilai mutlak :

|x| = x jika x ≥ 0

|x| = -x jika x < 0

misal :

1. |2|=2 tetap karena 2>0

 |-2|=-(-2)=2  karena -2<0 maka ditambahkan tanda negatif didepan -2

2. |x|=8

disini nilai x bisa 8 atau -8  sebab |8|=8  dan |-8|=-(-8)=8

3. |x|=-1

tidak ada nilai x yang memenuhi karena defenisi dari nilai mutlak adalah bilangan yang tidak negatif oleh sebab itu berapapun angka x tak mungkin hasilnya -1

Tentukan penyelesaian dari 2x+|8-3x|=|x-4|

Penyelesaian:

kita defenisikan dulu nilai mutlak dari |8-3x| dan |x-4|

|8-3x|=8-3x ,jika  8-3x≥0 ,-3x≥-8 ,x≤8/3

         -(8-3x) jika  x>8/3

|x-4|=x-4 ,jika  x-4≥0 ,x≥4

      -(x-4) jika  x<4

langkah selanjutnya kita buta garis bilangan pembuat nol dari nilai mutlaknya sebagai berikut

________8/3__________4_____

untuk x≤8/3 maka

|8-3x|=8-3x

|x-4|=-(x-4)=-x+4

maka

2x + |8 - 3x| = |x- 4|

2x+8-3x=-x+4

-x+8=-x+4

8=4 (selalu salah)

jadi untuk interval ini x tidak memiliki penyelesaian

untuk 8/3<x<4 maka

|8-3x|=-(8-3x)=-8+3x

|x-4|=-(x-4)=-x+4

jadi

2x + |8 - 3x| = |x- 4|

2x-8+3x=-x+4

5x-8=-x+4

5x+x=4+8

6x=12

x=12/6

x=2

karena x=2 tidak  berada pada interval 8/3<x<4 maka x=2 adalah bukan penyelesaian

Untuk x≥4 maka

|8-3x|=-8+3x

|x-4|=x-4

maka

2x + |8 - 3x| = |x- 4|

2x-8+3x=x-4

5x-8=x-4

5x-x=8-4

4x=4

x=1

karena x=1 tidak berada pada interval x≥4 maka x=1 bukan penyelesaian

jadi himpunan penyelesaian dari 2x + |8 - 3x| = |x- 4|  adalah hp;{x=∅}
