3. analisis reaksi berikut apakah termasuk reaksi reduksi atau reaksi oksidasi​


a) MnO2 => MnO4^-

Biloks Mn pada (MnO2) = +4Biloks Mn pada (MnO4^-) = +7=> Reaksi oksidasi

b) Cr2O7^2- => Cr^3+

Biloks Cr pada (Cr2O7^2-) = +6Biloks Cr pada (Cr^3+) = +3=> Reaksi reduksi

c) NO3^- => NO2

Biloks N pada (NO3^-) = +5Biloks N pada (NO2) = +4=> Reaksi reduksi

d) SbO3^3- => SbO4^3-

Biloks Sb pada (SbO3^3-) = +3Biloks Sb pada (SbO4^3-) = +5=> Reaksi oksidasi

e) SO2 => SO4^2-

Biloks S pada (SO2) = +4Biloks S pada (SO4^2-) = +6=> Reaksi oksidasi

f) C2O4^2- => CO2

Biloks C pada (C2O4^2-) = +3Biloks C pada (CO2) = +4=> Reaksi oksidasi

5. tenyukan reaksi reduksi, reaksi oksidasi, oksidator, reduktor, hasil reduksi dan hasil oksidasi dari reaksi reaksi berikut


reaksi reduksi, reaksi oksidasi, oksidator, reduktor, hasil reduksi dan hasil oksidasi dari reaksi reaksi berikut :

1. CrO₂⁻ + ClO⁻  + OH⁻ → Cl⁻ + CrO₄²⁻ + H₂O

Oksidator = mengalami reduksi = biloksnya turun = ClO⁻

Reduktor = mengalami oksidasi = biloksnya naik = CrO₂⁻

Hasil Oksidasi = CrO₄²⁻

Hasil Reduksi = Cl⁻

2. MnO₄⁻ + C₂O₄²⁻ + H⁺ → Mn²⁺ + CO₂ + H₂O

Oksidator = mengalami reduksi = biloksnya turun = MnO₄⁻

Reduktor = mengalami oksidasi = biloksnya naik = C₂O₄²⁻

Hasil Oksidasi = CO₂

Hasil Reduksi = Mn²⁺

3. 2CO + 2NO → 2CO₂ + N₂

Oksidator = mengalami reduksi = biloksnya turun = NO

Reduktor = mengalami oksidasi = biloksnya naik = CO

Hasil Oksidasi = CO₂

Hasil Reduksi = N₂

4. H₂SO₄ + 8HI → H₂S + 4I₂ + 4 H₂O

Oksidator = mengalami reduksi = biloksnya turun = H₂SO₄

Reduktor = mengalami oksidasi = biloksnya naik = HI

Hasil Oksidasi = I₂

Hasil Reduksi = H₂S

5. Sn + 4HNO₃ → SnO₂ + 4NO₂ + 2H₂O

Oksidator = mengalami reduksi = biloksnya turun = HNO₃

Reduktor = mengalami oksidasi = biloksnya naik = Sn

Hasil Oksidasi = SnO₂

Hasil Reduksi = NO₂

9. Mengapa reaksi oksidasi selalu diikuti reaksi reduksi?


Reaksi oksidasi adalah reaksi pengikatan oksigen. Sumber oksigen pada reaksi oksidasi disebut oksidator.Reaksi reduksi adalah reaksi pelepasan oksigen. Zat yang menarik oksigen pada reaksi reduksi disebut reduktor

Reaksi oksidasi adalah reaksi pelepasan electron.

Reaksi reduksi adalah reaksi penerimaan electron.

Reaksi pelepasan dan penerimaan electron terjadi secara simultan, artinya jika suatu unsur melepas electron berarti ada unsur lain yang menangkap electron itu.

Jadi dapat disimpulkan bahwa setiap oksidasi selalu diikuti reduksi, karena itu setiap reaksi yang melibatkan reaksi oksidasi dan reduksi disebut reaksi redoks.

Khusus untuk oksigen saja

Konsep

=> Oksidasi itu mengikat (berarti yang awalnya terpisah menjadi bergabung)

=> Reduksi itu melepas (berarti yang awalnya bergabung menjadi terpisah)

=> Oksidasi berarti jumlah biloksnya bertambah

=> Reduksi berarti jumlah biloksnya berkurang

5) FeCl3 -> FeCl2  ->  (Biloks)
