2. deret angka 1 8 81 1024 ...


Deret angka 1 8 81 1024 ..., angka selanjutnya adalah 15.625. Hasil tersebut diperoleh dengan mencari pola bilangan tiap suku. Pola bilangan adalah suatu aturan dalam barisan bilangan sehingga dapat menentukan bilangan berikutnya dari barisan bilangan tersebut

Pembahasan

Bilangan pertama (U₁) = 1 = 1² Bilangan kedua (U₂) = 8 = 2³ Bilangan ketiga (U₃) = 81 = 3⁴ Bilangan keempat (U₄) = 1.024 = 4⁵

Sehingga bilangan berikutnya yaitu bilangan kelima adalah:

U₅ = 5⁶

U₅ = 15.625

Jadi rumus suku ke n nya adalah

Un = [tex]n^{n + 1}[/tex]

30. Tentukan pola Sn 1+2+4+8+. . .+1024=. . .


Penyelesaian:

barisan geometri:
barisan bilangan: 1 + 2 + 4 + 8 + .. + 1024 = ...?
a = 1
r = 2

pertama cari nilai 1024 itu n ke berapa, dengan cara mencari nilai n:
Un = ar^n-1
1024 = 1 . 2^n-1
1024 = 2^n . 1/2
2048 = 2^n
2^11 = 2^n 
n = 11

kedua masukkan nilai yang diketahui ke persamaan untuk mencari Sn, didapat:
Sn = a(1-r^n) / (1 - r)
= 1(1 - 2^11) / 1 - 2
= -(1 - 2048)
= - (-2047)
Sn = 2047
