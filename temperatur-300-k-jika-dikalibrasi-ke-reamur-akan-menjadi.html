5. konversi 300 K dalam celcius reamur dan faahrenheit


A) Diket :

Suhu Kelvin (K) = 300

B) Ditan :

K ke C = ?

K ke R = ?

K ke F = ?

C) Jawab :

1) Kelvin (K) ke Celcius (C) :

-> C = K - 273

-> C = 300 - 273

-> C = 27

2) Kelvin (K) ke Reamur (R)

-> R = 4/5 x C

-> R = 4/5 x 27

-> R = 21,6

3) Kelvin (K) ke Fahrenhait (F)

-> F = 9/5 x (C + 32)

-> F = 9/5 x (27 + 32)

-> F = 9/5 x 59

-> F = 106,2
