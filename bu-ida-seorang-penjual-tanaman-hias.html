15. Karena ingin membeli yang baru, Bu Ida menjual perhiasannya dengan harga Rp3.600.000,00. Dari penjualan sepeda itu, Bu Ida memperoleh keuntungan 20%. Harga pembelian perhiasan tersebut adalah ….​


Jawab:

Rp 2.880.000,00

Penjelasan dengan langkah-langkah:

Keuntungan biasanya selalu lebih daripada harga biasa jadi rumus untuk harga pembelian sebelum menjadi keuntungan sepertinyaini

A = keuntungan

B = harga beli

C = harga jual

Rumusnya

A = C - B

Sekarang ke topik

B = C - [ C.A ]

B = 3.600.000 - [ 3.600.000 x 20% ]

B = 3.600.000 - 720.000

B = Rp 2.880.000,00

Semoga bisa membantu
