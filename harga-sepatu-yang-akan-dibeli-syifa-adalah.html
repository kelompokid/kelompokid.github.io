Harga sepatu Citra = 2 x Harga sepatu Syifa

Uang Citra mula-mula = Rp 40.000
Tiap hari Citra menabung = Rp 3.500
Uang Syifa mula-mula = Rp 10.000
Tiap hari Syifa menabung = Rp 2.000

Harga sepatu Syifa =.......?

misal x = jumlah hari menabung

Harga sepatu Citra = 2 (Harga sepatu Syifa)
40.000 + x (3.500) = 2 (10.000 + x (2.000))
40.000 + 3.500x = 20.000 + 4.000x
40.000 - 20.000 = 4.000x - 3.500x
20.000 = 500x
x = 20.000 : 500
x = 40

Jadi, lama menabung adalah 40 hari

= 10.000 + (2.000 x 40)
= 10.000 + 80.000
= 90.000
