5. Bagaimana cara membuat Na2S2O3 0,2 M sebanyak 50 ml dari Na2S2O3 2 M,buat dengan perhitungan!


Salam sejahtera. Cara membuat [tex]\sf Na_2S_2O_3 [/tex] 0,2 M sebanyak 50 mL dari [tex]\sf Na_2S_2O_3 [/tex] adalah dengan mencampurkan 5 mL [tex]\sf Na_2S_2O_3 [/tex] dengan 45 mL air. Perhitungannya sebagai berikut.

[tex]~[/tex]

Penjelasan dengan langkah-langkah:

Melakukan pengenceran pada campuran 1 dengan campuran 2 dapat menggunakan rumus berikut:

[tex]\boxed{\sf M_1 \times V_1 = M_2 \times V_2}[/tex]

Keterangan:

M1 = Konsentrasi larutan 1.

V1 = Volume larutan 1.

M2 = Konsentrasi larutan 2.

V2 = Volume larutan 2.

[tex]~[/tex]

Penyelesaian Soal

Diketahui:

M1 = 0,2 M.V1 = 50 mL.M2 = 2 M.

[tex]~[/tex]

Ditanyakan:

V2 = ..?

[tex]~[/tex]

Jawab:

[tex]\sf M_1 \times V_1 = M_2 \times V_2[/tex]

[tex]\sf 0,2~M \times 50~mL = 2~M \times V_2[/tex]

[tex]\sf 10~M ~mL = 2~M \times V_2[/tex]

[tex]\sf V_2 = \frac{10~M~mL}{2~M} [/tex]

[tex]\sf V_2 = 5~mL[/tex]

[tex]~[/tex]

Kesimpulan:

Jadi, cara membuat [tex]\sf Na_2S_2O_3 [/tex] 0,2 M sebanyak 50 mL dari [tex]\sf Na_2S_2O_3 [/tex] adalah dengan mencampurkan 5 mL [tex]\sf Na_2S_2O_3 [/tex] dengan 45 mL air.

[tex]~[/tex]

7. Tentukan biloks belerang dalam senyawa berikut:S8,SO2,SO2CL2,Na2S2O3,Na2S4O6

S8 ⇒ S=0

SO2

S + 2O = 0

S + 2(-2) = 0

S =+4

SO2Cl2

S + 2O + 2Cl = 0

S + 2(-2) + 2(-1) = 0

S=+6

Na2S2O3

2Na + 2S + 3O = 0

2(+1) + 2S + 3(-2) = 0

S =+2

Na2S4O6

2Na + 4S + 6O = 0

2(+1) + 4S + 6(-2) = 0

S =+2.5

Pembahasan

Reaksi redoks merupakan reaksi yang saling berkesinambungan. Dimana terjadi oksidasi maka akan terjadi oksidasi pula. Reaksi oksidasi merupakan reaski pelepasan elektron, molekul atau ion. reaksi ini juga ditandai dengan peningkatan bilangan oksisadi. Sedangkan reaksi reduksi merupakan reaksi penangkapan elektron, molekul atau ion. dengan kata lain, dalam reaksi reduksi terjadi pengurangan bilangan oksidasi.

zat yang bersifat reduktor akan mengalami reaksi oksidasi. Sedangkan zat yang berperan sebagai oksidator akan mengalami reduksi. Logam-logam golongan alkali dan alkali tanah merupakan contoh kelompok yang berperan sebagai reduktor. logam-logam golongan ini merupakan pembentuk basa kuat. Oksidasi :  Na ⇒ Na+  +  e

Reduksi : H+  +  e  ⇒  1/2 H2

12. K2CrO7, berapa bilangan biloks Cr? CaSO4, berapa biloks S? NaNO3 , berapa biloks N ? NH4+ Berapa biloks N ? SO2, berapa biloks S?


1)  Bilangan biloks Cr dalam K₂CrO₇ adalah +12.

2) Bilangan biloks S dalam CaSO₄ adalah +6.

3) Bilangan biloks N dalam NaNO₃ adalah +5.

4) Bilangan biloks N dalam NH₄⁺ adalah -3.

5) Bilangan biloks S dalam SO₂ adalah +4.

Pembahasan : 1)  Bilangan biloks Cr dalam K₂CrO₇

biloks K = +1 (Golongan IA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks K₂CrO₇ = 0  

(2 × biloks K) + biloks Cr + (7 × biloks O) = 0

(2 × (+1)) + biloks Cr + (7 × (-2)) = 0

(+2) + biloks Cr + (-14) = 0

biloks Cr + (-12) = 0

biloks Cr = +12

2) Bilangan biloks S dalam CaSO₄

biloks Ca = +2 (Golongan IIA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks CaSO₄ = 0  

biloks Ca + biloks S + (4 × biloks O) = 0

(+2) + biloks S + (4 × (-2)) = 0

(+2) + biloks S + (-8) = 0

biloks S + (-6) = 0

biloks S = +6

3) Bilangan biloks N dalam NaNO₃

biloks Na = +1 (Golongan IA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks NaNO₃ = 0  

biloks Na + biloks N + (3 × biloks O) = 0

(+1) + biloks N + (3 × (-2)) = 0

(+1) + biloks N + (-6) = 0

biloks N + (-5) = 0

biloks N = +5

4) Bilangan biloks N dalam NH₄⁺

biloks H = +1

aturan biloks :  

Bilangan oksidasi hidrogen dalam senyawanya adalah +1 kecuali dalam Hidrida, atom Hidrogen biloksnya adalah -1. Contoh : Bilangan oksidasi H dalam H₂O, NH₃, dan HCl = +1, kecuali Bilangan oksidasi H dalam NaH dan CaH₂ = -1.

biloks NH₄⁺ = +1

biloks N + (4 × biloks H)= +1

biloks N + (4 × (+1)) = +1

biloks N + (+4) = +1

biloks N + (+4) = +1

biloks N = (+1) + (-4)

biloks N = -3

5) Bilangan biloks S dalam SO₂

Biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks SO₂ = 0  

biloks S + (2 × biloks O) = 0

biloks S + (2 × (-2)) = 0

biloks S + (-4) = 0

biloks S = +4

13. Jika biloks H = 1 maka O = -2 tentukan atom atom selain H dan O. A) CH3OH B)PO3³- C)Na2S2O3 Mohon bantuannya :)


Untuk menentukan biloks masing-masing atom, perlu diingat peraturan biloks berikut ini!

Aturan BILOKS (BO)Unsur/ senyawa bebas, BO = 0H, BO = +1 (kecuali senyawa hidrida (LiH,CaH), BO = -1)O, BO = -2 (kecuali senyawa peroksida (H₂O₂, Na₂O₂), BO = -1)F, BO = -1Gol (I, II, III)A, BO = +1, +2, +3Unsur/senyawa ion, BO = muatan ionPembahasan

Tentukan atom-atom selain H dan O!

A)  CH₃OH (senyawa bebas)

C + 4H + O = 0

C + 4(+1) + (-2) = 0

C + 2 = 0

C = -2

Jadi, biloks C = -2.

B)  PO₃³⁻ (senyawa ion)

P + 3O = -3

P + 3(-2) = -3

P + (-6) = -3

P = -3 + 6

P = +3

Jadi, biloks P = +3.

C)  Na₂S₂O₃ (senyawa bebas)

2Na + 2S + 3O = 0

2(+1) + 2S + 3(-2) = 0

+2 + 2S + (-6) = 0

(-4) + 2S = 0

2S = +4

S = +2

Jadi, biloks Na = +1, dan biloks S = +2.

Dari reaksi berikut :  2 Na₂S₂O₃ + I₂ → 2 Nal + Na₂S₄O₆

Pernyataan yang benar adalah

a. sebagai oksidator adalah Na₂S₂O₃

b. sebagai reduktor adalah I₂

c. S mengalami perubahan biloks dari +4  menjadi +6

d. I mengalami perubahan biloks dari 0  menjadi -1

e. Na₂S₂O₃ tereduksi menjadi Na₂S₄O₆

2 Na₂S₂O₃ + I₂ → 2 Nal + Na₂S₄O₆

Tahap 1 :

Tentukan terlebih dahulu biloks  I₂ :

biloks  I₂ = 0

aturan biloks :  

Bilangan oksidasi unsur bebas adalah nol. Contoh : Bilangan oksidasi atom-atom pada C, Ne, H₂, O₂, Cl₂ , P₄, S₈, Cu, Fe, Na adalah nol.

Tahap 2 :

Selanjutnya tentukan biloks I dalam Nal :

Biloks I dalam Nal

biloks Na = +1          (Golongan IA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks NaI = 0  

biloks Na + biloks I = 0  

(+1) + biloks I = 0  

biloks I = -1
