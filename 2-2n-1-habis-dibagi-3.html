5. Buktikan bahwa 2 pangkat 2n-1 habis dibagi 3


Diketahui [tex]2^{2n}-1[/tex]. Berdasarkan pembuktian dengan induksi matematika, rumus tersebut akan habis dibagi 3 untuk [tex]n\geq 1[/tex].

Penjelasan dengan langkah-langkah:

Diketahui:

[tex]2^{2n}-1[/tex]

Ditanya:

Buktikan dengan induksi matematika bahwa rumus tersebut habis dibagi 3!

Pembahasan:

1. Langkah dasar

Untuk n = 1

[tex]2^{2n}-1=2^{2.1}-1=2^{2}-1=4-1=3[/tex] (habis dibagi 3)

Terbukti benar untuk n = 1

2. Langkah induksi

Asumsikan untuk n = k rumus tersebut benar

[tex]2^{2k}-1=3b[/tex] (b adalah sembarang bilangan bulat)

Akan dibuktikan bahwa untuk n = k + 1 rumus juga benar

[tex]2^{2(k+1)}-1=3b[/tex]

Pembuktian:

Ruas kiri

[tex]2^{2(k+1)}-1=2^{2k+2}-1\\\implies 2^{2k}.2^2-1\\\implies 2^{2k}.4-1\\\implies 4.2^{2k}-1\\\implies 4.2^{2k}-4+3\\\implies 4(2^{2k}-1)+3\\\implies 4(3b)+3\\\implies 3(4b)+3\\\implies 3(4b+1)[/tex](habis dibagi 3)

Terbukti benar untuk n = k + 1

Sehingga, [tex]2^{2n}-1[/tex] habis dibagi 3 untuk [tex]n\geq 1[/tex].

7. 3^2n-1 habis dibagi 8


3^2n – 1 habis dibagi 8, pernyataan tersebut dapat dibuktikan dengan menggunakan induksi matematika. Ada dua langkah dalam induksi matematika yaitu:

Buktikan bahwa untuk n = 1 benar Dengan mengasumsikan bahwa untuk n = k benar, maka buktikan bahwa untuk n = k + 1 juga benar

Pembahasan

[tex]3^{2n} - 1[/tex] habis dibagi 8

1. Akan dibuktikan untuk n = 1 benar

[tex]3^{2n} - 1[/tex]

= [tex]3^{2(1)} - 1[/tex]

= 3² – 1  

= 9 – 1

= 8

benar bahwa 8 habis dibagi 8

2. Misal kita asumsikan untuk n = k benar

[tex]3^{2k} - 1[/tex] habis dibagi 8

.

Akan dibuktikan untuk n = k + 1 juga benar

[tex]3^{2(k + 1)} - 1[/tex]

= [tex]3^{2k + 2} - 1[/tex]

= [tex]3^{2k} \: . \: 3^{2} - 1[/tex]

= [tex]3^{2k} \: . \: 9 - 1[/tex]

= [tex]3^{2k} \: . \: (8 + 1) - 1[/tex]

= [tex]8 \: . \: 3^{2k} \: + \: 3^{2k} - 1[/tex]

= [tex]8 \: . \: 3^{2k} \: + \: (3^{2k} - 1)[/tex]

⇒  [tex]8 \: . \: 3^{2k}[/tex] sudah jelas habis dibagi 8

⇒  [tex](3^{2k} - 1)[/tex] juga habis dibagi 8 (berdasarkan n = k)

Jadi [tex]8 \: . \: 3^{2k} \: + \: (3^{2k} - 1)[/tex] juga habis dibagi 8

Sehingga terbukti bahwa [tex]3^{2n} - 1[/tex] habis dibagi 8

12. Buktikan dg induksi bahwa 3^2n + 2^2n+2 habis dibagi 5


akan dibuktikan [tex]3^{2n}+2^{2n+2}[/tex] habis dibagi 5 berapapun nilai n yg positif
pembuktian:
1. akan dibuktikan bahwa jika n=1 maka bilangan yg dihasilkan habis dibagi 5
   masukan n=1 menjadi [tex]3^{2\times1}+2^{2\times1+2}=3^{2}+2^{4}=9+16=25[/tex]
   karena [tex]25=5\times5[/tex] berarti [tex]\frac{25}{5}=5[/tex]
   jadi jika n=1 maka hasilnya adalah 25 dan habis dibagi 5
2. akan dibuktikan bahwa seandainya jika n=k menjadi [tex]3^{2k}+2^{2k+2}[/tex], menghasilkan bilangan yg habis dibagi 5 maka jika n=k+1 juga menghasilkan bilangan yang habis dibagi 5.
masukan n=k+1 menjadi
[tex]3^{2(k+1)}+2^{2(k+1)+2}\\=3^{2k+2}+2^{2k+2+2}\\=3^{2k}\times3^{2}+2^{2k+2}\times2^{2}\\=3^{2k}\times9+2^{2k+2}\times4\\=9\times3^{2k}+4\times2^{2k+2}\\=5\times3^{2k}+4\times3^{2k}+4\times2^{2k+2}\\=5\times3^{2k}+4(3^{2k}+2^{2k+2})[/tex]
karena diandaikan jika n=k maka [tex]3^{2k}+2^{2k+2}[/tex] habis dibagi 5 sehingga kelipatannya [tex]4(3^{2k}+2^{2k+2})[/tex] juga habis dibagi 5.
karena sebarang bilangan yg dikali 5, habis dibagi 5 maka [tex]5\times3^{2k}[/tex] juga habis dibagi 5.
karena dua-duanya habis dibagi 5 maka jika dijumlahkan juga habis dibagi 5, yaitu [tex]5\times3^{2k}+4(3^{2k}+2^{2k+2})[/tex] habis dibagi 5.
Terbukti.

24. Buktikan n² + 2n habis dibagi 3


INDUKSI MATEMATIKA


Akan dibuktikan bahwa n² + 2n habis dibagi oleh 3, untuk setiap n anggota bilangan asli, n = 1, 2, 3 ...


Jawab...

Langkah Dasar:

untuk n = 1,
1² + 2(1) = 3
Benar, 3 habis dibagi oleh 3

Langkah Induksi:

misal untuk n = k berlaku
k² + 2k habis dibagi oleh 3

akan dibuktikan bahwa untuk n = k + 1 juga berlaku
(k+1)² + 2(k+1) habis dibagi oleh 3 juga

yaitu
(k+1)² + 2(k+1)
= k² + 2k + 1 + 2k + 2
= (k² + 2k) + 3 + 2k
= (k² + 2k) + 3(1 + ⅔k)
maka (k+1)² + 2(k+1) habis juga dibagi oleh 3


Semoga jelas dan membantu
