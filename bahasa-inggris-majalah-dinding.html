29. bahasa inggrisnya reza membaca majalah di perpustakaan​


Soal:

Bahasa inggrisnya

reza membaca majalah di perpustakaan​

Jawaban:

Reza reads magazine in the library

Penjelasan:

Karena kalimat bentuk umumu, tidak diberikan keterangan waktu yg spesifik maka gunakan Present Tense

(+) S + VERB 1 (-s/es) + O + complement

= Reza + read+s + magazine + in the library

Structure kalimat:

Subject = Reza

Verb + s = read+s = membaca

Object = Magazine = majalah

Complement = in the library = di perpustakaan

Catatan:

Verb+s karena Subjek tunggal Reza (He)

=== Semoga Membantu ===

Kelas: SMP

Materi: Present Tense

Kode Mapel: 5
