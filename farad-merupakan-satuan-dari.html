25. Tiga kapasitor identik dengan kapasitas 60 mikro Farad jika dirangkai tidak mungkin mempunyai kapasitas kapasitor pengganti sebesar...a. 20 mikro Faradb. 40 mikro Faradc. 80 mikro Faradd. 90 mikro Farade. 180 mikro Farad


Kemungkinan kapasitas kapasitor pengganti 
a) susunan seri
1/C = 1/60 + 1/60 + 1/60
1/C = 3/60
3C = 60
C = 20 mikro Farad

b) susunan paralel
C = 60 + 60 + 60
C = 180 mikro Farad

c) susunan campuran (paralel dan seri )

untuk yang paralel : Cp = 60 + 60 = 120 mikro Farad
yang seri : Cs = 60 mirko Farad
kapasitas totalnya : 1/C = 1/Cp + 1/Cs
1/C = 1/120 + 1/60
1/C = 3/120
3C = 120
C = 40 mikro Farad

jadi kemungkinan opsi jawaban yang benar adalah c dan d

30. Tiga buah kapasitor masing masing kapasitasnya 3 farad, 6 farad, dan 9 farad dihubungkan secara seri, kedua ujung gabungan itu di hubungkan dengan sumber tegangan yang besarnya 220 volt. Tegangan antara ujung kapasitor 3 farad adalah?


1/Cp = 1/C₁ + 1/C₂ + 1/C₃
1/Cp = 1/(3 F) + 1/(6 F) + 1/(9 F)
Cp = 18/11 F

qp = Cp V
      = (18/11 F)(220 V) = 360 V

qp = q₁ = q₂ = q₃
V₁ = q₁ / C₁
     = (360 V) / (3 F) = 120 V
