SEMOGA MEMBANTU

Sebuah lingkaran tepat berada di dalam persegi. Jika ukuran rusuk persegi tersebut adalah 14 cm, maka luas lingkaran tersebut adalah 154 cm².

Pendahuluan :

[tex] \rm \blacktriangleright Pengertian[/tex]

Lingkaran adalah himpunan semua titik dibidang datar yang berjarak sama dari suatu titik tetap di bidang tertentu.

[tex] \\[/tex]

[tex] \rm \blacktriangleright Rumus~Umum~Lingkaran :[/tex]

[tex] \boxed{L = \pi {r}^{2}}[/tex]

[tex] \boxed{K = 2\pi r \: atau \: K = \pi d}[/tex]

dimana :

L = luas lingkaran

K = keliling lingkaran

[tex] \pi [/tex]= [tex] \frac {22}{7}[/tex] atau 3,14

r = jari-jari lingkaran

d = diameter lingkaran

[tex]\\[/tex]

[tex] \rm \blacktriangleright Rumus~Dari~Unsur~Lingkaran :[/tex]

[tex] \boxed {Luas\:Juring = \frac {\alpha}{360^o}\times Luas\:Lingkaran}[/tex]

[tex] \boxed {Panjang\:Busur = \frac {\alpha}{360^o}\times Keliling\:Lingkaran}[/tex]

dimana :

[tex] \alpha[/tex] = besar sudut dari juring lingkaran

Luas Lingkaran = πr²

Keliling Lingkaran = 2πr atau πd

[tex] \\[/tex]

[tex] \rm \blacktriangleright Rumus~Garis~Singgung~Lingkaran :[/tex]

[tex] \boxed{d = \sqrt{ {p}^{2} - (R+r)^{2} } }[/tex]

[tex] \boxed{l = \sqrt{ {p}^{2} - (R-r)^{2} } }[/tex]

dimana :

d = panjang garis singgung persekutuan dalam

l = panjang garis singgung persekutuan luar

p = jarak titik pusat dua lingkaran

R = jari-jari lingkaran besar

r = jari-jari lingkaran kecil

Pembahasan :

Diketahui :

Sebuah lingkaran berada di dalam persegiUkuran rusuk persegi adalah 14 cm

Ditanya :

Luas lingkaran?

Jawab :

Ukuran rusuk persegi = panjang diameter lingkaran.

Cari panjang jari-jari lingkaran :

[tex] \rm r = d \div 2[/tex]

[tex] \rm r = 14 \div 2[/tex]

[tex] \rm r = 7~cm[/tex]

Gunakan rumus Luas Lingkaran :

[tex] \rm L = \pi r^2[/tex]

[tex] \rm L = \frac{22}{7} \times 7^2[/tex]

[tex] \rm L = \frac{22}{7} \times 49[/tex]

[tex] \rm L = 22 \times 7[/tex]

[tex] \bf L = 154~cm^2[/tex]

Kesimpulan :

Jadi, luas lingkaran tersebut adalah 154 cm².
