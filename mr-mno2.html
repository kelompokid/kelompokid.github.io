1. Berapa gram PbO2 (Mr = 239,2) akan bereaksi dengan 11,592 g MnO2 (Mr = 86,94), menghasilkan Pb2+ dan MnO4 - ?


Jawaban: 31,89 gram

Penjelasan:

PbO₂ + MnO₂ ⇒ Pb²⁺ + MnO₄⁻

perbandingan mol sama dengan perbandingan koefisien

maka n MnO₂ = n PbO₂

n MnO₂ = m/Mr

n = 11,592/86,94

n = 0,13 mol

nPbO₂ = m/Mr

m = n x Mr

m = 0,13 x 239,2

m = 31,89 gram

Penjelasan:

Diketahui:

massa MnO2 = 8,7 grammassa HCl = 28,8 gramMr MnO2 = 87 gram/molMr HCl = 36 gram/molreaksi : MnO2 + HCl -> MnCl2 + Cl2 + H2O

Dijawab:

Setarakan

MnO2 + 4HCl -> MnCl2 + Cl2 + 2H2O

Menghitung mol diketahui

rumus : mol (n) = massa / massa relatif

n MnO2 = (8,7 gram) / (87 gram/mol) = 0,1 mol

n HCl = (28, 8 gram) / ( 36 gram/ mol) = 0,8 mol

Menentukan pereaksi pembatas

mol /koefisien

untuk MnO2 = 0,1 mol / 1 = 0,1 mol

untuk HCl = 0,8 mol / 4 = 0,2 mol

karena mol/ koefisien adalah MnO2 < HCl, maka pereaksi pembatas adalah MnO2

Mereaksikan

MnO2 + 4HCl -> MnCl2 + Cl2 + 2H2O

mula-mula: 0,1 mol 0,8 mol - - -

bereaksi : 0,1 mol 0,4 mol 0,1 mol 0,1 mol 0,2 mol

sisa : 0 mol 0,4 mol 0,1 mol 0,1 mol 0,2mol

Jawaban : 0,2 mol

notes:

cara menghitung mol beraksi menggunakan perbandingan koefisien, contoh :

reaksi : aA -> bB + cC

dengan a, b, c adalah koefisien

maka jika di cari mol C dan diketahui mol A adalah mol C = (c/a) x mol A

cara menghitung mol sisa adalah jika

reaktan, sisa = mula-mula - bereaksi

produk, sisa = mula-mula + bereaksi

16. Nama kimia dari senyawa MnO2


Nama kimia dari senyawa MnO₂ adalah Mangan (IV) Oksida

Penjelasan:

Untuk menjawab soal ini maka kita perlu mengingat kembali tata cara pemberian nama senyawa didasarkan pada jenis ikatannya yaitu:

1. Senyawa Ionik

Senyawa ionik merupakan senyawa yang tersusun dari unsur logam utama atau transisi dan non logam yang membentuk ikatan ionik.

Adapun pemberian nama senyawa ionik yaitu:

Nama Unsur Logam utama + Nama Unsur Nonlogam + Akhiran -ida

adapun jika senyawa ionik mengandung ion poliatom maka tata namanya menjadi:

Nama Unsur Logam Utama + Nama ion Poliatom

Untuk senyawa dari logam transisi yaitu:

Nama unsur Logam Transisi + (Biloks) + Nama unsur Non logam + Akhiran –ida

Adapun jika mengandung ion poliatom maka tata namanya menjadi

Nama unsur Logam Transisi + (Biloks) + Nama ion Poliatom

*Biloks ditulis dalam angka romawi dan diberi tanda kurung  

2. Senyawa Kovalen

Senyawa kovalen merupakan senyawa yang tersusun dari unsur non logam dan non logam yang membentuk ikatan kovalen.

Adapun pemberian nama senyawa kovalen yaitu:

Jumlah unsur Nonlogam 1 + Nama Unsur Logam 1 + Jumlah unsur Nonlogam 2 + Nama Unsur Nonlogam 2 + Akhiran -ida

Jumlah unsur dinyatakan dengan:

1 = mono

2 = di

3 = Tri

4 = Tetra

5 = Penta

6 = Heksa

7 = Hepta

8 = Okta

9 = nona

10 = Deka

* Jika unsur non logam pertama berjumlah 1, maka bisa ditulis dengan awalan mono atau tanpa awalan.

Berdasarkan penjabaran di atas, maka kita bisa memberikan nama pada senyawa-senyawa di soal, yaitu:

MnO₂

Merupakan senyawa ionik dari unsur logam Transisi Mangan (Mn) dengan biloks +4 dan unsur non logam Oksigen (O), maka tata namanya yaitu:

Nama unsur Logam Transisi + (Biloks) + Nama unsur Non logam + Akhiran –ida

sehingga:

Mangan + (IV) + Oksigen + -ida

maka namanya adalah Mangan (IV) Oksida

Penentuan Biloks mengikuti aturan Biloks, yaitu nilai Biloks untuk Oksigen (O) adalah -2 sehingga:

secara umum nilai Biloks untuk senyawa netral adalah 0, sehingga:

Biloks MnO₂ = 0

Biloks Mn + (2 x Biloks O) = 0

Biloks Mn + (2 x (-2)) = 0

Biloks Mn + (-4) = 0

Biloks Mn = +4

Demikian pembahasan tentang tata cara penamaan senyawa ionik Transisi.
