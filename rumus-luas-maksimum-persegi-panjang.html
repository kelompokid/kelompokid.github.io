1. Rumus luas maksimum suatu daerah berbentuk persegi panjang


Jawab:

Tidak ada nilai maksimum tanpa aturan pembatas, ini seperti garis y=x, yg gak berhenti nanjak.

Kecuali, .....

Penjelasan dengan langkah-langkah:

Kecuali dg aturan/batasan keliling tertentu, msal keliling K, nilai maksimum ada.

Keliling k = 2 (P+L) ==> P=k/2 - L

Fungsi Luas = PL = (K/2 - L)L = LK/2 - L^2

Maksimum bila turunan fungsi luas f'(L)=0.

f(L)=LK/2 - L^2

f'(L)=K/2 - 2L = 0

k/2 = 2L

L= k/4 ==========

Luas maks didapat bila lebar besarnya 1/4 keliling. Itu berarti persegi panjang yang punya 4 sisi sama panjang alias PERSEGI.

3. keliling suAtu persegi panjang 100 meter, jika panjangnya x meter dan lebarnya y meter. tulislah rumus luas untuk persegi panjang tsb dalam x dan tentukan luas maksimumnya


keliling persegi panjang adalah 2 x panjang + 2 x lebar
keliing = 2x + 2y = 100
[tex]x= \frac{100-2y}{2} = 50 - y[/tex]

luas = panjang x lebar
luas = x . y
luas = (50 - y) y
luas = 50y - y²
luas maksimum = turunan dari luas sama dengan 0
50- 2y = 0
2y = 50
y = 25

masukkan y kedalam persamaan awal
luas = 50 (25) - 25² = 625
luas = 625Diketahui:

l = y meter

2x = 100 - 2y
x = [tex] \frac{100 - 2y}{2} [/tex]
x = 50 - y

L = p * l
L = x * y
L = (50 - y) * y
L = 50y - y²

Lmax = 50y - y²
0 = 50 - 2y
2y = 50
y = 50/2
y = 25

Subtitusi ke Luas Lingkaran:

Suatu persamaan yang memiliki variabel dimana variabel tersebut mempunyai pangkat yang paling tinggi dua maka persamaan tersebut disebut sebagai fungsi kuadrat. Bentuk umum dari fungsi kuadrat adalah [tex]y = f(x) = ax^2 + bx + c[/tex], dengan keterangan x disebut sebagai variabel, kemudian a, b adalah koefisien dan c adalah suatu konstanta, serta nilai dari a tidak sama dengan 0. Oke langsung aja yukk disimak penjabaran jawaban soal kali ini.

Nomor 1

Diketahui : Suatu persegi panjang kelilingnya 60 cm.

Diperoleh :  

[tex]2(p + l) = 60\\p + l = 30\\p = 30 - l[/tex]

akan dicari fungsi kuadrat untuk luas persegi panjang

[tex]L = p \times l\\L = (30 - l) \times l\\L = 30l - l^2[/tex]

Dari fungsi kuadrat diatas diperoleh a = -1, b = 30, c = 0, sehingga dapat dicari nilai diskriminannya [tex]D = b^2 - 4ac = 30^2 - 4(-1)(0) = 900[/tex], maka luas maksimumnya adalah  

[tex]L = -\frac{D}{4a}\\L = -\frac{900}{4(-1)}\\L = 225[/tex]

Nomor 2

Diketahui : sebuah segitiga siku-siku jumlah kedua sisi siku-sikunya adalah 50 cm

Dimisalkan :

a = alas segitiga siku-sikub = tinggi segitiga siku-siku

Diperoleh :  

[tex]a + b = 50\\a = 50 - b[/tex]

akan dicari fungsi kuadrat untuk luas segitiga siku-siku

[tex]L = \frac{1}{2} \times a \times b \\L = \frac{1}{2} \times (50 - b) \times b\\L = 25b - \frac{1}{2}b^2[/tex]

Dari fungsi kuadrat diatas diperoleh a = -1/2, b = 25, c = 0, sehingga dapat dicari nilai diskriminannya [tex]D = b^2 - 4ac = 25^2 - 4(-\frac{1}{2})(0) = 625[/tex], maka luas maksimumnya adalah  

[tex]L = -\frac{D}{4a}\\L = -\frac{625}{4\big(-\frac{1}{2}\big)}\\L = 312.5[/tex]

Nomor 3

Diketahui : seorang siswa memotong selembar kain. Kain hasil potongannya berbentuk persegi panjang dengan keliling 80 cm

Diperoleh :  

[tex]2(p + l) = 80\\p + l = 40\\p = 40 - l[/tex]

akan dicari fungsi kuadrat untuk luas persegi panjang

[tex]L = p \times l\\L = (40 - l) \times l\\L = 40l - l^2[/tex]

Dari fungsi kuadrat diatas diperoleh a = -1, b = 40, c = 0, akan dicari lebar maksimumnya agar diperoleh luas maksimum

[tex]l = -\frac{b}{2a}\\l = -\frac{40}{2(-1)}\\l = 20[/tex]

sehingga [tex]p = 40 - l = 40 - 20 = 20[/tex], jadi agar diperoleh luas maksimum ukuran panjangnya 20 cm dan lebarnya 20 cm

Akan dicari diskriminannya [tex]D = b^2 - 4ac = 40^2 - 4(-4)(0) = 1600[/tex]

Akan dicari tinggi maksimum yang dapat dicapai peluru

[tex]h(x) = -\frac{D}{4a}\\h(x) = -\frac{1600}{4(-4)}\\h(x) = 100[/tex]

Akan dicari waktu yang diperlukan

[tex]t = -\frac{b}{2a}\\t = -\frac{40}{2(-4)}\\t = 5[/tex]

Penjelasan dengan langkah-langkah:Nomor 1

Untuk menentukan ukuran persegi panjang agar luasnya maksimum dari keliling yang diketahui, terdapat beberapa cara.

Cara Pertama: Dengan Pemahaman Bangun Datar

Luas sebuah persegi panjang mencapai maksimum jika panjang dan lebar persegi panjang tersebut bernilai sama, atau dengan kata lain, persegi panjang tersebut adalah sebuah persegi.

Berdasarkan definisinya, semua persegi adalah persegi panjang, namun tidak semua persegi panjang adalah persegi. Dapat dikatakan bahwa persegi adalah persegi panjang yang khusus/spesial, yang memiliki nilai panjang dan lebar yang persis sama.

Oleh karena itu:

[tex]\begin{aligned}&K = 2(p+l)=100\\&\Rightarrow p+l=50\\&\qquad(p=l)\\&\Rightarrow 2p=2l=50\\&\Rightarrow p=l=\boxed{\bf25\ cm}\end{aligned}[/tex]

Cara Kedua: Dengan Turunan Pertama

[tex]\begin{aligned}&K = 2(p+l)=100\\&\Rightarrow p+l=50\\&\Rightarrow l=50-p\\&L=pl=p(50-p)\\&\Rightarrow L=f(p)=50p-p^2\\\end{aligned}[/tex]

Agar luas persegi panjang, yang dinyatakan dengan [tex]f(p)[/tex], mencapai nilai maksimum, [tex]f(p)[/tex] harus mencapai titik kritisnya.

Karena [tex]f(p)[/tex] adalah fungsi kuadrat, maka titik kritisnya adalah titik stasioner.Karena koefisien [tex]p^2[/tex] adalah [tex]-1[/tex], maka benar bahwa titik stasionernya adalah titik di mana [tex]f(p)[/tex] maksimum, karena kurva [tex]f(p)[/tex] menghadap ke bawah.

Titik stasioner dari sebuah fungsi dicapai ketika gradien garis singgung (garis tangen) pada titik tersebut sama dengan 0. Artinya: [tex]f'(p) = 0[/tex].

[tex]\begin{aligned}0&=f'(p)\\&=\frac{d}{dp}\left(50p-p^2\right)\\0&=50-2p\\2p&=50\\\therefore\ p&=\boxed{\bf25\ cm}\\\therefore\ l&=50-25=\boxed{\bf25\ cm}\end{aligned}[/tex]

Cara Ketiga: Dengan Rumus Titik Puncak Fungsi Kuadrat

Dari cara kedua di atas, kita telah memperoleh:

[tex]L=f(p)=50p-p^2[/tex]

Titik puncak fungsi kuadrat memiliki absis = sumbu simetri, yaitu:

[tex]\begin{aligned}x_{\sf simetri}=x_{\sf puncak}=\frac{-b}{2a}\end{aligned}[/tex]

Dari [tex]f(p)=50p-p^2[/tex] ⇒ [tex]a = -1,\ b = 50[/tex]

Maka:

[tex]\begin{aligned}p_{\sf puncak}&=\frac{-50}{-2}=\bf25\end{aligned}[/tex]

Karena [tex]p+l=50[/tex], maka [tex]l=\bf25[/tex].

Sehingga diperoleh:

[tex]p=l=\boxed{\bf25\ cm}[/tex]

Cara Keempat: Dengan Ketaksamaan AM-GM

AM = Arithmetic Mean (Rataan Aritmetik), dan GM = Geometric Mean (Rataan Geometrik).

Ketaksamaan AM-GM dinyatakan oleh:

[tex]\begin{aligned}AM\ & \ge \ GM\\\sum_{i=1}^{n}x_i\ & \ge \ \sqrt[n]{\prod_{i=1}^{n}x_i}\\\frac{x_1+x_2+{\dots}+x_n}{n}\ & \ge \ \sqrt[n]{x_1x_2\,{\dots}\,x_n}\\\end{aligned}[/tex]

Dari keliling persegi panjang, diperoleh [tex]p+l=50 \implies p=50-l[/tex].

Maka, berdasarkan ketaksamaan AM-GM:

[tex]\begin{aligned}\frac{p+l}{2}\ & \ge \ \sqrt{pl}\\\frac{p+50-p}{2}\ & \ge \ \sqrt{pl}\\\frac{50}{2}\ & \ge \ \sqrt{pl}\\25\ & \ge \ \sqrt{pl}\\625\ & \ge \ pl\\\end{aligned}[/tex]

Agar [tex]L=pl[/tex] bernilai maksimum, nilai [tex]pl[/tex] harus mencapai 625, sehingga:

[tex]\begin{aligned}&pl=p(50-p)=625\\&\Rightarrow 50p-p^2=625\\&\Rightarrow p^2-50p=-625\\&\Rightarrow p^2-50p+625=0\\&\Rightarrow (p-25)^2=0\\&\Rightarrow p=25\ \Rightarrow\ l=50-25=25\\ &\therefore\ p=l=\boxed{\bf25\ cm}\end{aligned}[/tex]

KESIMPULAN
∴ Dengan demikian, agar luasnya maksimum, persegi panjang tersebut berukuran 25 cm × 25 cm (panjang = lebar = 25 cm).
