3. Sebuah trafo memiliki efisiensi 75%, jika daya sekunder yang dihasilkan adalah 1200 watt, maka daya primer trafo adalah.....


`Efisiensi Transformator

Diketahui :

efisiensi trafo (η) = 75%daya sekunder (Pₛ) = 1.200 Watt

Ditanya :

daya primer trafo (Pₚ) ?

Dijawab :

η = Pₛ/Pₚ × 100%

75% = 1.200/Pₚ × 100%

75/100 = 1.200/Pₚ × 100/100

Pₚ = 1.200 ÷ 75/100 ÷ 1

Pₚ = 1.200 × 100/75 ÷ 1

Pₚ = 120.000/75

Pₚ = 1.600 Watt

9. Efisiensi sebuah trafo 75% jika daya yg keluar dari trafo525 watt maka daya yg masuk kedalam trafo adalah


"Efesiensi Transformator 75% jika daya yang keluar 525 watt berapa daya yang masuk  ?"


Penjelasan Dengan langkah-langkah :

Dari hubungan jumlah lilitan, tegangan, dan arus secara matematis di Rumuskan sebagai berikut  

[tex]\frac{VP}{VS}[/tex] = [tex]\frac{NP}{NS}[/tex] = [tex]\frac{IS}{IP}[/tex]

Untuk efisiensi Transformator di Rumuskan sebagai berikut :  

μ = [tex]\frac{PS}{PP}[/tex] x 100%  μ = [tex]\frac{VS.IS}{VP.IP}[/tex] x 100%  μ = [tex]\frac{NS.IS}{NP.IP}[/tex] x 100%  

Keterangan  

VP = Tegangan Primer (volt)

VS = Tegangan Sekunder (volt)

NP = Lilitan Primer  

NS = Lilitan Sekunder

IS = Arus Sekunder ( A )

IP = Arus Primer (A)

PS = Tegangan Sekunder (watt)

PP = Tegangan Primer (watt)

μ = Efisiensi (%)  

=================================================================

Pembahasan soal  


Diketahui :  

μ   =  Efesiensi  =  75%

PS =  Daya sekunder   =  525 watt


Ditanyakan :

PP  =  Daya Primer  = ...   ?


Penyelesaian :

Gunakan Rumus :

%μ  =  PS/PP x 100%

75%  =  525/PP x 100%

75PP = 52.500

   PP = 52.500/75

   PP = 700


Jadi Daya Masukkan sebesar 700 watt

13. Sebuah trafo memiliki efisiensi 80% , jika daya sekunder trafo 320 watt, maka berapakah daya primer pada trafo tersebut!


Jawaban:

[tex]\bf Daya\ primer\ trafo\ 400\ Watt [/tex]

Penjelasan:

[tex]\sf\large Diketahui: [/tex]

[tex]\sf\large η=80\% [/tex]

[tex]\sf\large P_{Sekunder}=320\ Watt [/tex]

Ditanyakan besarnya daya Primer.

Jawab:

Gunakan rumus:

[tex]\sf\large η=\Large{\frac{P_{Sekunder}}{P_{Primer}}}\large×100\% [/tex]

[tex]\sf\large P_{Primer}=\Large{\frac{P_{Sekunder×100\%}}{η}} [/tex]

[tex]\sf\large P_{Primer}=\Large{\frac{320×100}{80}} [/tex]

[tex]\sf\large P_{Primer}=\Large{\frac{32.000}{80}} [/tex]

[tex]\sf\large P_{Primer}=400\ Watt [/tex]

Jadi daya sekundernya adalah [tex]\sf\large400\ Watt. [/tex]

Jawaban:

300 W

Penjelasan:

Efisiensi trafo = 75% = 0,75Daya primer = 400 W

Ditanya: Daya sekunder?

Jawab:

Rumus efisiensi trafo adalah

η = (daya sekunder / daya primer) x 100%

daya sekunder = (η x daya primer) / 100%
