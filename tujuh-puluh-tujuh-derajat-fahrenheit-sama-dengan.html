Lambang bilangan dari tujuh puluh tujuh ribu tujuh ratus tujuh adalah 77.707

• • •

☆ Pendahuluan

Bilangan Bulat adalah himpunan bilangan dimana nilainya tidak desimal atau tidak berbentuk pecahan dengan kata lain nilainya bulat, Menurut Wikipedia. Bilangan bulat adalah bilangan yang terdiri dari bilangan cacah dan negatifnya. Bilangan bulat dituliskan tanpa komponen desimal atau pecahan. Himpunan semua bilangan bulat dalam matematika dilambangkan dengan Z, berasal dari kata Zahlen ( Bahasa Jerman ) yang artinya bilangan.

Berikut beberapa sifat dalam operasi hitung bilangan bulat

a - ( - b ) → a + b → c

a + ( - b ) → a - b → Dapat c atau - c, Tergantung besar nilai positif ( + ) atau nilai negatif ( - )

( - a ) + ( - b ) → - ( a + b ) → - c

( - a ) × ( - b ) → c

( - a ) × b → - c

( - a ) ÷ ( - b ) → c( - a ) ÷ b → - c

( - a ) - ( - b ) → ( - a ) + b → Dapat c atau - c, Tergantung besar nilai positif ( + ) atau nilai negatif ( - )

a × b → c

a ÷ b → c

( - b ) × a → - c

( - b ) ÷ a → - c

Dalam bilangan bulat terdapat yang namanya nilai tempat satuan, Misalnya 1.234.567

Dimana:

Angka 1 → JutaAngka 2 → Ratus ribuAngka 3 → Puluh ribuAngka 4 → RibuanAngka 5 → RatusanAngka 6 → PuluhanAngka 7 → Satuan

Dalam menyelesaikan Operasi hitung bilangan bulat atau Operasi hitung campuran ada beberapa hal yang perlu diperhatikan yaitu, Bagian mana yang harus dikerjakan terlebih dahulu dan bagian mana yang perlu diselesaikan terakhir, yang biasa disebut Sistem Prioritas atau disebut " PEMDAS "

Berikut Sistem Prioritas ( PEMDAS )

P » ( Parentheses ) Angka Dalam Kurung ( )

E » ( Eksponents ) Eksponen ( a² atau a³ dan sejenisnya )

M & D » ( Multiplication & Division ) Perkalian ( × ) dan Pembagian ( ÷ )

A & S » ( Addition & Subtraction ) Penjumlahan ( + ) dan Pengurangan ( - )

» Pembahasan

"Tujuh puluh tujuh ribu" → 77.000, dimana:

Angka tujuh yang di tebalkan merupakan puluh ribuYang tidak ditebalkan merupakan ribuan sehingga dituliskan demikian.

setelah kalimat "Tujuh puluh tujuh ribu" yaitu "tujuh ratus tujuh" dan jika dituliskan akan menjadi 707, dimana:

7 → ratusan07 → satuan

Maka jika disatukan diperoleh:

➞ 77.000 + 707

➞ 77.707

• Kesimpulan

Jadi, Lambang bilangan dari tujuh puluh tujuh ribu tujuh ratus tujuh adalah 77.707
