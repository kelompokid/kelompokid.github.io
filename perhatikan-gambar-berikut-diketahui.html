4. Perhatikan gambar berikut. Diketahui berat benda = w, g = 10.....


Nilai T2 - T1= C. W/ 3. Hal ini sesuai dengan rumus mencari masing masing tegangan tali.

Penjelasan dengan langkah langkah :

Diketahui :

Berat benda = wsin a1 = 5/13 sin a2 = 4/5

Ditanya

Nilai T2 - T1 adalah

Dijawab

[tex]Sin \ a \ = \frac{5}{13} \ \ \ Cos \ a = \frac{12}{13} \\\\ sin \ a2 \ = \frac{4}{5} \ \ \ cos \ a2 \ = \frac{3}{5}[/tex]

Langkah 1 : Tentukan nilai T1

[tex]Fx \ = 0 \\\\ T2x \ - \ T1x \ = 0 \\\\ T2x \ = T1 x \\\\ \frac{3}{5} \ x \ = \frac{12}{13} \ T1 \\\\\\ T1 \ = \frac{\frac{3}{5} \ x \ T2 }{\frac{12}{13} } \\\\ T1 \ =\frac{3 \ x \ 13 \ x \ T2 }{5 \ x \ 12 } \\\\ T1 \ = \frac{13 \ T2 }{20}[/tex]

Langkah 2 : Tentukan nilai w

[tex]Fy \ = 0 \\\\ T1 y \ + T2y \ - \ w \ = 0 \\\\ \frac{5}{13} \ T1 \ + \frac{4}{5} \ T2 \ - \ w \ = 0 \\\\ \frac{5}{13} \ T1 \ + \frac{4}{5} \ T2 \ = \ w[/tex]

Langkah 3 : Subtitusikan nilai T1

[tex]\frac{5}{13} \ . \ \frac{13\ . \ T2 }{20 } \ + \frac{4}{5} \ T2 \ = w \\\\ \frac{5T2 }{20 } \ + \frac{16\ T2 }{20 } \ =\ w \\\\ \frac{1}{4} \ T2 \ + \frac{4}{5 } \ T2 \ = w \\\\ 0,25 \ T2 \ + 0,8 \ T2 \ = w \\\\ T2 \ = \frac{w}{1,05 }[/tex]

Langkah 4 : Tentukan nilai T2 - T1

[tex]T2 \ - T1 \ = \frac{w}{1,05 } \ - \frac{13 \ w }{21} \\\\ T2 \ - T1 \ = \frac{w}{3}[/tex]

Materi tentang keseimbangan benda tegar kelas 11. Dalam hal ini kita harus mengetahui bahwa syarat keseimbangan sistem partikel adalah F = 0. Pada sumbu x = 0 dan pada sumbu y = 0. Sehingga gaya yang bernilai 0, benda terus diam atau bergerak secara GLB. Pada keseimbangan sistem, benda bergerak lurus beraturan tetapi ini adalah keseimbangan kinetis. Keseimbangan statis yaitu benda tegar yang selalu diam tidak bergerak dan tidak berotasi sehingga nilai gayanya adalah 0.

V = 1/3×πr²×t

3.140 = 1/3×3,14×r²×30

3.140 = 3,14×r²×10

3.140 = 31,4 × r²

r² = 3.140 ÷ 31,4

r² = √100

r = 10 cm.

jadi, diameter alas nya adalah :

diameter = 2×jari-jari

• d = 2×r

= 2×10

d = 20 cm.

jadi,diameter alasnya adalah 20 cm.

=186-3y

= 126
