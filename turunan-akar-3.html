20. nilai turunan dari g(x)=akar x- akar x+1 di x=3 adalah


Materi Turunan

Diketahui [tex] g(x) = \sqrt{x - \sqrt{x + 1}} [/tex]
Nah, yang ditanyakan adalah g'(3).
Pertama-tama, turunkan dulu g(x) dengan menggunakan Aturan Rantai.
[tex] g(x) = (x - \sqrt{x + 1})^{\frac{1}{2}} \\ g'(x) = \dfrac{1}{2} \times \left(x - \sqrt{x + 1})^{-\frac{1}{2}} \times (1 - \dfrac{1}{2} \times (x+1)^{-\frac{1}{2}}\right) [/tex]
Langsung aja masukkan x = 3, sehingga ditulis
[tex] g'(3) = \dfrac{1}{2} \times (3 - \sqrt{3 + 1})^{-\frac{1}{2}} \times (1 - \dfrac{1}{2} \times (3+1)^{-\frac{1}{2}})\\ g'(3) = \dfrac{1}{2} \times 1 \times \dfrac{3}{4} = \dfrac{3}{8} [/tex]

27. turunan pertama dari f(x) = ×/ akar 2x+3​


Cara kerja

f(x) = x / √(2x + 3)

u = x

u'= 1

v = √(2x + 3) = (2x + 3)^½

v' = ½(2x + 3)^-½ (2)

= (2x + 3)^-½

f'(x) = (u'v - uv') / v²

= (1(2x+3)^½ - x(2x+3)`½) / (2x + 3)^½)²

= (√(2x + 3) - x/(√2x + 3)) / (2x + 3)

= (√(2x + 3)² - x / √(2x + 3)) / (2x + 3)

= (2x + 3 - x)/(√2x + 3) / (2x + 3)

= (x + 3)/(√(2x + 3) × 1/(2x + 3)

= (x + 3) / (2x + 3)√(2x + 3)

untuk x = -1

f'(-1) = (-1 + 3) / (2(-1) + 3))√(2(-1) + 3)

= 2 / 4√4

= 2 / 8

= ¼

untuk x = 1

f'(1) = (1 + 3) / (2(1) + 3)√(2(1) + 3)

= 4 / 5√5

= 4√5 / 5.5

= (4/25) √5
