27. sebuah segitiga A, B, dan C. besar sudut B sama dengan dua kali besar sudut A. besar sudut C sama dengan tiga kali besar sudut A. tentukan besar sudut setiap sudut segitiga tersebut ?​


Jawaban:

<A = 30°

<B = 60°

<C = 90°

Penjelasan dengan langkah-langkah:

Perincian pernyataan dalam soal :

• <B = 2 × <A

<B = 2<A

• <C = 3 × <A

<C = 3<A

• <A = 1 × <A

<A = 1<A atau <A

Jumlah sudut segitiga = 180°. Sehingga :

<A + <B + <C = 180°

<A + <B + <C = 180°

1<A + 2<A + 3<A = 180°

6<A = 180°

<A = 180°/6

<A = 30°

Masukkan nilai <A = 30° ke <B untuk mencari nilai <B.

<B = 2<A

<B = 2 × <A

<B = 2 × 30°

<B = 60°.

Masukkan nilai <A = 30° ke <C untuk mencari nilai <C.

<C = 3<A

<C = 3 × <A

<C = 3 × 30°

<C = 90°.

Jadi, besar sudut A = 30°, sudut B = 60°, sudut C = 90°.

Keterangan :

• < = sudut
