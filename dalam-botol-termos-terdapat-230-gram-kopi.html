mk = 230 g
tk = 90'c

ts = 5'c

suhu campuran (tc)....?

tc = 20800/250

Ingat asas Black !

Q lepas = Q serap

m kopi × c kopi × ( T kopi - Tx ) = m susu × c susu × ( Tx - T susu )

230 × 1 × ( 90 - Tx ) = 200 × 1 × ( Tx - 5 )

230 × ( 90 - Tx ) = 200 ( Tx - 5 )

23 (90 - Tx ) = 20 ( Tx - 5 )

2070 - 23Tx = 20Tx - 100

2070 + 100 = 23Tx + 20Tx

2170 = 43Tx

2170/43 = Tx

Tx = 50,46°C ←


Maka, suhu campurannya adalah 50,46°C.

5. Botol termos berisi 230 gram kopi pada suhu 80°c kemudian ditambah susu sebanyak 20 gram bersuhu5°c maka berapakah suhukesetimbangan campuran?​tolong dong buat hari senin


Pada soal, kopi bercampur dengan air, susu juga sama bercampur dengan air. berarti kalor jenis air kopi sama dengan kalor jenis air kopi
massa kopi = 230 gr
suhu kopi = 80° C
massa susu = 20 gr
suhu susu = 5° C

m × c × ∆T = m × c × ∆T
230 × c × (80° - Tc) = 20 × c × (Tc - 5°)
230 × (80 - Tc) = 20 × (Tc - 5)
18400 - 230 Tc = 20 Tc - 100
20 Tc + 230 Tc = 18400 + 100
250 Tc = 19400
Tc = 19400 ÷ 250
Tc = 77,6°C
Suhu kesetimbangan campuran adalah 77,6°C
