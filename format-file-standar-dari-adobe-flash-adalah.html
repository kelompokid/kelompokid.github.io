17. Sebut dan jelaskan beberapa format file pada program adobe photoshop


Beberapa format file pada program Adobe Photoshop adalah sebagai berikut


- PSD,PDD (Photoshop Document). Merupakan format penyimpanan asli dari Photoshop

- PSB (Photoshop Big). Digunakan untuk meningkatkan resolusi maksimum

- BMP, RLE, DIB. Digunakan untuk menyimpan gambar tipe bitmap

- GIF. Digunakan untuk menyimpan gambar tipe GIF. Memiliki kombinasi 256 warna. Biasa digunakan untuk logo, line drawing dan icon

- EPS (Encapsutaed Potscript). Digunakan untuk pencetakan atau pertukaran dokumen antar program grafis

- IFF (Interchange File). Digunakan untuk video

- JPEG. Digunakan untuk menyimpan mode warna RGB, CYMK, Grayscale

- JPEG 2000 (JPF,JPX,JP2,J2C,J2K,JPC). Merupakan standar kompresi untuk meningkatkan rasio kualitas dan kompresi

- JPS (JPEG Stereo). Terdiri dari side by side berdasarkan JPEG

- MPO (Multi Picture Format). Digunakan untuk format 3D

- PCX (Personal Computer Exchange). Berbentuk rester.

- Photoshop PDF. Digunakan oleh Adobe Acrobat

- RAW. Digunakan untuk menyimpan warna RGB, CMYK dan Grayscale

- PXR (Pixar). Digunakan untuk pertukaran file dengan Pixar Image Computer

- PNG (Portable Network Graphics). Digunakan untuk transparasi dalam browser

- Portable Bit Map (PBM,PGM,PPM,PNM,PFM,PAM). Digunakan untuk menyimpan informasi dengan kualitas 1 bit hingga 24 bit

- Scitex CT (SCT). Digunakan untuk image processing pada komputer Scitex

- Targa (TGA,VDA,ICB,VST)

- Tagged Image Format File (TIF,TIFF). Digunakan untuk pertukaran file antara aplikasi dan platform komputer.

22. format file hasil penyimpanan scanner dengan standar Windows format​


Jawaban:

PDF (Portable Document Format)

Format penyimpanan file dalam bentuk “.pdf” yang mampu menampilkan data berupa teks maupun gambar. Setiap file menyimpan bentuk format penulisan dan jenis huruf. Secara umum dibuka menggunakan software Adobe Acrobat Reader.

JPEG (Joint Photography Expert Group) atau JPG

Format ini dikhususkan untuk penyimpanan file gambar/foto. Format “.jpeg” ini dapat diatur tingkat kompresi nya, sehingga antara kualitas dan kapasitas penyimpanan bisa diatur (semakin besar kompresi gambar maka semakin kecil ukuran file, begitu juga sebaliknya). Akan tetapi, hasil scan JPEG apabila dikompres terlalu banyak akan menghilangkan detail gambar dan warna yang kurang tajam. Secara umum dibuka menggunakan Windows Picture, Paint, dan software lain yang menunjang.

PNG (Portable Network Graphic)

Format file “.png” ini dikhususkan juga untuk penyimpanan file gambar/foto, hampir sama seperti format .jpeg. Namun bentuk file ini optimal untuk menyimpan gambar yang banyak menggunakan warna solid (tidak lebih dari 256 warna), karena dasar algoritma penyimpanannya menggunakan “jumlah warna” hasil penyimpanannya hampir tanpa kompresi sama sekali, sehingga file gambar akan tetap sama seperti aslinya. Bentuk format ini bisa menampilkan gambar transparan. Secara umum dibuka menggunakan software Windows Picture, Paint, dan software lain yang menunjang.

BMP (Bitmap)

Hampir sama dengan format .tiff hanya bedanya “.bmp” biasanya digunakan untuk format file uncompressed (tidak ada kompresi) di sistem operasi Windows. Akan tetapi format ini sudah jarang digunakan karena ukuran file yang besar, sedangkan kualitasnya bisa hampir disamakan dengan format lainnya. Kelebihannya adalah file ini mendukung transparant layer. Secara umum dibuka menggunakan software Windows Picture, Paint, dan software lain yang menunjang.

30. format file apa yg dihasilkan adobe frame marker


adobe frame maker adalah prosesor dokumen yang dirancang untuk menulis dan mengedit dokumen besar atau kompleks, termasuk dokumen terstruktur. Hal ini dihasilkan oleh Adobe System.

FrameMaker menjadi salah satu produk dari Adobe pada tahun 1995 ketika Adobe membeli Frame Technology Corp Adobe menambahkan dukungan SGML, yang akhirnya berubah menjadi dukungan XML hari ini. Pada bulan April 2004, Adobe berhenti mendukung FrameMaker untuk Macintosh. Ini kebangkitan rumor muncul ke permukaan pada tahun 2001 bahwa pengembangan produk dan dukungan untuk FrameMaker sedang down. Adobe membantah rumor tersebut pada tahun 2001, kemudian melepaskan FrameMaker 8 pada akhir Juli 2007, FrameMaker 9 tahun 2009, FrameMaker 10 tahun 2011, FrameMaker 11 tahun 2012, dan FrameMaker 12 tahun 2014.  FrameMaker memiliki dua cara mendekati dokumen: terstruktur dan tidak terstruktur.

FrameMaker terstruktur digunakan untuk mencapai konsistensi dalam dokumentasi dalam industri seperti aerospace, di mana beberapa model produk kompleks yang sama ada, atau obat-obatan, di mana terjemahan dan standarisasi persyaratan penting dalam komunikasi tentang produk. FrameMaker terstruktur menggunakan SGML dan XML konsep. Penulis bekerja dengan EDD (Document Definition Element), yang merupakan DTD-FrameMaker spesifik (Document Type Definition).

EDD mendefinisikan struktur dokumen dimana unit-unit yang ditunjuk sebagai elemen bersarang satu sama lain tergantung pada hubungan mereka, dan di mana format elemen ini didasarkan pada konteks mereka. Atribut atau Metadata dapat ditambahkan ke elemen-elemen ini dan digunakan untuk penerbitan sumber tunggal atau unsur-unsur penyaringan selama proses output (seperti penerbitan untuk mencetak atau tampilan berbasis web). Penulis dapat melihat kondisi dan konteks dalam struktur seperti pohon yang berasal dari tata bahasa (sebagaimana ditentukan oleh DTD) atau sebagai diformat dalam bentuk hasil akhir yang khas.

FrameMaker Unstructured menggunakan paragraf tag tanpa struktur logis yang dikenakan, kecuali yang diungkapkan oleh konsep penulis, organisasi topik, dan format yang disediakan oleh tag paragraf. Ketika pengguna membuka file terstruktur di FrameMaker terstruktur,maka struktur hilang.
