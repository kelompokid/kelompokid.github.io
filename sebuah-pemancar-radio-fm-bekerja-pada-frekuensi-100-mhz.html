9. cepat pemancar radio "Sutrisno Jaya FM" mengudara dengan frekuensi 6 mhz. Jika cepat rambat gelombang radio 3.10pangkat8 m/s, panjang gelombang radio adalah


Panjang gelombang radio adalah [tex]\boxed{\bold{50\: m}}[/tex]

Pembahasan

Ada gelombang kalau “ada getaran” , walaupun gelombangnya tak kasat mata, saat getaran besar menandakan cepat rambat gelombangnya yang terlalu tinggi.

Gelombang itu ada dua :

Berdasarkan butuh tidaknya medium : Gelombang mekanik (selalu membutuhkan medium), Gelombang elektromagnetik (tanpa menggunakan medium rambat)

Berdasarkan bentuk dan arah gerak : (1) Gelombang transversal -> merambat naik turun dan Gelombang longitudinal (merambat seperti per).

Rumus mencari panjang gelombang yaitu

[tex]\boxed{\bold{\lambda = \frac{S}{n}}}[/tex]

Dimana :

[tex]\bold{\lambda}[/tex] = panjang gelombang. [tex]\bold{S}[/tex] = jarak gelombang.[tex]\bold{n}[/tex] = banyak gelombang.

Rumus mencari cepat rambat gelombang yaitu

[tex]\boxed{\bold{v = \lambda \times f}}[/tex] atau

[tex]\boxed{\bold{v = \frac{\lambda}{T}}}[/tex]

Dimana :

[tex]\bold{v}[/tex] = cepat rambat gelombang. [tex]\bold{f}[/tex] = frekuensi. [tex]\bold{T}[/tex] = periode.

Langkah Penyelesaian

Diket :

[tex]\bold{f = 6 \: MHz = 6.10^6 \: Hz}[/tex][tex]\bold{v = 3.10^8 \: m/s}[/tex]

Ditanya :

[tex]\bold{\lambda}[/tex] ?

Dijawab :

[tex]\bold{\lambda = \frac{v}{f}}[/tex]

[tex]\bold{\lambda = \frac{3.10^8}{6.10^2}}[/tex]

[tex]\bold{\lambda = 0.5 \: . \: 10^2}[/tex]

[tex]\bold{\lambda = 50 \: m}[/tex]

Kesimpulan :

Jadi panjang gelombang radio adalah [tex]\boxed{\bold{50\: m}}[/tex]

15. 5. Pemancar gelombang radio FM beroperasi pada frekuensi 60 MHz, Jika kecepatan gelombangradio 3.10⁸m/s. Berapa kecepatan gelombang tersebut?​


Mungkin maksud Anda : Pemancar gelombang radio FM beroperasi pada frekuensi 60 MHz, jika kecepatan gelombang radio 3.10⁸m/s. Berapa panjang gelombang tersebut ? [Didasarkan pada adanya informasi hilang yang memang harus dicari yaitu panjang gelombang].

Jadi, panjang gelombang radio tersebut yaitu 5 m.

Pendahuluan :

Hi ! Kali ini kita akan membahas mengenai hubungan frekuensi dengan cepat rambat gelombang. Cepat rambat gelombang yaitu banyaknya getaran yang dirambatkan dalam suatu panjang gelombang dalam waktu tertentu. Konsep yang berlaku pada kejadian ini, yaitu :

Pertambahan frekuensi sebanding dengan pertambahan cepat rambat gelombang, hal ini berarti : "Semakin besar nilai frekuensi dari suatu benda, maka menyebabkan getaran yang semakin banyak dalam waktu yang lebih singkat. Hal ini juga mempengaruhi cepat rambat gelombang.".Pertambahan panjang gelombang sebanding dengan pertambahan cepat rambat gelombang, hal ini berarti : "Semakin panjang gelombang yang dilintasi dalam waktu dari frekuensi yang tetap, maka seolah-olah mempercepat kecepatan rambat suatu gelombang".Pembahasan :

Berdasarkan konsep-konsep dasar yang telah dipelajari, maka disimpulkan rumus yang berlaku sebagai berikut :

[tex] \boxed{\bold{v = \lambda \times f}}[/tex]

Dengan ketentuan :

v = cepat rambat gelombang (m/s)[tex] \lambda [/tex] = panjang gelombang (m) f = frekuensi getaran (Hz)Langkah Penyelesaian :

Diketahui :

v = cepat rambat gelombang = [tex] 3 \times 10^8 [/tex] m/sf = frekuensi getaran = 60 MHz = [tex] 60 × 10^6 [/tex] = [tex] \boxed{6 \times 10^7} [/tex] Hz

Ditanya : [tex] \lambda [/tex] = panjang gelombang = ... m

Jawaban :

[tex] v = \lambda \times f [/tex]

[tex] 3 \times 10^8 = \lambda \times (6 \times 10^7)[/tex]

[tex] \lambda = \frac{3 \times 10^8}{6 \times 10^7} [/tex]

[tex] \lambda = 0,5 \times 10^{8-7} [/tex]

[tex] \lambda = 0,5 \times 10^1 [/tex]

[tex] \boxed{\lambda = 5 \: m} [/tex]

23. sebuah pemancar radio bekerja pada frekuensi setinggi 3,5 Mhz. hitung panjang gelombang pemancar tersebut


Sebuah pemancar radio bekerja pada frekuensi setinggi 3,5 Mhz. Maka panjang gelombang pemancar tersebut adalah 600/7 m

PEMBAHASAN

Gelombang sinusoidal adalah gelombang yang membentuk fungsi sinus. Beberapa rumus dasar gelombang yang perlu di ingat adalah sebagai berikut ini :

f = 1 / T

v = λ . f

Δx / λ = ΔΘ / 2π

Δt / T = ΔΘ / 2π

dimana :

f = frekuensi gelombang ( Hz )

T = periode gelombang ( s )

v = cepat rambat gelombang ( m/s )

λ = panjang gelombang ( m )

Δx = jarak pisah antar dua titik pada gelombang ( m )

Δt = selang waktu ( s )

ΔΘ = beda fase ( radian )

Bentuk persaman gelombang adalah :

y = A sin ( ωt - kx )

dimana :

y = simpangan gelombang ( m )

A = amplitudo gelombang ( m )

ω = frekuensi sudut ( rad/s )

t = waktu ( s )

k = bilangan gelombang ( 1/m ) = 2π/λ

x = jarak titik dari sumber ( m )

Okay mari kita gunakan rumus ini untuk menyelesaikan soalnya.

Diketahui :

f = 3,5 MHz = 3,5 x 10⁶ Hz

v = 3 x 10⁸ m/s

Ditanyakan :

λ = ?

Penyelesaian :

λ = v / f

λ = 3 x 10⁸ / (3,5 x 10⁶)

λ = 600/7 m
