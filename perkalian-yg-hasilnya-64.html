Gagal : 25% = 1/4
Peluang saling berhubungan:
P(A) = P(1) x P(2) x P(3)
P(A) = Gagal x Berhasil x Berhasil
P(A) = 1/4 x 3/4 x 3/4
P(A) = 9/64hasil bidik yang tepat = 75% = 75/100 = 3/4
hasil bidik yang gagal = 25% = 25/100 = 1/4
peluang menembak 3 kali dg hasil utk pertama kali meleset dan dua kali berikut tepat adalah
Solusi :
P(A) = P(1) x P(2) x P(3) = gagal x tepat x tepat
P(A) = 1/4 x 3/4 x 3/4 = 3/16 x 3/4 = 9/64 = B
