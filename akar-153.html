29. akar-akar persamaan kuadrat x²+8x-153=0​


x² + 8x - 153 = 0

x² - 9x + 17x - 153 = 0

x(x - 9) + 17(x - 9) = 0

(x + 17)(x - 9) = 0

x = -17 atau x = 9

Akar-akar persamaan kuadrat x² + 8x - 153 = 0 adalah {-17, 9}.
