9. BENI LIKES THE PARK BECAUSE


Jawaban:

BENI LIKES THE PARK BECAUSE the park is wide, clean and green.

Penjelasan:

Beni menyukai taman karena tamannya luas, bersih dan hijau. Kalimat menggunakan bentuk Simple Present Tense karena menjelaskan tentang fakta.

Simple Present Tense:

Nominal: (+) Subject + Tobe (is/am/are) + Adjective (kata sifat)

Verbal: (+) Subject + Verb-1 S/ES + Object

Semoga membantu ya.

Jawaban:

BENI LIKES THE PARK BECAUSE THE PARK lS BEAUTIFUL, TIDY, SHADY, AND THE FLOWER IS COLOURFUL.

Penjelasan:

menggunakan kata sifat atau adjective, yaitu

beautiful ( cantik/indah)tidy ( rapi ) Shady ( teduh )Colourful ( banyak warna/berwarna warni )

Menggunakan pola simple present tense nominal, karena terdapat ToBe dan kata sifat.

(+) Subject + ToBe(ls/Am/Are) + Adjective/Noun/C

(-) Subject + ToBe(ls/Am/Are) + not + Adjective/Noun/C

(?) ToBe(ls/Am/Are) + S + Adjective/Noun/C?

(-?) ToBe(ls/Am/Are) + not + S + Adjective/Noun/C?

Menggunakan pula pola simple present tense verbal, karena ada Subject ( beni ) + V1 + s ( likes )

(+) S + V1 + O/C <---- They/We/You/I

(+) S + V1 .s/es + O/C <----- He/She/lt

(-) S + V1 + do + not + O/C <---- They/We/You/I

(-) S + V1 + does + not + O/C <---- He/She/lt

(?) Do/Does + S + V1 + O?

(-?) Do/Does + not + S + V1 + O?

*Dapat pula sebelum ToBe atau pada awal kalimat interogatif (?) bisa menggunakan pola 5W1H5W1HWhat WhyWhereWhenWhoHow
