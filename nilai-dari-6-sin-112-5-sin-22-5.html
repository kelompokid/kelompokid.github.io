1. Tentukan nilai dari:a) cos 22, 5°b) sin 67, 5°c) sin 112 1/2°d) cos 157 1/2°​


a) cos 22,5° = cos 22 1/2 °

cos 22 1/2 ° = cos 1/2 45 °

[tex] \cos( \frac{1}{2} ) {45}^{0} = \sqrt{ \frac{1 \ + \cos( {45}^{0} ) }{2} } \\ = \sqrt{ \frac{1 + \cos( \frac{1}{2} ) \sqrt{2} }{2} } \\ = \sqrt{ \frac{1}{2} + \frac{ \sqrt{2} }{4} } \\ = \sqrt{ \frac{2 + \sqrt{2} }{4} } \\ = \sqrt{ \frac{2 + \sqrt{2} }{ \sqrt{4} } } \\ = \frac{1}{2} \sqrt{2 + \sqrt{2} } [/tex]

a] cos 22,5° = cos (45°/2)

cos (45°/2) = V[(cos 45° + 1)/2]

cos 22,5° = V[(1/2 V2 + 1)/2]

cos 22,5° = V[(V2 + 2)/4]

cos 22,5° = 1/2 V(V2 + 2)

b] sin 67,5° = sin (135°/2)

sin (135°/2) = V[(1 - cos 135°)/2]

sin 67,5° = V[(1 - (-1/2 V2))/2]

sin 67,5° = V[(1 + 1/2 V2)/2]

sin 67,5° = V[(2 + V2)/4]

sin 67,5° = 1/2 V(2 + V2)

c] sin 112 1/2° = sin (225°/2)

sin (225°/2) = V[(1 - cos 225°)/2]

sin 112 1/2° = V[(1 - (-1/2 V2))/2]

sin 112 1/2° = V[(1 + 1/2 V2)/2]

sin 112 1/2° = V[(2 + V2)/4]

sin 112 1/2° = 1/2 V(2 + V2)

d] cos 157 1/2° = cos (315°/2)

cos (315°/2) = V[(cos 315° + 1)/2]

cos 157 1/2° = V[(1/2 V2 + 1)/2]

cos 157 1/2° = V[(V2 + 2)/4]

cos 157 1/2° = 1/2 V(V2 + 2)

karena berada di Kuadran II , maka :

cos 157 1/2° = -1/2 V(V2 + 2)

18. jika a = cos 22° dan b = sin 22°, maka berapa hasil dari cosec 112° cos 202° + cos 292°? tolongg ya kak dijawab....​


Jawab:

-1 + b

Penjelasan dengan langkah-langkah:

diketahui

cos 22° = asin 22° = b

ditanya:

cosec 112° · cos 202° + cos 292°

jawab:

cosec 112°

= cosec (90° + 22°)

= sec 22° (karena 112° berada di kuadran II, maka sin cosec bernilai positif)

= 1 / cos 22°

= 1/a

cos 202°

= cos (180° + 22°)

= -cos 22° (karena 202° berada di kuadran III, maka cos bernilai negatif)

= -a

cos 292°

= cos (270° + 22°)

= sin 22° (karena 292° berada di kuadran IV, maka cos bernilai positif)

= b

cosec 112° · cos 202° + cos 292°

= 1/a · -a + b

= -1 + b
