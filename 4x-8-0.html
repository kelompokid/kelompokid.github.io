2. persamaan garis sejajar garis 4X + Y + 20 = 0 dan memotong sumbu x dititik (2,0) adalah. A.4x+y-8=0 B.-4X-Y-8=0 C.4X-Y+8=0 D-4X+Y+8=0 ​


Jawaban:

C

Penjelasan dengan langkah-langkah:

Dari persamaan pertama, kita dapat menuliskan kembali persamaannya menjadi bentuk y = mx + b, di mana m adalah kemiringan garis dan b adalah titik potong sumbu y (intercept). Kita dapat menghitung nilai m dengan membagi koefisien y dengan koefisien x, yaitu m = -4/1 = -4. Kemudian, kita dapat menghitung nilai b dengan menggunakan titik (2,0) yang diberikan. Untuk menghitung b, kita perlu mengganti x dengan 2 dan y dengan 0 dalam persamaan y = mx + b. Ini akan memberikan kita b = -4(2) + 0 + 20 = 8. Dengan demikian, persamaan garis yang sejajar dengan garis 4x + y + 20 = 0 dan memotong sumbu x di titik (2,0) adalah 4x - y + 8 = 0, sehingga jawabannya adalah C.

Persamaan Garis

ax + by + c = 0

gradien m = -a/b

2 garis sejajar → a dan b senilai

garis 1 :

4x + y + 20 = 0

persamaan garis yang sejajar garis 1 :

4x + y + c = 0

memotong sumbu x di (2,0)

4x + y + c = 0

4.2 + 0 + c = 0

8 + c = 0

c = -8

PGL :

4x + y - 8 = 0

Opsi : A
