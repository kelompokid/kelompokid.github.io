8. apasih pengertian address line dan control line


Address line digunakan untuk menentukan sumber data dari sebuah data bus. Oleh karena memori terbagi menjadi array linear dari tipe data bytes atau words, maka untuk membaca atau menulis sebuah informasi ke memori CPU harus menentukan alamat dari lokasi tersebut. Alamat ini disediakan oleh address bus ( address line dapat juga disebut dengan address bus ). Dengan demikian, lebar dari alamat tersebut menentukan jumlah memori maksimal yang dimungkinkan oleh sebuah sistem.  Sebagai contoh, jika sebuah sistem membuat lebar address bus sebesar 16 bit maka sistem tersebut dapat memiliki ukuran memori sebesar 2(16) = 65536 bytes.


Control lines digunakan untuk mengatur akses ke data dan address bus. Hal ini diperlukan karena address bus merupakan medium yang berbagi. Control lines dapat juga disebut dengan control bus. Control lines digunakan untuk transmisi perintah dan pengaturan waktu dari signal ( yang meng-validasi data dan alamat ) antara modul sistem. Pengaturan waktu dari signal atau Timing signals menentukan apakah data dan informasi alamat tersebut valid atau tidak valid, dan signal perintah menentukan jenis opeasi yang akan dilakukan. Beberapa control lines dari sebuah bus diperlukan untuk menyediakan clock signal untuk melakukan sinkronisasi operasi yang ada, mengatur ulang signal untuk inisialisasi modul. Control lines juga diperlukan untuk membaca dan menulis ke alat input output atau memori.

24. buat perintah command line untuk ip address LAN-1 dengan 192.168.100.12/24​


JAWABAN

Mengkonfigurasi pengalamatan jaringan (ip address) pada sistem operasi windows adalah hal yang sangat mudah, mungkin kawan-kawan semua setuju. Tampilan windows yang berbasis grafik (GUI - Graphical User Interface) sangat memudahkan user untuk mengoperasikannya. Kita cukup menekan icon-icon dan menu menggunakan mouse atau touchpad.

Konfigurasi IP Melalui Command Prompt

Untuk mengkonfigurasi ip address pada sistem operasi windows, biasanya kita akan menuju ke Control Panel > Network and Internet > Network and Sharing Center atau dengan mengeklik kanan icon network yang ada pada taskbar kemudian memilih opsi Open Network and Sharing Center lalu memilih change adapter settings, Kemudian mengkonfigurasi adapter jaringannya.

Bagaimana jika konfigurasi ip dilakukan menggunakan command line seperti saat kita mengkonfigurasi ip di sistem operasi linux berbasis teks? Tentu akan terasa menarik dan mungkin saja proses konfigurasinya akan terasa lebih simple dan cepat.

Di windows kita dapat menggunakan command prompt untuk mengkonfigurasi pengalamatan jaringan (ip address, gateway, dns). Untuk melakukan hal tersebut kita bisa menggunakan perintah netsh yang ada pada command prompt.

Netsh (Network Shell) merupakan utility perintah yang digunakan untuk memodifikasi konfigurasi jaringan seperti alamat ip, subnet prefiks, gateway serta server dns. Bahkan perintah netsh juga bisa digunakan untuk mengkonfigurasi firewall.

Pada postingan ini kita hanya fokus membahas tentang perintah-perintah netsh yang digunakan untuk melakukan konfigurasi pengalamatan jaringan pada komputer. Berikut adalah beberapa perintahnya :

Melihat daftar interface yang ada pada komputer

netsh interface show interface

Perintah tersebut akan menampilkan daftar interface jaringan yang ada pada komputer. Contoh :

Melihat konfigurasi ip

netsh interface ipv4 show config

Perintah di atas akan menampilkan info konfigurasi pengalamatan ip versi 4 pada setiap interface yang ada pada komputer, seperti ip address,  gateway, serta dns server.

Kita bisa juga menggunakan perintah berikut :

netsh interface ipv4 show addresses

Bedanya dari perintah sebelumnya adalah, tidak ada informasi tentang dns server dan wins server yang ditampilkan. Perintah tersebut hanya menampilkan info konfigurasi ip address, subnet mask, gateway, serta metric.

26. Address by ID artinya


Address by ID is address which based on your identity card, for example Kartu Tanda Penduduk/KTP or Kartu Pelajar/Student Card. Jadi alamat ID adalah alamat berdasarkan kartu identitas masing-masing orang yang biasanya dijadikan patokan dalam mendaftar sesuatu yang resmi.

Pembahasan

ID adalah kependekan dari Identity Document atau dokumen identitas pribadi. Di negara kita yang menjadi patokan adalah:

Kartu Tanda Penduduk atau KTP bagi orang dewasa.Kartu Keluarga bagi orang dewasa atau bagi anak-anak. Kartu Pelajar bagi remaja yang masih sekolah. Paspor bisa juga dijadikan kartu identitas. SIM atau Surat Izin Mengemudi bagi orang dewasa bisa juga dijadikan ID.

Secara singkat patokan dalam dokumen resmi yang diperlukan dalam membuat surat resmi atau mendaftarkan diri pada sesuatu yang legal dan memerlukan identitas resmi maka dipakailah ID kita.

Ada juga hal penting lainnya dalam menuliskan address atau alamat kita dalam bahasa Inggris yaitu kita tidak perlu menerjemahkan alamat kita ke dalam bahasa Inggris, karena alamat masing-masing negara adalah khusus bagi negara tersebut. Misalnya alamat kita adalah:

Jalan Kenangan No. 99 Bogor

Alamat tersebut tidak perlu diterjemahkan menjadi "99 th Kenangan Street, Bogor."  Cukup ditulis seadanya saja dalam bahasa Indonesia. Mengapa demikian? Karena bisa jadi alamat tersebut akan dipakai untuk pengiriman dari luar negeri, bila alamatnya ditulis dalam bahasa Inggris tentu akan menyulitkan kurir di Indonesia untuk memahami alamat tersebut.
