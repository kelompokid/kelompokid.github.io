2. Sebuah mobil pemadam kebakaran dan motor bergerak saling mejauh motor bergerak dengak kecepatan 40m/s mobi pemadaml kebakaran membunyikan sirine dengan frekuensi 400Hz dan di dengar oleh pengendara motor dengan frekuensi 300Hz jika cepat rambat bunyi di udara 340m.s hitunglah kecepatan mobil pemadam kebakaran


Sebuah mobil pemadam kebakaran dan motor bergerak saling mejauh motor bergerak dengak kecepatan 40m/s mobi pemadam kebakaran membunyikan sirine dengan frekuensi 400Hz dan di dengar oleh pengendara motor dengan frekuensi 300Hz jika cepat rambat bunyi di udara 340m/s , maka kecepatan mobil pemadam kebakaran 60 m/s

PEMBAHASAN

Bunyi di golongkan ke dalam gelombang longitudinal dimana arah rambatnya sejajar dengan arah getarnya.

Efek Doppler adalah peristiwa dimana frekuensi yang didengar berbeda dengan frekuensi sumber akibat adanya gerak relatif antara sumber bunyi dengan pendengar.

fp = ( v ± vp ) / ( v ± vs ) x fs

dimana :

fp = frekuensi yang didengar oleh pendengar ( Hz )

v = kecepatan bunyi ( m/s )

vp = kecepatan pendengar ( m/s )

vs = kecepatan sumber ( m/s )

fs = frekuensi sumber ( Hz )

Okay mari kita gunakan rumus ini untuk menyelesaikan soalnya.

Diketahui :

v = 340 m/s

fs = 400 Hz

fp = 300 Hz

vp = 40 m/s

Ditanyakan :

vs = ?

Penyelesaian :

fp = ( v - vp ) / ( v + vs ) x fs

300 = ( 340 - 40 ) / ( 340 + vs ) x 400

340 + vs = 400

vs = 60 m/s

MOBIL PEMADAM KEBAKARAN BERGERAK DENGAN KECEPATAN 30 M/S MENDEKATI PENGENDARA MOTOR SAMBIL MEMBUNYIKAN SIRINE DENGAN FREKUENSI 1550 HZ, PENGENDARA MOTOR BERGERAK MENJAUHI MOBIL PEMADAM KEBAKARAN DENGAN KECEPATAN 20 M/S, JIKA CEPAT RAMBAT BUNYI DI UDARA SAAT ITU 340 M/S MAKA FREKUENSI YG DI DENGAR MOTOR SEBESAR ...


-mobil pemadam: sumber (s)
-pengendara motor: pendengar (p)

Diketahui
Vs=30 m/s
Fs=1550 Hz
Vp=20 m/s
V=340 m/s

Ditanya
Fp=....?

Jawab
-karna S mendekati P maka nilai Vp adalah -
-karna P menjauhi S maka nilai Vs adalah -
    
         V - Vp
Fp= ---------- x Fs
         V - Vs

          340-20
Fp= -------------- x 1550
          340-30

         320
Fp= ------- x 1550
         310

Fp= 1,03 x 1550
Fp= 1596,5 Hz

EFEK DOPPLER!!!
fp = fs(v + vp) / (v + vs)
    = 1550(340 - 20) / (340 - 30)
    = 1550(32 / 31)
    = 49600 / 31
    = 1600 Hz

Bila cepat rambat bunyi di udara pada saat itu 340 ms-¹ maka frekuensi yang di dengar pendengar adalah


Jawaban:

Diket :

Vs = 30 m/s

fs = 1550 Hz

Vp = 20 m/s

v = 340 m/s

Ditanya : fp ?

Jawab :

fp = (v - Vp / v - Vs) . fs

fp = (340 - 20 / 340 - 30) . 1550

fp = (320 / 310) . 1550

fp = 1600 Hz

Jadi, frekuensi yang didengar pendengan adalah 1600Hz.

30. sebuah mobil pemadam kebakaran dan motor bergerak saling menjauhi.motor bergerak dengan kecepatan 40m.s.mobil pemadam kebakaran membunyikan sirene dengan frekwensi 400HZ.DAn di dengar oleh pengandara motor dengan frekwensi 300HZ. Jika cepat rambat bunyi di udara 340 m.s. maka kecepatan mobil pemadam kebakaran adalah


Maka kecepatan mobil pemadam kebakaran adalah 60 m/ s. Hal ini didapat dari mencari vs nya.

Penjelasan dengan langkah langkah :

Diketahui :

v = 340 ms^-1

vp = 40 ms^-1

fp = 300 hz

fs = 400 Hz

Ditanya :

vs ?

Dijawab :

[tex]Hukum\ Dopler \\\\ fp = \frac{v \ - vp}{v\ + vs } \ x\ fs \\\\ 300 = \frac{340 \ -\ 40 }{340 \ + \ vs } \ x\ 400 \\\\ 300 \ = \frac{300}{340\ + \ vs } \ x\ 400 \\\\ 300. ( 340\ + vs) \ = 300. 400 \\\\ 340 + vs = 400 \\\\ vs = 400 \ - 340 \\\\ vs = 60 \ ms^-1[/tex]

Efek Doppler, jika sumber bunyi yang mendekati pendengar akan berikan frekuensi yang lebih besar daripada frekuensi aslinya sehingga nilainya adalah plus. Jika sebaliknya sumber bunyi bergerak menjauhi pendengar bunyi akan terdengar semakin lemah sehingga nilainya adalah negatif (-) , kuat bunyi akan semakin lemah karena sumber bunyi dan pendengar sama-sama saling menjauhi .

hukum doppler memiliki rumus matematika sebagai berikut  :

[tex]fp = \frac{V\ +\ Vp }{V \ + \ Vs } \ x\ fs[/tex] ( jika sumber bunyi mendekati )

[tex]fp = \frac{V\ -\ Vp }{V \ - \ Vs } \ x\ fs[/tex] ( jika sumber bunyi menjauhi )

Keterangan :

Fp = frekuensi didengar oleh pendengar

V  = Rambat bunyi udara

Vp = Kecepatan pendengar

Vs = Kecepatan sumber bunyi

fs = frekuensi yang di pancarkan
