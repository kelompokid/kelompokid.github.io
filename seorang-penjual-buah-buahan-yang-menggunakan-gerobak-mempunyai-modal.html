3. Seorang penjual buah-buahan yang menggunakan gerobak mempunyai modal Rp4.000.000,00. Ia membeli jeruk dengan harga Rp16.000,00 per kg dan salak Rp6.400,00 per kg. Gerobaknya dapat memuat tidak lebih dari 400 kg, dengan laba penjualan jeruk Rp4.000,00 per kg dan salak Rp2.000,00 per kg. Laba maksiimum yang bisa penjual dapatkan jika semua jeruk dan salak terjual adalah


Jawab: Laba maksimum yang bisa penjual dapatkan jika semua jeruk dan salak terjual adalah Rp1.100.000. Cara penentuannya bisa disimak dalam penjelasan.

Penjelasan dengan langkah-langkah:

Dimisalkan jeruk adalah j dan salak adalah s. Model matematika yang terbentuk:

(a) 16.000j + 6.400s = 4.000.000 masing-masing dibagi 3.200 ➡ 5j + 2s = 1.250.

(b) j + s ≤ 400

(c) laba maksimum dicari dengan f(j,s) = 4.000j + 2.000s.

Eliminasi persamaan a dan persamaan b :

5j + 2s = 1.250 |×1| 5j + 2s = 1.250

j + s = 400 |×2| 2j + 2s = 800 –

3j = 450 ➡ j = 150

Substitusi nilai j ke persamaan b :

j + s = 400

150 + s = 400

s = 400 – 150 = 250

Banyaknya jeruk dan salak yang bisa dijual dengan modal tersebut masing-masing adalah 150 kg dan 250 kg. Maka laba maksimum yang bisa diperoleh apabila terjual semua :

f(j,s) = 4.000j + 2.000s

= 4.000(150) + 2.000(250)

= 600.000 + 500.000 = 1.100.000

Kelas : 11
Mapel : Matematika
Kategori : Program Linier
Kata Kunci : Pertidaksamaan garis, menggambar grafik suatu garis.
Kode : 11.2.11 (Kelas 11 Matematika Bab 11 - Program Linier)

Pembahasan :

Misal :
1 kg jeruk = x
1 kg jambu = y

Gerobak tersebut dapat menampung tidak lebih dari 150 kg buah.

x + y ≤ 150

Heris membeli buah jeruk dengan harga Rp15.000,00 per kg dan buah jambu dengan harga Rp12.000,00 per kg serta Heris mempunyai modal Rp2.100.000,00

15.000x + 12.000y ≤ 2.100.000
=> kedua ruas bagi 1.000 <=
15x + 12y ≤ 2.100
=> kedua ruas bagi 3 <=
5x + 4y ≤ 700

Jadi model matematika dari permasalahan di atas adalah :
x + y ≤ 150,
5x + 4y ≤ 700,
x ≥ 0,
y ≥ 0

22. seorang penjual buah-buahan yang menggunakan gerobak mempunyai modal Rp4.000.000,00. ia membeli jeruk dengan harga Rp16.000 per kg dan salak Rp6.400 per kg. gerobak mendapat memuat tidak lebih 400 kg, dengan laba penjualan jeruk Rp4.000,00 per kg dan salak Rp2.000,00 per kg. laba maksimum yang bisa penjual dapatkan jika semua jeruk dan salak terjual adalah​


Laba maksimum yang bisa penjual dapatkan jika semua jeruk dan salak terjual adalah​ Rp 1.100.000,00

Pembahasan

Diketahui:

Modal penjual = Rp 4.000.000,00

Harga jeruk/kg = Rp 16.000,00

Harga salak/kg = Rp 6.400,00

Gerobak memuat tidak lebih dari 400 kg

Laba penjualan jeruk = Rp 4.000,00 / kg

Laba penjualan salak = Rp 2.000,00 / kg

Ditanya:

Laba maksimum yang didapatkan jika semua salak dan jeruk terjual ?

Dijawab:

Harga jeruk = x

Harga salak = y

Keuntungan masing - masing penjualan buah :

Keuntungan jeruk = 4.000

Keuntungan salak = 2.000

Dengan variabel x dan y, fungsi tujuannya adalah

F (x,y) = 4.000x + 2.000y

Model matematika sesuai dengan informasi diatas adalah :

1. Jumlah jeruk dan salak tidak lebih dari 400 kg = x + y  ≤ 400

2. Harga beli jeruk dan salak = 16.000x + 6.400y ≤ 4.000.000

= 5x + 2y ≤ 1.250

3. Banyak jeruk tidak mungkin negatif = x ≥ 0

4. Banyak salak tidak mungkin negatif = y ≥ 0

Titik potong masing - masing garis terhadap sumbu x dan sumbu y :

Garis x + y  ≤ 400

a. x = 0 maka y = 400 > titik potong (0, 400)

b. y = 0 maka x = 400 > titik potong (400, 0)

Garis 5x + 2y ≤ 1.250

a. x = 0 maka y = 625 > titik potong (0. 625)

b y = 0 maka x = 250 > titik potong (250, 0)

(Grafik terlampir)

Titik B merupakan perpotongan antara x + y = 400 dan 5x + 2y = 1.250

x + y = 400       Ix2I 2x + 2y = 800

5x + 2y = 1.250 Ix1I 5x + 2y = 1.250 -

                               - 3x = - 450

                                   x = 150

Substitusikan nilai x kedalam persamaan x + y = 400

x + y = 400

150 + y = 400

y = 400 - 150

y = 250

Substitusikan titik - titik tersebut pada fungsi tujuannya F (x,y) = 4.000x + 2.000y

A (0, 400) > F (x, y) = 2.000 (400) = 800.000

B (150, 250) >  F (x, y) = 4.000 (150) + 2.000 (250) = 600.000 + 500.000 = 1.100.000

C (250, 0)  > F (x, y) = 4.000 (250) = 1.000.000

Jadi, laba maksimum yang bisa penjual dapatkan jika semua jeruk dan salak terjual adalah​ Rp 1.100.000,00
