2. salin dan setarakan persamaan ion berikut dengan cara setengah reaksi dan cara perubahan bilangan oksidasi dalam suasana asam


D. IO₃⁻ + HSO₃⁻  ==> I⁻ + SO₄⁻²  (Suasana Asam)

Jawab:

Sebelum mulai mereaksikan, maka kita hitung biloks masing-masing unsur terlebih dahulu untuk mengetahui unsur yang mengalami reduksi dan oksidasi.

Biloks IO₃⁻ = -1  

Biloks I + 3 x Biloks O = -1

Biloks I + 3 x (-2) = -1

Biloks I + (-6) = -1

Biloks I = +5

Biloks HSO₃⁻ = -1

Biloks H + Biloks S + 3 x Biloks O = -1

+1 + Biloks S + 3 x (-2) = -1

+1 + Biloks S + (-6) = -1

Biloks S - 5 = -1

Biloks S = +4

Biloks I⁻ = -1

Biloks SO₄⁻² = -2

Biloks S + 4 x Biloks O = -2

Biloks S + 4 x (-2) = -2

Biloks S + (-8) = -2

Biloks S = +6

Cara Setengah Reaksi:

Langkah 1: menuliskan kerangka dasar setengah reaksi reduksi dan setengah reaksi oksidasinya, yaitu:


Reduksi:

IO₃⁻  ==> I⁻

Oksidasi:

HSO₃⁻  ==> SO₄⁻²


Langkah 2: Menyetarakan masing-masing setengah reaksi.

a. Menyetarakan atom unsur yang mengalami perubahan biloks

Reduksi:

IO₃⁻  ==> I⁻  (Atom I sudah Setara)

Oksidasi:

HSO₃⁻  ==> SO₄⁻²   (Atom S sudah setara)

b. Menyetarakan atom Oksigen dan Hidrogen (Oksigen terlebih dahulu)


Reduksi:

IO₃⁻  ==> I⁻  

Karena reaksi berjalan dalam suasana asam, maka kita lakukan penyetaraan atom oksigen dengan penambahan molekul H₂O pada sisi yang kekurangan atom Oksigen. Jumlah molekul H₂O yang ditambahkan sesuai dengan selisih dari atom Oksigen yang ada.

Karena pada sisi reaktan (kiri) terdapat 3 atom oksigen dan pada produk (kanan) tidak ada atom oksigen, maka terdapat selisih 3, sehingga kita tambahkan 3 molekul H₂O pada sisi produk (kanan), menjadi:

IO₃⁻  ==> I⁻ + 3 H₂O

Selanjutnya kita setarakan atom hidrogen dengan menambahkan ion H⁺ (suasana asam) pada sisi berlawanan sebanyak selisih atom Hidrogen yang ada, yaitu:


IO₃⁻ + 6 H⁺  ==> I⁻ + 3 H₂O

Oksidasi:

HSO₃⁻  ==> SO₄⁻²   

Seperti halnya pada reaksi reduksi, kita lakukan penambahan molekul H₂O pada sisi yang mengalami kekurangan oksigen, yaitu pada sisi reaktan (kiri) sebanyak selisih atom oksigen, yaitu:

HSO₃⁻ + H₂O ==> SO₄⁻²

Selanjutnya kita setarakan jumlah atom Hidrogen dengan penambahan ion H⁺ pada sisi berlawanan sesuai dengan selisih atom Hidrogen yang ada, yaitu:

HSO₃⁻ + H₂O ==> SO₄⁻² + 3 H⁺  

c. Menyetarakan Muatan

Muatan disetarakan dengan menambahkan elektron pada sisi yang memerlukan, yaitu:

Reduksi:

IO₃⁻ + 6 H⁺  ==> I⁻ + 3 H₂O

Muatan di kiri = -1 + 6 = +5

Muatan di kanan = -1

Maka kita tambahkan elektron di sisi kiri sebanyak 6 elektron agar muatannya sama menjadi -1, yaitu:

IO₃⁻ + 6 H⁺ + 6 e⁻  ==> I⁻ + 3 H₂O

Oksidasi:

HSO₃⁻ + H₂O ==> SO₄⁻² + 3 H⁺  

Muatan di kiri = -1

muatan di kanan = -2 + 3 = +1

Maka kita tambahkan elektron di sisi kanan sebanyak 2 elektron agar muatannya sama menjadi -1, yaitu:

HSO₃⁻ + H₂O ==> SO₄⁻² + 3 H⁺  + 2 e⁻

Langkah 3: Menyetarakan jumlah elektron

menyetarakan jumlah elektron dengan menyamakan koefisien elektron dari reaksi oksidasi dan reduksi, karena pada reduksi koefisien elektron adalah 6 dan pada oksidasi adalah 2, maka kita carikan KPK nya yaitu 6 sehingga pada reduksi kita kalikan 1 dan pada oksidasi kalikan 3, menjadi:

IO₃⁻ + 6 H⁺ + 6 e⁻ ==> I⁻ + 3 H₂O               | x 1|

HSO₃⁻ + H₂O ==> SO₄⁻² + 3 H⁺ + 2 e⁻        | x 3|

----------------------------------------------------------------------- Menjadi:

IO₃⁻ + 6 H⁺ + 6 e⁻  ==> I⁻ + 3 H₂O

3 HSO₃⁻ + 3 H₂O ==> 3 SO₄⁻² + 9 H⁺ + 6 e⁻

----------------------------------------------------------------------------- +

IO₃⁻ + 3 HSO₃⁻  ==>  I⁻ + 3 SO₄⁻² + 3 H⁺   (Setara)

(Coret yang sama antara sisi kanan dan kiri)

8. Setarakanlah persamaan reaksi redoks berikut dengan cara setengah reaksi!


Jawaban:

Persamaan reaksi redoks setara tersebut adalah:

Soal 1: 2MnO₄⁻ + 5SO₃²⁻ + 6H⁺ → 2Mn²⁺ + 5SO₄²⁻ + 3H₂O.Soal 2: Cr₂O₇²⁻ + 6Fe²⁺ + 14H⁺ → 2Cr³⁺ + 6Fe³⁺ + 7H₂O.Soal 3: 2SO₄⁻ + 8Cl⁻ + 10H⁺ →S₂O₃²⁻ + 4Cl₂ + 5H₂OSoal 4: 2Al + NO₂⁻ + 5OH⁻ → 2AlO₃³⁻ + NH₃ + H₂O.Soal 5: Bi₂O₃ + 2ClO⁻ + 2OH⁻ → 2BiO₃⁻ + 2Cl⁻ + H₂O.Soal 6: 2CrO₂²⁻ + 3Br₂ + 8OH⁻ → 2CrO₄²⁻ + 6Br⁻ + 4H₂O.

Penjelasan:

Penyetaraan reaksi redoks dengan metode setengah reaksi, yaitu sebagai berikut.

Soal 1: MnO₄⁻ + SO₃²⁻ → Mn²⁺ + SO₄²⁻ (Asam)

Reduksi: MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O

Oksidasi: SO₃²⁻ + H₂O → SO₄²⁻ + 2H⁺ + 2e⁻

menjadi,

Reduksi: 2MnO₄⁻ + 16H⁺ + 10e⁻ → 2Mn²⁺ + 8H₂O

Oksidasi: 5SO₃²⁻ + 5H₂O → 5SO₄²⁻ + 10H⁺ + 10e⁻

Reaksi total:

2MnO₄⁻ + 5SO₃²⁻ + 6H⁺ → 2Mn²⁺ + 5SO₄²⁻ + 3H₂O.

Soal 2: Cr₂O₇²⁻ + Fe²⁺ → Cr³⁺ + Fe³⁺ (Asam)

Reduksi: Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O

Oksidasi: Fe²⁺ → Fe³⁺ + e⁻

Menjadi,

Reduksi: Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O

Oksidasi: 6Fe²⁺ → 6Fe³⁺ + 6e⁻

Reaksi total:

Cr₂O₇²⁻ + 6Fe²⁺ + 14H⁺ → 2Cr³⁺ + 6Fe³⁺ + 7H₂O.

Soal 3: SO₄⁻ + Cl⁻ → S₂O₃²⁻ + Cl₂ (Asam)

Reduksi: 2SO₄⁻ + 10H⁺ + 8e⁻ → S₂O₃²⁻ + 5H₂O

Oksidasi: 2Cl⁻ → Cl₂ + 2e⁻

Menjadi,

Reduksi: 2SO₄⁻ + 10H⁺ + 8e⁻ → S₂O₃²⁻ + 5H₂O

Oksidasi: 8Cl⁻ → 4Cl₂ + 8e⁻

Reaksi total:

2SO₄⁻ + 8Cl⁻ + 10H⁺ →S₂O₃²⁻ + 4Cl₂ + 5H₂O

Soal 4: Al + NO₂⁻ → AlO₃³⁻ + NH₃

Oksidasi: Al + 6OH⁻ → AlO₃³⁻ + 3H₂O + 3e⁻

Reduksi: NO₂⁻ + 3OH⁻ + 5H₂O + 6e⁻ → NH₃ + 10OH⁻

Menjadi,

Oksidasi: 2Al + 12OH⁻ → 2AlO₃³⁻ + 6H₂O + 6e⁻

Reduksi: NO₂⁻ + 3OH⁻ + 5H₂O + 6e⁻ → NH₃ + 10OH⁻

Reaksi total:

2Al + NO₂⁻ + 5OH⁻ → 2AlO₃³⁻ + NH₃ H₂O.

Soal 5: Bi₂O₃ + ClO⁻ → BiO₃⁻ +2Cl⁻ (Basa)

Oksidasi: Bi₂O₃ + 6OH⁻ → 2BiO₃⁻ + 3H₂O + 4e⁻

Reduksi: ClO⁻ + H₂O + 2e⁻ → Cl⁻ + 2OH⁻

Menjadi,

Oksidasi: Bi₂O₃ + 6OH⁻ → 2BiO₃⁻ + 3H₂O + 4e⁻

Reduksi: 2ClO⁻ + 2H₂O + 4e⁻ → 2Cl⁻ + 4OH⁻

Reaksi total:

Bi₂O₃ + 2ClO⁻ + 2OH⁻ → 2BiO₃⁻ +2Cl⁻ + H₂O.

Soal 6: CrO₂²⁻ + Br₂ → CrO₄²⁻ + Br⁻ (Basa)

Oksidasi: CrO₂²⁻ + 4OH⁻ → CrO₄²⁻ + 2H₂O + 3e⁻

Reduksi: Br₂ + 2e⁻ → 2Br⁻

Menjadi,

Oksidasi: 2CrO₂²⁻ + 8OH⁻ → 2CrO₄²⁻ + 4H₂O + 6e⁻

Reduksi: 3Br₂ + 6e⁻ → 6Br⁻

Reaksi total:

2CrO₂²⁻ + 3Br₂ + 8OH⁻ → 2CrO₄²⁻ + 6Br⁻ + 4H₂O.
