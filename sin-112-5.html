9. Tentukan nilai dari:a) cos 22, 5°b) sin 67, 5°c) sin 112 1/2°d) cos 157 1/2°​


a) cos 22,5° = cos 22 1/2 °

cos 22 1/2 ° = cos 1/2 45 °

[tex] \cos( \frac{1}{2} ) {45}^{0} = \sqrt{ \frac{1 \ + \cos( {45}^{0} ) }{2} } \\ = \sqrt{ \frac{1 + \cos( \frac{1}{2} ) \sqrt{2} }{2} } \\ = \sqrt{ \frac{1}{2} + \frac{ \sqrt{2} }{4} } \\ = \sqrt{ \frac{2 + \sqrt{2} }{4} } \\ = \sqrt{ \frac{2 + \sqrt{2} }{ \sqrt{4} } } \\ = \frac{1}{2} \sqrt{2 + \sqrt{2} } [/tex]

a] cos 22,5° = cos (45°/2)

cos (45°/2) = V[(cos 45° + 1)/2]

cos 22,5° = V[(1/2 V2 + 1)/2]

cos 22,5° = V[(V2 + 2)/4]

cos 22,5° = 1/2 V(V2 + 2)

b] sin 67,5° = sin (135°/2)

sin (135°/2) = V[(1 - cos 135°)/2]

sin 67,5° = V[(1 - (-1/2 V2))/2]

sin 67,5° = V[(1 + 1/2 V2)/2]

sin 67,5° = V[(2 + V2)/4]

sin 67,5° = 1/2 V(2 + V2)

c] sin 112 1/2° = sin (225°/2)

sin (225°/2) = V[(1 - cos 225°)/2]

sin 112 1/2° = V[(1 - (-1/2 V2))/2]

sin 112 1/2° = V[(1 + 1/2 V2)/2]

sin 112 1/2° = V[(2 + V2)/4]

sin 112 1/2° = 1/2 V(2 + V2)

d] cos 157 1/2° = cos (315°/2)

cos (315°/2) = V[(cos 315° + 1)/2]

cos 157 1/2° = V[(1/2 V2 + 1)/2]

cos 157 1/2° = V[(V2 + 2)/4]

cos 157 1/2° = 1/2 V(V2 + 2)

karena berada di Kuadran II , maka :

cos 157 1/2° = -1/2 V(V2 + 2)
