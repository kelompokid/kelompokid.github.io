18. Diketahui sistem persamaan linear berikut: x + 2y - z = 7 2x - y + 3z = 4 3x + y + z = 10 Tentukan nilai x, y, dan z yang memenuhi sistem persamaan linear tersebut!


Jawaban:

Semoga membantu

Penjelasan:

Untuk menyelesaikan sistem persamaan linear tersebut, dapat menggunakan metode eliminasi Gauss-Jordan. Dengan menggunakan metode ini, sistem persamaan linear dapat diselesaikan menjadi bentuk matriks augmented dan kemudian ditransformasi menjadi matriks segitiga atas. Selanjutnya, nilai variabel dapat ditentukan dengan melakukan substitusi mundur.

Setelah melakukan operasi pada matriks augmented, didapat matriks segitiga atas sebagai berikut:

| 1 2 -1 | 7 |

| 0 -5 5 | -10 |

| 0 0 2 | 4 |

Dari matriks segitiga atas tersebut, dapat ditemukan nilai z = 2. Kemudian, melakukan substitusi mundur pada baris kedua, didapat nilai y = 2. Selanjutnya, melakukan substitusi mundur pada baris pertama, didapat nilai x = 1.

Jadi, nilai x = 1, y = 2, dan z = 2 memenuhi sistem persamaan linear tersebut.
