3. Sebuah benda mempunyai massa 2 kg. Ketika bergerak, massa benda menjadi 2,5 kg. Kecepatan gerak benda tersebut adalah…​


Kecepatan gerak benda tersebut adalah 0,6 c.

Diketahui

mo = 2 kgm = 2,5 kg

Ditanya

Kecepatan gerak benda (v)

JAWAB

[tex] \boxed{m = \frac{mo}{ \sqrt{1 - \frac{ {v}^{2} }{ {c}^{2} } } } }[/tex]

[tex]2,5 = \frac{2}{ \sqrt{1 - \frac{ {v}^{2} }{ {c}^{2} } } } [/tex]

[tex] \frac{2}{2,5} = \sqrt{1 - \frac{ {v}^{2} }{ {c}^{2} } } [/tex]

[tex]0,8 = \sqrt{1 - \frac{ {v}^{2} }{ {c}^{2} } } [/tex]

[tex]0,64 = 1 - \frac{ {v}^{2} }{ {c}^{2} } [/tex]

[tex] \frac{ {v}^{2} }{ {c}^{2} } = 1 - 0,64[/tex]

[tex] \frac{ {v}^{2} }{ {c}^{2} } = 0,36[/tex]

[tex]v = \sqrt{0,36 \: {c}^{2} } [/tex]

[tex]v = 0,6 \: c[/tex]

Kesimpulan

Berdasarkan pemabahasan di atas, kecepatan gerak benda tersebut adalah 0,6 c.

m = 30 kg

v = 2 m³

ρ = ?

Jawaban:

[tex]ρ = \frac{m}{v} [/tex]

Jadi, massa jenis benda tersebut adalah 15 kg/m³.
