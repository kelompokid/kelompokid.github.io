1. faktor suku banyak x³+kx²-x-2 adalah (x+2) ,faktor yang lain


karena salah satu faktornya adalah x+2, maka suku banyak akan bernilai 0 jika x disubtitusi dengan -2

sehingga

(-2)³ + k(-2)² -(-2) - 2 = 0
-8 + 4k + 2 - 2 = 0
4k - 8 = 0
4k = 8
k = 8/4
k = 2

jadi, suku banyak tersebut adalah :

x³ + 2x² - x - 2

x³ + 2x² - x - 2 = (x+2)(x² - 1)

= (x + 2)(x + 1)(x - 1)

jadi, faktornya yang lain adalah (x + 1) dan (x - 1)

24. jika x^2 + x - 6 merupakan faktor dari x^4+ x^3 -10x^2 + ax+b maka faktor lain adalah...


Jawab:

x^2 + x - 6 = (x + 3) (x - 2)

karena merupakan faktor maka,

f (-3) = 0

(-3)^4 + (-3)^3 - 10 (-3)^2 + (-3)a + b = 0

-3a + b = 36

f (2) = 0

(2)^4 + (2)^3 - 10 (2)^2 + (2)a + b = 0

2a + b = 16

eliminasi dan subtitusi

-3a + b = 36

2a + b = 16

------------------ -

     -5a = 20

        a = -4

2a + b = 16

b = 16 + 8

b = 24

x^4 + x^3 - 10x^2 -4x + 24

= (x^2 + x - 6) (x^2 - 4)

maka faktor lainnya adalah (x^2 - 4) = (x - 2) (x + 2)
