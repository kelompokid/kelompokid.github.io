Jika asam bromida ( HBr ) bereaksi dengan natrium oksida ( Na20 )

maka reaksi yg terjadi adalah :

2 HBr + Na20 = 2NaBr + H20

Persamaan ion lengkap :

2H+ + 2Br- + 2Na+ O²- => 2NaBr + H20

persamaan ion bersih :

2Br- + 2Na+ => 2NaBr

2H+ O²- => H20

◾◾◾¶¶®PENJELASAN®¶¶ ◾◾◾◾◾◾

Persamaan reaksi kimia merupakan pernyataan yg ditulis dengan rumus kimia yg menginformasikan identitas dan kuantitas zat-zat yg terlibat dalam suatu perubahan kimia ataupun fisika.

◾◾◾¶¶®CONTOH®¶¶◾◾◾◾◾◾◾◾

Zn + bHCI => cZnCI2 + dH2.

Sehingga diperoleh persamaan matematis berikut ini :

atom Zn : a => C

atom H : b => 2d

atom CI : b => 2c

Jika C = 1

Maka diperoleh nilai : a=1, b=2 dan d=1. Sehingga diperoleh reaksi setara berikut :

Zn (s) + 2HCI (aq) => ZnCI2 (s) + H2 (g)

13. Larutan kalsium hidroksida di reaksikan dengan gas karbon dioksida menghasilkan apa?


Kalsium hidroksida = Ca(OH)₂
Karbon dioksida = CO₂

Ca(OH)₂ (aq) + CO₂ (g) → CaCO₃ (s) + H₂O (l)

maka dihasilkan endapan kalsium karbonat + air



14. 1. logam besi direaksikan dgn larutan asam sulfat menghasilkan larutan besi(II) sulfat dan gas hidrogen. 2. natrium karbonat padat direaksikan dgn larutan asam sulfat menghasilkan larutan natrium sulfat, gas karbon dioksida, dan air. 3. kalsium karbonat padat direaksikan dgn larutan asam klorida menghasilkan larutan kalsium klorida, air, dan gas karbon dioksida. 4. Magnesium karbonat padat + larutan asam klorida -> larutan magnesium klorida + gas karbon dioksida + air. 5. Amonium klorida padat + larutan barium hidroksida -> larutan barium klorida + air + gas amonia. 6. natrium karbonat padat + larutan asam sulfat -> larutan natrium sulfat gas karbon dioksida + air. 7. serbuk aluminium oksida + larutan asam klorida -> larutan aluminium klorida + air. 8. larutan barium hidroksida + larutan natrium sulfat -> endapan barium sulfat + larutan natrium hidroksida.


1. Logam besi direaksikan dgn larutan asam sulfat menghasilkan larutan besi(II) sulfat dan gas hidrogen.

Fe + H₂SO₄ → FeSO₄ + H₂

2. Natrium karbonat padat direaksikan dgn larutan asam sulfat menghasilkan larutan natrium sulfat, gas karbon dioksida, dan air.

Na₂CO₃ + H₂SO₄ → Na₂SO₄ + H₂O + CO₂

3. Kalsium karbonat padat direaksikan dgn larutan asam klorida menghasilkan larutan kalsium klorida, air, dan gas karbon dioksida.

CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂

4. Magnesium karbonat padat + larutan asam klorida -> larutan magnesium klorida + gas karbon dioksida + air.

MgCO₃ + 2HCl → MgCl₂ + H₂O + CO₂

5. Amonium klorida padat + larutan barium hidroksida -> larutan barium klorida + air + gas amonia.

2NH₄Cl + Ba(OH)₂ → BaCl₂ + 2H₂O + 2NH₃

6. Natrium karbonat padat + larutan asam sulfat -> larutan natrium sulfat gas karbon dioksida + air.

Na₂CO₃ + H₂SO₄ → Na₂SO₄ + H₂O + CO₂

7. Serbuk aluminium oksida + larutan asam klorida -> larutan aluminium klorida + air.

Al₂O₃ + 6HCl → 2AlCl₃ + 3H₂O

8. Larutan barium hidroksida + larutan natrium sulfat -> endapan barium sulfat + larutan natrium hidroksida.

Ba(OH)₂ + Na₂SO₄ → BaSO₄ + 2NaOH

Pembahasan

[tex]\boxed{\boxed{\bold{Persamaan~Reaksi }}}[/tex]

Persamaan reaksi adalah proses kimia yang menjelaskan terjadinya reaksi antara zat reaktan dan zat produk. Antara zat reaktan dan zat produk dipisahkan dengan tanda panah (--->) yang menunjukkan arah reaksi. Kemudian persamaan reaksi juga menuliskan wujud zat yaitu (g) berarti gas, (l) berarti liquid/cairan, (aq) berarti larutan (aqueous), (s) berarti padatan (solid). Persamaan reaksi setara ketika persamaan reaksi sudah diberikan koefisien reaksi. Koefisien reaksi adalah bilangan angka yang terletak mendahului rumus kimia zat.

Larutan asam sulfat

Larutan natrium sulfat

[tex]\boxed{\boxed{\bold{Jawab}}}[/tex]  

Berikut ini adalah langkah - langkah untuk menuliskan persamaan reaksi yang sudah setara yaitu :

1. Menentukan terlebih dahulu zat pereaksi (reaktan) dan zat hasil reaksi (produk)

Zat pereaksi adalah

Natrium karbonat padat Larutan asam sulfat

Zat hasil reaksi adalah

Larutan natrium sulfatGas karbon dioksidaAir

2. Menuslikan persamaan rumus kimia zat reaktan dan produk, lengkap dengan keterangan wujud zat (g / aq/ s/ l)

Natrium karbonat padat

memiliki rumus kimia Na₂CO₃ karena memiliki wujud padat (s) maka menjadi Na₂CO₃ (s)

Larutan asam sulfat

memiliki rumus kimia H₂SO₄ memiliki wujud larutan maka H₂SO₄ (aq)

Larutan natrium sulfat

memiliki rumus kimia Na₂SO₄ memiliki wujud larutan maka Na₂SO₄ (aq)

Gas karbon dioksida

memiliki rumus kimia CO₂ memiliki wujud gas maka CO₂ (g)

Air

memiliki rumus kimia H₂O memiliki wujud cair maka H₂O (l)

maka,

dapat disimpulkan persamaan reaksi adalah

Na₂CO₃ (s) + H₂SO₄ (aq) ---> Na₂SO₄ (aq) + CO₂ (g) + H₂O (l)

3. Menyetarakan jumlah atom setiap unsur dengan memberikan koefisien pada ruas kiri dan kanan

Na₂CO₃ (s) + H₂SO₄ (aq) ---> Na₂SO₄ (aq) + CO₂ (g) + H₂O (l)

persamaan reaksi diatas sudah setara karena jumlah atom setiap unsur baik ruas kiri dan kanan memiliki jumlah yang sama.
