10. Diketahui Matriks A =(a 2b 4 3c) Dan Matriks B =( 2c-3b a 2a+1 b+7) Jika Bt Menyatakan Teranspose B Dan A = 2bt Tentukan Nilai Dari a+2b+c​


mencari nilai transpose dari matriks B,

Bt = (2c, a, 2a+1, b+7)

Kemudian, karena A = 2Bt, maka:

(a 2b 4 3c) = 2(2c, a, 2a+1, b+7)

(a 2b 4 3c) = (4c, 2a, 4a+2, 2b+14)

gunakan sistem persamaan untuk mencari nilai a, b, dan c. Dari baris pertama dan ketiga matriks A dan B:

a = 4c

4a + 2 = 4

a = (4 - 2)/4

a = 1/2

Dari baris kedua matriks A dan B:

2b = 2a

b = a = 1/2

Dan dari baris keempat matriks A dan B:

3c = 2b + 14

3c = 2(1/2) + 14

3c = 15

c = 5

Jadi nilai a = 1/2, b = 1/2, dan c = 5. Dan untuk nilai dari a + 2b + c :

a + 2b + c = 1/2 + 2(1/2) + 5 = 1/2 + 1 + 5 = 6.5

Jadi, nilai dari a + 2b + c adalah 6.5.


11. Ada yang tahu bagaimana pengerjaan nomer 2?? Diketahui persamaan matriks A = 2BT (BT adalah transpose matriks B), dengan A = [tex] \left[\begin{array}{ccc}a&4\\2b&3c\end{array}\right] [/tex] dan B =[tex]\left[\begin{array}{ccc}2c-3b&2a+1\\a&b+7\end{array}\right] \end{array}\right] [/tex] . Nilai a + b + c = … a. 6 b. 10 c. 13 d. 15 e. 16


Nilai a + b + c = 15.

Pembahasan

Diketahui matriks A dan B sebagai berikut:

[tex]A = \left[\begin{array}{ccc}a&4\\2b&3c\\\end{array}\right][/tex]

[tex]B = \left[\begin{array}{ccc}2c - 3b&2a + 1\\a&b + 7\\\end{array}\right][/tex]

Transpose matriks adalah mengubah kedudukan unsur atau elemen-elemen pada baris menjadi kolom. Transpose matriks B adalah sebagai berikut:

[tex]B^T = \left[\begin{array}{ccc}2c - 3b&a\\2a + 1&b + 7\\\end{array}\right][/tex]

Terdapat persamaan [tex]A = B^T[/tex], dengan demikian dapat kita tulis sebagai berikut:

[tex] \left[\begin{array}{ccc}a&4\\2b&3c\\\end{array}\right] = 2 \left[\begin{array}{ccc}2c - 3b&a\\2a + 1&b + 7\\\end{array}\right][/tex]

[tex] \left[\begin{array}{ccc}a&4\\2b&3c\\\end{array}\right] = \left[\begin{array}{ccc}4c - 6b&2a\\4a + 2&2b + 14\\\end{array}\right][/tex]

Kesesuaian antarelemen

Baris-1 kolom-1 ⇒ a = 4c - 6b (persamaan-1)

Baris-1 kolom-2 ⇒ 4 = 2a, diperoleh [tex]\boxed{~a = 2~}[/tex]

Baris-2 kolom-1 ⇒ 2b = 4a + 2, substitusikan nilai a menjadi 2b = 8 + 2, selanjutnya 2b = 10, dan diperoleh [tex]\boxed{~b = 5~}[/tex]

Baris-2 kolom-2 ⇒ 3c = 2b + 14 (persamaan-2)

Substitusikan nilai b ke dalam persamaan-2, yakni 3c = 10 + 14, selanjutnya 3c = 24, dan diperoleh [tex]\boxed{~c = 8~}[/tex]

Dengan demikian nilai a + b + c adalah 2 + 5 + 8 = 15.

28. persamaan kecepatan benda di tuliskan sebagai berikut. v=At kuadrat + 2Bt kubik + Ct4 jika v dalam m/s dan t dalam sekon, tentukan dimensi dari A,B,C



v = A t² + 2B t³ + C t⁴

Persamaan merupakan jumlah beberapa suku.
Jika persamaan itu benar, setiap suku harus memiliki satuan yang sama, karena hanya besaran yang sama (besaran sejenis) yang dapat dijumlahkan secara aljabar.
Jadi setiap suku pada ruas kanan harus memiliki satuan yang sama dengan satuan v.

satuan t  → s   (sekon)
satuan v → m/s   (diketahui)

satuan A t² = satuan v
   satuan A = satuan v / satuan t²
                  = (m/s) / s²
                  = m/s³   atau m s⁻³     
dimensi A → [L] [T]⁻³  ← jwb

satuan B t³ = satuan v
   satuan B = satuan v / satuan t³
                  = (m/s) / s³
                  = m/s⁴   atau m s⁻⁴
dimensi B → [L] [T]⁻⁴  ← jwb

satuan C t⁴ = satuan v
   satuan C = satuan v / satuan t⁴
                  = (m/s) / s⁴
                  = m/s⁵   atau m s⁻⁵
dimensi C → [L] [T]⁻⁵  ← jwb
