Penjelasan dengan langkah-langkah:

s = d = 84 m

a) luas kolam = luas lingkaran

luas lingkaran = 1/4 x π x d x d

luas lingkaran = 1/4 x 22/7 x 84 x 84

luas lingkaran = 5.544 m²

b) selisih = luas persegi - luas persegi

selisih = s² - 5.544

selisih = 84² - 5.544

selisih = 7.056 - 5.544 = 1.512 m²

c) keliling kolam = keliling lingkaran

keliling lingkaran =  π x d

keliling lingkaran = 22/7 x 84 = 264 meter

d) keliling area di luar kolam = keliling persegi

keliling persegi = 4 s

keliling persegi = 4 x 84 = 336 meter
