4. kowe mau wis mangan apa durungkrama lugu=krama alus=ngojo alus=​


Ngowahi ukara dadi ragam basa.

Kowe mau wis mangan apa durung

→ krama lugu = sampeyan wau mpun nedha napa dereng?

→ krama alus = panjenengan kalawau sampun dhahar punapa dereng?

→ ngoko alus = panjenengan mau wis di dhahar apa durung?

__________________Pembahasan

Unggah ungguh basa yaiku aturan adat istiadat ing tlatah Jawa utawa babadan sopan santun, tata krama, tata susila, nanging nggunakake basa Jawa.

Unggah ungguh basa ana 2 :1. Ngoko :Ngoko luguNgoko alus2. Krama :Krama luguKrama alus

⚫ Ngoko lugu

Basane ngoko, ora kecampuran tembung Krama.

→ panggonane :

padha kanca sadrajat

marang wong kang luwih enom

→ tuladhane :

Kowe arep menyang endi?

⚫ Ngoko alus

Basane ngoko, kecampuran Krama Inggil.

→ sing dikramakne :

wonge, umume arupa tembung aran lan tembung sesulih

panindake, umume arupa tembung kriya

kaduwekane, awujud tembung Aran

→ panggonane :

dulur enom marang dulur tuwa

kapernah tuwa nanging derajate luwih dhuwur

wong sadrajat kang padha ngajeni

ibu marang bapak sing kulina Ngoko

→ tuladhane :

Panjenengan arep tindak menyang endi?

⚫ Basa Krama lugu

Basane Krama, ora dicampur ngokolan krama inggil. Ater ater lan panambange sing dikramakne.

→ panggonane :

wong sadrajat sing durung akrab

wong sing durung kenal

→ tuladhane :

sampeyan badhe kesah ten pundi?

⚫ Basa Krama Alus

Basane luwih Alus lan luwih ngajeni, sing dikramakne wonge, panindake lan kaduwekane, nanging awake dhewe ora dikramakne.

→ panggonane :

wong enom marang wong tuwa

marang wong sing dhuwur drajate

→ tuladhane :

Panjenengan badhe tindak pundi?

9. KOWE MAU APA WIS MANGAN ?jadikan Bahasa Ngoko Alus, Krama Lugu dan Krama Alus​


Jawaban:

Ngoko alus :

Kowe mau apa uwis mangan?

Krama lugu :

Sampeyan wau punapa sampun nedha?

Krama alus :

Panjenengan wau punapa sampun dhahar?

Penjelasan:

=> Basa Ngoko1. Ngoko Lugu

Ngoko lugu yaiku basa ngoko kang kaya biasane diucapake marang wong sing sapadha-padha umure utawa marang wong sing wis akrab banget.

2. Ngoko Alus

Ngoko alus yaiku ngoko kang tetembungabe dicampue krama alus utawa krama Inggil. Ngoko alus biasane digunakake marang pasrawungan wong dewasa utawa wong tuwa kang wis kulina banget lan wong tuwa marang wong dewasa kang luwih dhuwur pangkate.

______________________

=> Basa Krama

1. Krama Lugu

Krama lugu yaiku krama alus nanging ora alus banget. Krama iki digunakake yen ana pambicaraan saka derajat kang padha-padha.

2. Krama Inggil

Basa krama Inggil yaiku basa krama sing nganggo logat alus. Biasane krama Inggil digunakake marang wong sing enom kang micara karo wong sing luwih tuwo. Tuladhane bocah marang simbah.
