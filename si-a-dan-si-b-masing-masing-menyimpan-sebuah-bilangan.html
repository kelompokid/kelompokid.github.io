a - b = 15

Dari a - b = 15, diperoleh a = 15 + b

Substitusi ke a × b = 1000

a × b = 1000

(15 + b) × b = 1000

15b + b² = 1000

b² + 15b - 1000 = 0

(b - 25)(b + 40) = 0

b - 25 = 0 maka b = 25

b + 40 = 0 maka b = -40

Diperoleh nilai a:

b = 25, maka a = 15 + 25 = 40

b = -40, maka a = 15 + (-40) = -25

Jika a > 0 dan b > 0, maka

a + b = 40 + 25

= 65

Jika a < 0 dan b < 0, maka

a + b = -25 + (-40)

= -65

[tex]a \times b = 1000 \\ a - b = 15 [/tex]

Kita substitusikan nilai persamaan 1 ke persamaan 2. dengan nilai a= 1000/b

[tex] \frac{1000}{b} - b = 15 \\ \frac{1000 - {b}^{2} }{b} = 15 \\ 1000 - {b}^{2} = 15b \\ {b}^{2} + 15b - 1000 = 0 \\ (b + 40)(b - 25) = 0 \\ b = - 40 \: atau \: b = 25[/tex]

Substitusikan ke persamaan 1

Untuk b=-40 dan b=25

[tex]a = \frac{1000}{ - 40} = - 25 \\ \\ atau \: \\ a = \frac{1000}{25} = 40[/tex]
