5. bilangan oksidasi mangan (Mn) terendah terdapat dalam senyawa


Bilangan oksidasi Mn terendah terdapat dalam atom Mn dengan bilangan oksidasi 0. Mn (mangan) merupakan atom logam sehingga memiliki bilangan oksidasi positif. Bilangan oksidasi Mn berada pada rentang 0 hingga +7.

Bilangan oksidasi Mn sebagai atom bebas = 0Bilangan oksidasi Mn dalam senyawa MnO = +2Bilangan oksidasi Mn dalam senyawa MnO₂ = +4Bilangan oksidasi Mn dalam senyawa KMnO₄ = +7

Pembahasan

Bilangan oksidasi unsur adalah bilangan bulat (utuh) yang menunjukkan besarnya muatan berupa angka yang disumbangkan oleh atom dari unsur pada molekul atau ion yang dapat dibentuknya. Besarnya bilangan oksidasi berada di antara -7 sampai dengan +7.

19. Disajikan beberapa senyawa yg terbentuk dari atom:KMnO³,Na²MnO⁴,Mn²O³,MnO²,dan Mn²+,senyawa mangan yg memiliki harga bilangan oksidasi tertinggi terdapat pada senyawa?


Diantara senyawa KMnO₃, Na₂MnO₄, Mn₂O₃, MnO₂ dan Mn²⁺ yang merupakan senyawa yang memiliki bilangan oksidasi mangan tertinggi adalah senyawa Na₂MnO₄ dengan biloks Mn sebesar +6

Pembahasan

Penentuan bilangan oksidasi

KMnO₃

Jumlah total biloks atom pada KMnO₃ = 0

BO K + BO Mn + 3 × BO O                     = 0

(+1) + BO Mn + 3 × (-2)                            = 0

(+1) + BO Mn + (-6)                                  = 0

BO Mn + (-5)                                           = 0

BO Mn                                                    = +5

Na₂MnO₄

Jumlah total biloks atom pada Na₂MnO₄ = 0

2 × BO Na + BO Mn + 4 × BO O                = 0

2 × (+1) + BO Mn + 4 × (-2)                          = 0

(+2) + BO Mn + (-8)                                     = 0

BO Mn + (-6)                                               = 0

BO Mn                                                        = +6

Mn₂O₃

Jumlah total biloks atom pada Mn₂O₃ = 0

2 × BO Mn + 3 × BO O                          = 0

2 × BO Mn + 3 × (-2)                              = 0

2 × BO Mn + (-6)                                    = 0

2 × BO Mn                                             = +6

BO Mn                                                   = +3

MnO₂

Jumlah total biloks atom pada MnO₂ = 0

BO Mn + 2 × BO O                               = 0

BO Mn + 2 × (-2)                                   = 0

BO Mn + (-4)                                         = 0

BO Mn                                                  = +4

Mn²⁺

Biloks Mn sebagai ion dengan muatan +2 adalah sama dengan muatannya yaitu +2.

Jawab

Aturan-aturan untuk menentukan Bilangan Oksidasi

• Unsur bebas mempunyai biloks =0 Contoh → H,N,Fe berturut-turut dalam H2, N2, dan Cl= 0

• Fluorin, unsur yang mempunyai biloks -1

• Golongan I (logam alkali ) biloks = +1 contoh: Li, Na, K, Rb, Cs

• Golongan II (Alkali Tanah) biloks = +2 contoh = Be, Mg, Ca, Sr, Ba

• biloks suatu unsur dalam suatu ion tunggal sama dengan muatannya Contoh: biloks Fe dalam ion Fe²+ = +2 S²- = 2-

• Biloks H umumnya= +1, kecuali dalam senyawa dengan logam maka biloks = -1 Contoh: NaH BaH2 = -1

• biloks O umunya = -2 Contoh: biloks O dalam H2O, MgO = -2 kecuali
-Dalam Fe2O = +2
-Dalam Peroksida, seperti H2O2, biloks O = -1 -Dalam Senyawa superoksida, seperti KO2 biloks O= -1/2
-Dalam senyawa terner, biloks O selalu = -2
-Dalam senyawa biner, biloks O belum tentu = +2

• biloks halogen, sperti F, Cl, I, Br, At = -1

• biloks O dalam unsur-unsur dalam suatu senyawa = 0 Contoh: H2SO4= 0

• bilols unsur-unsur dalam senyawa poliatom = muatannya contoh = SO4²- = -2

• biloks dalam senyawa ion = muatan kation dan anionya.

Bila Sudah memahami aturan-aturan biloks Mari Kita Tentukan b.o dalam pertanyaan

25. diantara senyawa senyawa mangan berikut yang mempunyai bilangan oksidasi tertinggi adalah


Diantara senyawa senyawa mangan berikut yang mempunyai bilangan oksidasi tertinggi adalah KMnO₄ (B) dengan bilangan oksidasi Mn = +7.

Pembahasan : A. Biloks Mn dalam MnO₂  

biloks O = -2

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks MnO₂ = 0

biloks Mn + (2 × biloks O) = 0

biloks Mn + (2 × (-2)) = 0

biloks Mn + (-4) = 0

biloks Mn = +4

B. Biloks Mn pada KMnO₄

biloks K = +1         (merupakan unsur golongan IA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks KMnO₄ = 0  

biloks K + biloks Mn + (4 × biloks O) = 0  

(+1) + biloks Mn + (4 × (-2)) = 0  

(+1) + biloks Mn + (-8) = 0  

biloks Mn + (-7) = 0

biloks Mn = +7

C. Biloks Mn dalam MnSO₄

biloks SO₄ = -2

aturan biloks :  

Bilangan oksidasi senyawa atau molekul ion adalah sesuai dengan jenis muatan dan jumlahnya. Contoh : NH₄⁺ = +1, SO₄²⁻ = -2, dan CO₃²⁻ = -2.

biloks MnSO₄ = 0  

biloks Mn + biloks SO₄ = 0

biloks Mn + (-2) = 0

biloks Mn = +2

D. Biloks Mn pada K₂MnO₄

biloks K = +1          (merupakan unsur golongan IA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks K₂MnO₄ = 0  

(2 × biloks K) + biloks Mn + (4 × biloks O) = 0  

(2 × (+1)) + biloks Mn + (4 × (-2)) = 0  

(+2) + biloks Mn + (-8) = 0  

biloks Mn + (-6) = 0  

biloks Mn = +6

E. Biloks Mn dalam MnO

biloks O = -2

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks MnO = 0

biloks Mn + biloks O = 0

biloks Mn + (-2) = 0

biloks Mn = +2

⇒ Kesimpulan, maka diantara senyawa senyawa mangan berikut yang mempunyai bilangan oksidasi tertinggi adalah KMnO₄ (B) dengan bilangan oksidasi Mn = +7.

biloks Mn + 2 biloks O = 0

biloks Mn + biloks O = 0

Aturan-aturan untuk menentukan Bilangan Oksidasi

• Unsur bebas mempunyai biloks =0 Contoh → H,N,Fe berturut-turut dalam H2, N2, dan Cl= 0

• Fluorin, unsur yang mempunyai biloks -1

• Golongan I (logam alkali ) biloks = +1 contoh: Li, Na, K, Rb, Cs

• Golongan II (Alkali Tanah) biloks = +2 contoh = Be, Mg, Ca, Sr, Ba

• biloks suatu unsur dalam suatu ion tunggal sama dengan muatannya Contoh: biloks Fe dalam ion Fe²+ = +2 S²- = 2-

• Biloks H umumnya= +1, kecuali dalam senyawa dengan logam maka biloks = -1 Contoh: NaH BaH2 = -1

• biloks O umunya = -2 Contoh: biloks O dalam H2O, MgO = -2 kecuali
-Dalam Fe2O = +2
-Dalam Peroksida, seperti H2O2, biloks O = -1 -Dalam Senyawa superoksida, seperti KO2 biloks O= -1/2
-Dalam senyawa terner, biloks O selalu = -2
-Dalam senyawa biner, biloks O belum tentu = +2

• biloks halogen, sperti F, Cl, I, Br, At = -1

• biloks O dalam unsur-unsur dalam suatu senyawa = 0 Contoh: H2SO4= 0

• bilols unsur-unsur dalam senyawa poliatom = muatannya contoh = SO4²- = -2

• biloks dalam senyawa ion = muatan kation dan anionya.
