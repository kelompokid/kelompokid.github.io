19. fungsi f dengan rumus fx = akar x kuadrat dikurang X per x + 1 terdefinisi pada himpunan


Daerah himpunan domain dari fungsi f (x) = [tex]\frac{\sqrt{x^2 \:-\: x}}{x \:+\: 1}[/tex] adalah {x | x ≤ 0 U x ≥ 1, x ≠ - 1 , x ∈ R}

Fungsi f (x) akan terdefinisi pada daerah himpunan tersebut sebagai domain atau daerah asal himpunan f (x).

Pembahasan:

DAERAH DOMAIN FUNGSI

Fungsi dikatakan terdefinisi bila memiliki daerah asal atau domain dari fungsi tersebut.

Syarat domain fungsi

1. Pada fungsi berbentuk pecahan f(x) = [tex]\frac{u(x)}{v(x)}[/tex]

penyebut tidak boleh nol. Bila nol, fungsi menjadi tidak terdefinisi.

v (x) ≠ 0

2. Pada fungsi berbentuk akar f(x) = [tex]\sqrt{u (x)}[/tex]

Nilai di dalam bentuk akar tidak ada yang negatif.

u (x) ≥ 0

3. Pada fungsi berbentuk akar pada penyebutnya f(x) = [tex]\frac{u(x)}{\sqrt{v(x)}}[/tex]

Karena penyebut tidak boleh nol dan akar harus lebih besar sama dengan nol, maka

v (x) > 0

Untuk mencari domain fungsi, pertama masukkan syaratnya dan cari nilai x. Setelah mendapatkan nilai x, buat garis bilangannya. Masukkan nilai setiap selang batas x pada fungsi. Lalu cari batas yang sesuai dengan syaratnya.

Diketahui:

f (x) = [tex]\frac{\sqrt{x^2 \:-\: x}}{x \:+\: 1}[/tex]

Ditanyakan:

Daerah domain ?

Penjelasan:

f (x) = [tex]\frac{\sqrt{x^2 \:-\: x}}{x \:+\: 1}[/tex]

Karena memiliki bentuk akar sekaligus pecahan, maka diselesaikan secara terpisah dahulu.

• Karena pecahan

x + 1 ≠ 0

x ≠ 0 - 1

x ≠ - 1

Jadi daerah domain dari syarat ini x ≠ - 1

Maka di garis bilangan hanya angka - 1 yang bolong, yang lain positif atau memenuhi.

• Karena bentuk akar

x² - x ≥ 0

Anggap sama dengan nol dahulu.

x² - x = 0

Faktorkan

x (x - 1) = 0

x = 0 atau x - 1 = 0

x = 0 atau x = 1

Buat garis bilangannya. Lihat lampiran. Angka 0 dan 1 dibulatkan penuh karena ikut batas domain.

Ada 3 daerah selang,

x ≤ 00 ≤ x ≤ 1x ≥ 1

Untuk menguji nilai selang, masukkan sembarang nilai dari selang ini dan masukkan ke fungsi x² - x. Yang dicari adalah nilai yang ≥ 0 sesuai syarat x²- x ≥ 0.

• x ≤ 0

Ambil x = - 2

(-2)² - (-2) = 4 + 2 = 6 ≥ 0 maka daerah ini sesuai syarat.

Beri tanda positif pada selang.

Batas ini memenuhi.

• 0 ≤ x ≤ 1

Ambil x = 0,5

(0,5)² - 0,5 = 0,25 - 0,5 = - 0,25 ≤ 0

Beri tanda negatif pada selang, tidak memenuhi. Karena yang dicari yang positif.

• x ≥ 1

Ambil x = 2

2² - 2 = 4 - 2 = 2 ≥ 0 maka daerah ini sesuai syarat.

Beri tanda positif pada selang.

Batas ini memenuhi.

Daerah domain dari syarat ini

x ≤ 0 atau x ≥ 1

Satukan kedua syarat

Domainnya menjadi x ≤ 0 atau x ≥1 , x ≠ - 1

Jadi f (x) akan terdefinisi bila daerah domainnya

{x | x ≤ 0 U x ≥ 1, x ≠ - 1 , x ∈ R}
