9. Tentukan akar - akar dari persamaan trigonometri berikut kemudian tuliskan himpunan penyelesaiannya. tan (3x-45) = 1, 0 ≤ x ≤ 360


Penjelasan dengan langkah-langkah:

PERSAMAAN TRIGONOMETRI.

Diketahui :

tan (3x-45°) = 1 , 0° ≤ x ≤ 360°

Penyelesaian:

tan (3x-45°) = 1

3x -45° = 45° atau 225°

• 3x-45° = 45° + k. 180

3x = 90°+k.180°

x = 30° + k. 60°

k=0 => x = 30°

k=1 => x = 30+60=90°

k=2 => x = 30+120 = 150°

k=3 => x = 30+180= 210°

k=4 => x = 30+240 = 270°

k=5 => x = 30+300 = 330°

HP = {30°, 90°, 150°, 210°, 270°, 330°}
