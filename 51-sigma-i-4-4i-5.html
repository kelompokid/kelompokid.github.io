1. tentukan nilai 51 sigma i=4 (4i-5)


Notasi sigma adalah suatu simbol untuk menjumlahkan beberapa bilangan terurut yang mengikuti pola dan aturan tertentu. Jadi bentuk dari

[tex]\mathop{\sum}\limits_{i=1}\limits^{n}{U_{i}}[/tex] = U₁ + U₂ + U₃ + .... + Ui. Notasi sigma sering digunakan dalam materi induksi matematika, deret aritmatika dan deret geometri.


Pembahasan

Nilai dari [tex]\mathop{\sum}\limits_{i = 4}\limits^{51}{4i - 5}[/tex] adalah ...


Cara I :

Bisa menggunakan beberapa sifat dari notasi sigma yaitu

[tex]\mathop{\sum}\limits_{i = 1}\limits^{n}{ai + b} = a\mathop{\sum}\limits_{i = 1}\limits^{n}{i} + \mathop{\sum}\limits_{i = 1}\limits^{n}{b}[/tex][tex]\mathop{\sum}\limits_{i = 1}\limits^{n}{ai + b} = a\mathop{\sum}\limits_{i = 1+m}\limits^{n+m}{a(i-m)+b)[/tex][tex]\mathop{\sum}\limits_{i = 1}\limits^{n}{ai + b} = a\mathop{\sum}\limits_{i = 1-m}\limits^{n-m}{a(i+m)+b)[/tex][tex]\mathop{\sum}\limits_{i = 1}\limits^{n}{a} = an[/tex][tex]\mathop{\sum}\limits_{i = 1}\limits^{n}{i} = \frac{n(n + 1)}{2}[/tex]

Berdasarkan sifat diatas maka

[tex]\mathop{\sum}\limits_{i = 4}\limits^{51}{4i - 5}[/tex]

= [tex]\mathop{\sum}\limits_{i = 4 - 3}\limits^{51 - 3}{4(i + 3) - 5}[/tex]

= [tex]\mathop{\sum}\limits_{i = 1}\limits^{48}{4i + 12 - 5}[/tex]

= [tex]\mathop{\sum}\limits_{i = 1}\limits^{48}{4i + 7}[/tex]

= [tex]4\mathop{\sum}\limits_{i = 1}\limits^{48}{i} + \mathop{\sum}\limits_{i = 1}\limits^{48}{7}[/tex]

= [tex]4\frac{48(48 + 1)}{2} + 7(48)[/tex]

= 4(24)(49) + 336

= 4.704 + 336

= 1.512

= 5.040



Cara 2

Dengan menggunakan rumus deret aritmatika yaitu

Sn = [tex]\frac{n}{2}[/tex](a + Un)

[tex]\mathop{\sum}\limits_{i = 4}\limits^{51}{4i - 5}[/tex]

= [4(4) - 5] + [4(5) - 5] + [4(6) - 5] + ... + [4(51) - 5]

= [16 - 5] + [20 - 5] + [24 - 5] + ... + [204 - 5]

= 11 + 15 + 19 + .... + 199

a = 11

Un = 199

n = batas atas sigma - batas bawah sigma + 1

n = 51 - 4 + 1

n = 48

Jadi

Sn = [tex]\frac{n}{2}[/tex](a + Un)

S₄₈ = [tex]\frac{48}{2}[/tex](11 + 199)

S₄₈ = 24(210)

S₄₈ = 5.040
