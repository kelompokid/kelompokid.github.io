27. perintah untuk menyisipkan gambar adalah


FUNGSI – FUNGSI PADA MICROSOFT WORD Jawaban

Perintah untuk menyisipkan gambar adalah Insert Picture.

Pembahasan

Microsoft Word atau Microsoft Office Word adalah perangkat lunak pengolah kata (word processor) yang dibuat oleh Microsoft. Pertama kali terbit pada tahun 1983, perangkat lunak ini diberi nama Multi-Tool Word untuk Xenix. Beberapa versi lainnya kemudian dikembangkan untuk berbagai sistem operasi. Misalnya untuk DOS (1983), Apple Macintosh (1984), SCO UNIX, OS/2, dan untuk Microsoft Windows (1989). Setelah menjadi bagian dari Microsoft Office System 2003, bersama Microsoft Excel, Microsoft Power Point, dan Microsoft Access, software ini diberi nama Microsoft Office Word. Pada Office 2013 ke atas, perangkat lunak ini cukup dinamakan dengan Word.

Beberapa fungsi insert dalam Microsoft word adalah:

Insert Picture: berfungsi untuk memasukkan suatu gambar ke dalam dokumen. Gambar yang ingin kamu masukkan bisa kamu cari sendiri di dalam file – file yang kamu miliki. Insert Clip Art: berfungsi untuk memasukkan / menyisipkan gambar yang telah disediakan oleh Microsoft di dalam menu clip art. Ada banyak gambar di dalam menu ini. Insert Shapes: digunakan untuk menyisipkan gambar bangun seperti garis (lines), tanda panah (block arrow) flowchart, callouts, dan juga stars and banners. Gunakan fungsi ini untuk mempercantik dokumenmu.

29. berisi perintah perintah untuk menyisipkan gambar,simbol,tabledan grafik adalah


Jawaban:

Berisi perintah-perintah untuk menyisipkan gambar, simbol, table dan grafik adalah Insert Tab (Tab Insert/Penyisipan).

Penjelasan:

Insert Tab terletak di bawah Title Bar, Quick Access Toolbar dan Control Window serta terletak di atas jendela kerja/dokumen Microsoft Word. Insert Tab berisi perintah-perintah/tools untuk menyisipkan berbagai konten untuk mempermudah, memperjelas dan menyempurnakan tulisan dokumen kita. Insert Tab berada di antara Home Tab dan Page Layout Tab.
