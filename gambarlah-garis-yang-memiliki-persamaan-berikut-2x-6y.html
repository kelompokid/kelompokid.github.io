GAMBARLAH GARIS YANG MEMILIKI PERSAMAAN BERIKUTA.2X=6Y B.3Y-4=4Y C.4X+2Y=6 D.Y+3X-4=0Jawaban Pendahuluan

Untuk menggambar garis pada bidang kartesius dibuthkan minimal dua titik. Dari dua titik tersebut lalu ditarik garis. Agar lebih mudah, titik yang dipakai adalah x = 0 dan y = 0. Semakin banyak titik yang digunakan maka grafik akan semakin akurat dan bagus.


Pembahasan

a. 2x = 6y

saat x = 0, y = 0    => (0,0)

saat y = 1, x = 3     => (3,1)

saat y = 2, = 6        => (6,2)

Gambar ada dapat dilihat pada gambar 1


b. y = x + 2

saat x = 0, y = 2    => (0, 2)

saat x = 1, y = 3     => (1, 3)

saat x = 2, = 4       => (2, 4)

Gambar ada dapat dilihat pada gambar 2


c. y = -x + 3

saat x = 0, y = 3    => (0, 3)

saat x = 1, y = 2     => (1, 2)

saat x = 2,  y = 1       => (2, 1)

Gambar ada dapat dilihat pada gambar 3


d. y + 3x - 4 = 0

saat x = 0, y = 4    => (0, 4)

saat x = 2, y = -2   => (2, -2)

Gambar ada dapat dilihat pada gambar 4


Kesimpulan

Untuk menggambar garis pada bidang kartesius dibuthkan minimal dua titik. Titik tersebut dicari dengan memisalkan nilai x atau y pada persamaan

21. gambarlah garis dengan persamaan berikut. a. 4x+6y=18 b. -3y=12 c. 2x=7


Gambarlah garis dengan persamaan berikut:a. 4x + 6y = 18 b. -3y = 12 c. 2x = 7PENDAHULUAN

Untuk mencari titik yang dilewati oleh sebuah persamaan, subtitusilah sebarang nilai untuk menggantikan salah satu variabelnya.


PEMBAHASAN

DIKETAHUI

a. 4x + 6y = 18

b. -3y = 12

c. 2x = 7


DITANYA

Gambar persamaan garis masing-masing = ?


PENYELESAIAN

Bagian A: 4x + 6y = 18

Pilih x = 0

⇔ 4 × 0 + 6y = 18

⇔ 6y = 18

⇔ y = 3

Jadi, persamaan 4x + 6y = 18 melalui titik (0, 3)


Pilih y = 0

⇔ 4x + 6 × 0 = 18

⇔ 4x = 18

⇔ x = 18/4

⇔ x = 9/2 = 4,5

Jadi, persamaan 4x + 6y = 18 melalui titik (9/2, 0)


Bagian B: -3y = 12

⇔ -3y = 12

⇔ y = -4


Bagian C: 2x = 7

⇔ 2x = 7

⇔ x = 7/2 = 3,5


KESIMPULAN

a. 4x + 6y = 18 : persamaan ini melalui titik (0, 3) dan (9/2, 0)

b. -3y = 12        : persamaan ini setara dengan y = -4

c. 2x = 7           : persamaan ini setara dengan x = 7/2
