Sifat-sifat bilangan berpangkat

kᵃ x kᵇ = kᵃ⁺ᵇ

kᵃ : kᵇ = kᵃ⁻ᵇ

(kᵃ)ᵇ = kᵃᵇ

(kl)ᵃ = kᵃ.lᵃ

([tex]\frac{a}{b}[/tex])ⁿ = [tex]\frac{a^n}{b^n}[/tex] , untuk b ≠ 0

k⁻ⁿ = [tex]\frac{1}{k^n}[/tex]

kⁿ = [tex]\frac{1}{k^-^n}[/tex]

k⁰ = 1

Penyelesaian Soal

2⁻¹ + 3⁻¹

= [tex]\frac{1}{2}[/tex] + [tex]\frac{1}{3}[/tex]

= [tex]\frac{1.3}{2.3}[/tex] + [tex]\frac{1.2}{3.2}[/tex]

= [tex]\frac{3}{6}[/tex] + [tex]\frac{2}{6}[/tex]

= [tex]\frac{5}{6}[/tex]

a² = a × a

Jika kita akan mencari jumlah bilangan kuadrat dari 1² sampai n², maka dapat menggunakan rumus berikut

1² + 2² + 3² + 4² + … + n² =  [tex]\frac{n(n + 1)(2n + 1)}{6}[/tex]
